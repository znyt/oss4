　　“归墟天中，机缘无数，但是其中也是危险重重，若是为了些小奇遇，倒是轻松，但是其中真正的大奇遇，也是有着极大的危险。”
<br/>　　玄北圣者点了点头道。
<br/>　　“归墟天中，并非只有我们北域的武者，其他八域之中的天才，都在其中，我们北域，在九域之中，只能算是中下游，像是中域，天域的天才，才是真正恐怖的人物。”
<br/>　　“我曾经在年少之时，游览九域，我们北域的天才，跟真正七大宗门的天才，根本无法比拟，就算是夜天君，放在七大宗门中，也只能算是小有名气的天才，但是都无法被七大宗门特意培养，所以你此番前去归墟天，还是要小心行事才行。”
<br/>　　“嗯。”
<br/>　　楚风眠点了点头。
<br/>　　他听得出来，这是玄北圣者在提醒着他。
<br/>　　楚风眠如今在北域年轻一代，所向披靡，没有敌手，所以玄北圣者是在提醒着他，以防止是在归墟天的时候，招惹大祸。
<br/>　　不过玄北圣者所说的这些，楚风眠自然是知道的一清二楚，万年之前，九域的大部分地方，楚风眠都曾经踏足过。
<br/>　　对着七大宗门的了解，楚风眠也是要远远超越寻常武者。
<br/>　　越是了解七大宗门，楚风眠也就越是知道七大宗门的强大，玄北圣者所说的话，其实是没有任何一点的夸张。
<br/>　　夜天君，放在七大宗门之中，也只能算是优秀，而并非是无敌。
<br/>　　像是七大宗门中真正的天才，绝对是要远超夜天君的强大，就算是现在的楚风眠，都没有把握，足以一定战胜。
<br/>　　但是现在的楚风眠，在同境界上，也已经是走到了最顶峰了，不逊色与任何人。
<br/>　　在妖孽的天才，在楚风眠心中，也未必不是没有一战的能力。
<br/>　　这一次归墟天的开启，足以吸引整个九域的天才，想要得到其中的宝藏，都必须要经历无数竞争才行。
<br/>　　楚风眠的心中，倒是无比期待，跟无数天才强者交手，才足以令他的实力，再进一步。
<br/>　　玄北圣者跟楚风眠，又交谈了几句，便是直接离开了。
<br/>　　而另一边寒月漓，也是去寻找了冰漓长老，交谈了几句。
<br/>　　半日之后，寒月漓便是回到了这悟剑峰中。
<br/>　　“前往归墟天，路途遥远，我们还是快点赶过去才行。”
<br/>　　楚风眠看着寒月漓，开口道。
<br/>　　两道遁光，直接从这北邙圣域中离开。
<br/>　　归墟天，是位于九域之外的一处独特的空间之中，但是前往归墟天的入口，却是就在九域之中的西域。
<br/>　　西域的整体，跟北域极为相似。
<br/>　　不过西域最大的一方势力，却是一个巨大无比的王朝，终焉王朝，整个西域，都属于终焉王朝的领地。
<br/>　　虽然在西域之中，也并非什么没有这其他的势力，但是却远远无法跟终焉王朝抗衡，只能够是隐藏在阴影之中。
<br/>　　这归墟天的入口，正是西域终焉王朝之中的一方边境之中。
<br/>　　只要是前往那里，七大宗门的人便是会等待时日，寻找这归墟天空间最为薄弱的一天，打开一道通道，让武者进入其中。
<br/>　　所以这归墟天开启的日子，只有一天时间，所以必须要提前到达。
<br/>　　若是错过，便是也就只能在等待三年时间。
<br/>　　在这一路上，楚风眠跟寒月漓，也在不断交谈着关于归墟天的事。
<br/>　　临走之前，玄北圣者便是他所知道的事情，都告诉了楚风眠，玄北圣者年轻之时，也曾经前往过归墟天，所以他倒是告诉了楚风眠不少的事。
<br/>　　这归墟天，是一处极为庞大的区域，虽然不如北域，但是也比起无数国度要庞大的多。
<br/>　　这整个归墟天，其实便是一处古老宗门的遗迹，在其中，似乎是曾经林立这不少宗门，但是却都不知道什么因为，最终毁灭。
<br/>　　但是那些宗门曾经的一些功法，丹药，却都是留了下来。
<br/>　　因为归墟天是一处完全封闭的空间，所以其中的功法丹药，都是完好无损的保留了下来，成为了一处蕴含无数宝藏的宝地。
<br/>　　在其中，无数宝藏，几乎是搓手可得。
<br/>　　不过这归墟天中，虽然藏有无数宝藏，但是其中的危险，却也足以比拟九域之中的无数凶地。
<br/>　　在其中最大的危险，其实便是曾经那些宗门，所留下来的护山妖兽。
<br/>　　妖兽的寿命，要远远超过武者，所以那些宗门毁灭，但是一些妖兽，却是活了下来，在归墟天中，繁衍生息。
<br/>　　这些妖兽，也依靠着曾经那些宗门留下来的无数财富修炼，壮大，在其中已经是形成了无数族群，俨然已经是成为了一方妖域。
<br/>　　无数武者，进入其中，跟这些妖兽，便是会陷入恶战，尤其是其中的无数宝藏，不止是武者，对那些妖兽来说，也都是极为渴望。
<br/>　　所以归墟天中的任何一座宝藏，想要得到，都是凶险无比。
<br/>　　而除了妖兽之外，另外一点，才是更加可怕的，便是一同进入到其中的武者。
<br/>　　进入到归墟天，便是相当于前往了一处截然不同的空间，在归墟天中发生的一切，就算是在强大的武者，都不能推算出来。
<br/>　　所以在其中杀人，都不需要任何掩饰，便是不会被其他人知道。
<br/>　　因此，这归墟天几乎足以称得上是一处三不管地带。
<br/>　　进入其中的武者，互相厮杀，几乎是极为常见的事情。
<br/>　　可以说死在其中的大部分武者，都不是死在了妖兽一族的手中，而都是死在了同样进入的武者手中。
<br/>　　直接掠夺其他人身上的财富，比起辛辛苦苦去跟妖兽搏斗，要轻松的多。
<br/>　　甚至是有些传闻，在归墟天中，一些宗门的武者，都会联合起来，形成小团体，一起去掠夺其他武者，进行分赃。
<br/>　　在归墟天中，武者，比起妖兽还要危险。
<br/>　　“这归墟天，九域每名武者，都只可以进入这一次，能够在其中得到奇遇，也就只有这一次机会了。”
<br/>　　他们二人的遁光，在北域中穿行，寒月漓开口交谈道。
<br/>　　“据说从归墟天中，有着一处寒域，其中盛产冰系圣药，是最为适合我的地方，我进入到归墟天中，便是要前往这寒域，一探究竟。”
