　　整个拍卖场的氛围，因为两人的争夺，而变得有些剑拔弩张。
<br/>　　“好好好！我倒要看看，你如何跟我争？”
<br/>　　“九百万！”
<br/>　　愤怒之下的屠苏元让，再次加价一百万块灵石，让下方众人不由得一阵咋舌。
<br/>　　然而，
<br/>　　“一千万！”
<br/>　　冷蝉衣一脸淡然，同样也加价一百万。
<br/>　　“一千二百万！”
<br/>　　屠苏元让再次沉声开口，而且一口气提升了两百万灵石。
<br/>　　“一千三百万！”
<br/>　　在众人的惊呼声，冷蝉衣继续加价。
<br/>　　屠苏元让顿时两眼一瞪，死死的盯着冷蝉衣，恨不得要把她给吞了。
<br/>　　其实这三颗化婴丹，对于他们这样的顶级家族来说，价值并不算大。
<br/>　　而他之所以要出高价拍下这三颗灵丹，无非是想借机，交好这灵丹背后的那位是神秘丹道大师。
<br/>　　要知道，能够炼制出四品完美品阶的宝丹，此人的炼丹水平，至少也应该是五品炼丹宗师，甚至极有可能是六品丹君！
<br/>　　而放眼整个云丹城，六品丹君也绝不超过一手之数！
<br/>　　他们屠苏家若是拉拢到此人，必然可以令家族实力大增。
<br/>　　一念及此，屠苏元让虽然觉得无比肉痛，还是一咬牙，再次加价，
<br/>　　“一千八百万！！”
<br/>　　轰！
<br/>　　屠苏元让一口气提了五百万，顿时让全场众人都为之震撼。
<br/>　　就连冷蝉衣也不由得眉头微微一蹙，精致的脸庞之上，露出一抹犹豫之色。
<br/>　　就在这时，屠苏元让所在的包厢之内，突然想起来一道充满威严的声音，
<br/>　　“冷家丫头，可否卖老夫一个面子，将宝丹让给我屠苏家，可好？”
<br/>　　听到这道声音，冷蝉衣脸色微微一变，下方拍卖场中，似乎也有人听出了这道声音，也是满脸骇然。
<br/>　　冷蝉衣似乎也清楚，方才开口之人，在云丹城的地位，便是她的师父，云丹宗的宗主，也要给几分面子，于是当即冲着包厢的方向一拱手，道：
<br/>　　“既然屠苏家主开口，蝉衣自当遵从！”
<br/>　　哗！
<br/>　　冷蝉衣此话一出口，全场顿时轰然。
<br/>　　“没想到连屠苏家的家主大人，都亲自来了！”
<br/>　　“哈哈哈！”
<br/>　　伴随着一道爽朗、大气的声音响起，一名魁梧的中年男子，从包厢中缓步走了出来。
<br/>　　赫然正是屠苏元让的父亲，屠苏家的家主，屠苏玄！
<br/>　　哗哗哗！
<br/>　　屠苏玄一出现，其余几个贵宾包厢的法阵，也顿时纷纷散开，
<br/>　　“快看！那位好像是夏侯家的家主！”
<br/>　　“还有那边那位，是西门家的家主！”
<br/>　　“好像云丹城五大家族的家主全都到了！”
<br/>　　……
<br/>　　在众人惊呼中，各大家族的家主，彼此寒暄了一阵。
<br/>　　最终，所有人的目光，全都聚焦到，叶尘所在的那个包厢。
<br/>　　因为其他包厢的法阵，全都已经撤去，唯独他这个包厢，始终没有任何动静。
<br/>　　“看来剩下的这个包厢里面，应该就是炼制出四品宝丹的那位神秘大师了！不知道今日，可有缘一见？”
<br/>　　屠苏玄冲着叶尘所在的包厢，笑着朗声开口问道。
<br/>　　一时间，所有人的目光，全都纷纷转移到最后一个包厢上面。
<br/>　　然而，
<br/>　　过了许久，那包厢之中，却毫无反应。
<br/>　　屠苏玄顿时觉得面子有些挂不住，只好干笑了一声，接着又道：
<br/>　　“看来这位大师，不太喜欢这种热闹的场合，不如我们另外再约个僻静的地方一叙，如何？”
<br/>　　在屠苏玄看来，自己好歹刚花了一千八百万，买了对方的丹药，对方怎么着也应该给自己几分面子才对。
<br/>　　可惜，依旧毫无任何反应！
<br/>　　屠苏玄顿时怒了，
<br/>　　他身为屠苏家的家主，五大家族之首，而他的父亲，又是云丹宗十大长老之首，便是云丹宗宗主，也要给他几分面子，绝对是云丹城大佬级的人物，何曾被人如此冷淡对待过？
<br/>　　“阁下如此轻视于我，轻视我屠苏家，未免有些太狂妄了吧！”
<br/>　　说完这话，屠苏玄也顾不得天藏阁的规矩，直接虚手一抓、一扯！
<br/>　　刺啦啦！
<br/>　　伴随着一阵空间扭曲声音响起，
<br/>　　下一刻，
<br/>　　叶尘所在的包厢，周围的法阵，便已经被屠苏玄生生撕破！
<br/>　　哗啦！
<br/>　　空间一阵晃动之后，很快又稳定了下来，
<br/>　　众人定睛一看，
<br/>　　只见，在法阵之后，包厢之内，站在一名约莫二十岁上下的年轻人。
<br/>　　“是他！”
<br/>　　“叶尘！”
<br/>　　屠苏元让和冷蝉衣，更是同时惊呼起来。
<br/>　　“什么！”
<br/>　　其他各大家族的家主和高手们，听到屠苏元让和冷蝉衣的惊呼声，也全都为之一愣。
<br/>　　“原来他就是那个，昨日在考核大会上，出尽风头的丹道天才，叶尘？”
<br/>　　“难道这四品宝丹，竟也是出自他的手笔？”
<br/>　　“这不可能吧！能够炼制四品宝丹，至少也得是五品炼丹宗师以上的水平，他年纪轻轻，竟有如此实力？”
<br/>　　“莫非他背后还有一位丹道高人？”
<br/>　　……
<br/>　　众人纷纷议论着，
<br/>　　至于另外一边的屠苏玄，早已经气得七窍生烟，
<br/>　　“原来你就是那个，将我儿子打残的小畜生！还真是踏破铁无觅处，得来全不费工夫啊！今日老夫便来和你好好算算这笔账！”
<br/>　　屠苏玄说出这话的同时，直接朝叶尘虚手一抓，
<br/>　　“给我，滚过来！”
<br/>　　轰！
<br/>　　一道气势骇然的巨掌虚影，直奔叶尘笼罩而来。
<br/>　　“小心！”
<br/>　　另外一边包厢的冷蝉衣当即惊呼了一声，想要上前搭救，已经来不及了。
<br/>　　毕竟，屠苏玄的修为，要远远超过她，她就算事先知道屠苏玄要对叶尘动手，也根本无力援救。
<br/>　　“唉，早知道就应该把师尊请来的，现在可如何是好？”
<br/>　　在冷蝉衣看来，叶尘修为，连她都远远不如，更可况是屠苏玄？
<br/>　　然而很快，她便发现，事情的发展，和他想象的，似乎不太一样。
<br/>　　轰隆！
<br/>　　叶尘轻飘飘的一掌拍出，竟很轻松的便挡下了屠苏玄的一击。
<br/>　　“咦！”
<br/>　　看到这一幕，不但屠苏玄大为惊愕，周围众人也全都个个目瞪口呆。
<br/>　　要知道，屠苏玄可是半步化神境的强者，居然没能拿下一个只有金丹境的年轻人？
<br/>　　“看在你出高价买了我的灵丹的份上，我不想和你计较，你若是现在滚的话，我可以饶你不死！”
<br/>　　叶尘轻松接下屠苏玄这一掌之后，淡淡开口道。
<br/>　　全场众人听到这话，顿时一片轰然。
