　　寂静！
<br/>　　全场死一般的寂静！
<br/>　　众人预想过无数种可能，可是却唯独没有想到，屠苏霸天竟然会被叶尘杀死，而且是一招秒杀！
<br/>　　那可是化神境后期的超级强者，在整个云丹城，就算不是无敌的存在，那也是最最顶级的强者！
<br/>　　可是如今，竟然就这死了？
<br/>　　难道这个神秘的年轻人，竟是传说中的返虚真君不成？
<br/>　　所有人都懵了，即便是躲在暗处的那几位超级巨头，也全都一个个震惊的说不出话来。
<br/>　　“几位，热闹也看够了，还不出来吗？”
<br/>　　叶尘昂然立在九天之上，目光扫向远处的几个方向，缓缓开口。
<br/>　　那几位巨头大佬，顿时全都身体一震，被叶尘强大的威势所慑，个个脸色无比难看。
<br/>　　“咳咳！”
<br/>　　一名满头银发、一身道袍的老者，率先从暗处跳了出来，缓步走到叶尘跟前，脸上挂着着一抹温和笑容，拱手行了个一礼，道：
<br/>　　“在下慕容丹，添为云丹宗宗主，见过尊驾！”
<br/>　　轰！
<br/>　　轰！
<br/>　　轰！
<br/>　　这老者一出现，很快便又有几道气息，从远处踏空而来，转眼之间便已经来到了跟前。
<br/>　　“云丹宗刘振寿，见过阁下！”
<br/>　　“李家李天雄，见过阁下！”
<br/>　　……
<br/>　　足足七道，不弱于屠苏霸天的气息，从四面八方而来，将叶尘围在了中央，隐隐呈合围之势。
<br/>　　远处围观的众人，看到天空上这无比震撼的一幕，顿时一下子炸开了锅，
<br/>　　“连云丹宗宗主都来了吗？”
<br/>　　“还有两名云丹宗的长老，以及其他四大家族的四位老祖！”
<br/>　　“我的天！云丹城最顶级的强者，这是全都到齐了吗？”
<br/>　　“这些人联合在一起，便是返虚真君，也不能等闲视之吧？”
<br/>　　……
<br/>　　就连叶尘，看到这一幕，脸上也不由得露出一抹凝重之色，
<br/>　　也多亏方才，他动用雷霆手段，灭掉了屠苏霸天，倘若真拖上一时半刻，这些人在暗处偷袭，他还真是危险！
<br/>　　“怎么？你们是打算为屠苏老儿报仇？”
<br/>　　叶尘目光在七人身上一一扫过，冷冷开口道。
<br/>　　在说出这话的同时，叶尘已经悄悄将逍遥仙珠取了出来，暗暗藏于掌心之中。
<br/>　　叶尘就算再自信，也没有自信到，以一己之力，去对抗七大化神天君，而且这其中，还有一位，可是化神后期的强者！
<br/>　　即便他动用全部底牌，也未必能赢，甚至还会有生命危险！
<br/>　　不过，他有逍遥仙珠在手，随时都能劈开空间，从容离开这里。
<br/>　　只要对方没有那种空间封锁类的大神通，又或者是用空间法阵将他困住，他便可以立于不败之地。
<br/>　　“呵呵！”
<br/>　　云丹宗宗族慕容丹，再次满脸笑容的开口了，
<br/>　　“尊驾修为惊人，我等都十分佩服！不过屠苏长老，毕竟是我云丹宗十大长老之首，如今被阁下所杀，倘若此事就这么算了，我云丹宗的颜面何存？我这个宗主，恐怕没有办法，向数百万云丹宗弟子交代啊！”
<br/>　　慕容丹虽然语气温和，看上去慈眉善目，但是眼眸深处却透着一抹冰冷，而且话语之中，显然透着一股浓浓的威胁之意。
<br/>　　叶尘自然清楚，这件事肯定不会就这么轻易揭过，不由得冷冷一笑，
<br/>　　“那慕容宗主意欲何为？”
<br/>　　慕容丹两眼微微一眯，继续笑眯眯道：
<br/>　　“倘若阁下愿意加入我云丹宗，成为我云丹宗的客卿长老，那你和屠苏长老的事情，就属于你们之间的个人恩怨，宗门自然也就不会偏袒任何一方！”
<br/>　　哗！
<br/>　　慕容丹此言一出，众人顿时一片哗然，不过想想也不难理解。
<br/>　　屠苏霸天虽然在云丹宗有着举足轻重的地位，可毕竟已经死了！
<br/>　　人既然死了，自然便没有了任何价值，倒不如将眼前这个恐怖的妖孽，发展成为自己人。
<br/>　　如此一来，云丹宗不但没有损失，反而可以实力大增！
<br/>　　没办法，这种做法，虽然有些凉薄冷漠，但是在弱肉强食的修真界，从来就只有利益可言。
<br/>　　更何况，这些巨头大佬们，虽然同属于云丹宗，彼此之间却没什么感情，反而因为利益，彼此争斗不休。
<br/>　　而这个屠苏霸天，平日里行事霸道，其他大佬们也多有微词，如今他死了，这些人说不定心底反而欢喜，谁又愿意冒险为他报仇？
<br/>　　“叶尘，快答应下来啊！”
<br/>　　叶尘没有着急回答，倒是后方的冷蝉衣急了，忍不住催促起来。
<br/>　　在冷蝉衣看来，自己师尊提出的这个条件，实在太优厚了！
<br/>　　不但可以免受云丹宗的报复，还能成为云丹宗的长老，可谓是一步登天！
<br/>　　如此优厚的条件，根本就不需要考虑，叶尘居然还在犹豫，简直让人费解！
<br/>　　叶尘回头冲着冷蝉衣微微一笑，转而望向慕容玄，似笑非笑，道：
<br/>　　“慕容宗主，要我加入云丹宗也行，不过我还有要事在身，马上便会离开云丹城，你可能接受？”
<br/>　　慕容丹听到这话，顿时眉头紧皱，脸色有些难看。
<br/>　　叶尘似乎早料到，慕容丹的目的，并没有那么简单，冷冷一笑，道：
<br/>　　“慕容宗主，我并非三岁小孩，还是说出你的真正的目的吧！”
<br/>　　慕容丹微微一愣，继而哈哈大笑，抚掌道：
<br/>　　“好好！看来本宗主小瞧你了，既然话说到这个份上，本宗主就不拐弯抹角了！”
<br/>　　说到这里，慕容丹微微一顿，张了张口，似乎说了一句什么话，但是周围的众人，却什么也没有听到。
<br/>　　原来，慕容丹的这句话，是用传声入密的法门，只针对叶尘和其他六位强者，其他人自然无法听到。
<br/>　　至于他说的内容是：
<br/>　　“尊驾若是想要平安离开云丹城，只需把你所修炼的功法和炼丹法门留下即可！”
<br/>　　原来，这慕容丹真正的目的，是叶尘修炼的功法，和炼丹法门！
<br/>　　不过这也难怪！
<br/>　　叶尘以仅仅只有金丹境的修为，却可以秒杀化神天君，必然身怀绝世功法！
<br/>　　而他年纪轻轻，就可以炼制出四品完美品阶的宝丹，炼丹法门必然也是无比逆天！
<br/>　　听到慕容丹这话，其他六大强者，也纷纷反应了过来，顿时一个个全都露出炙热之色。
