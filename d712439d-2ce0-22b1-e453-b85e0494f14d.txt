　　叶少阳笑了笑，再问另外三个，也都是被法术公会的人抓来的，那个鱿鱼精最倒霉，他本来是在海里愉快地做一个小妖，与世无争的，结果前段时间，有一艘渔船从他生活的水域通过，他想趴在船上借着这艘船旅游——他很喜欢航海旅游，而且没有明确的目的地，只要到了港口，他就化成人形，上去溜达一下，感受一下异域风情，去景点转悠，感悟人生……不，鱿鱼生。
<br/>　　总而言之，他是一个文艺范小清新的妖怪。
<br/>　　可惜这次上错了船，船上有几个法师，他没能打得过人家，被抓了，本来是要被做成铁板鱿鱼的，后来法师里头有管事的觉得他本事不错，说什么本来任务就太重，正好拿他来充数，然后就被关起来，不知道被带到什么地方，之后就什么都不知道了，再次醒来……就是刚刚了。
<br/>　　说道这里，鱿鱼精也哭了。
<br/>　　另外一鬼一妖感同身受，也都伤心地哭了。
<br/>　　“别哭了，你们跟着我干，我把你们都带回去，听话啊，都别哭了……卧槽，说了都别哭了！”叶少阳无奈吼了一嗓子，这几个才停止哭泣。
<br/>　　“好了，现在我们回去，记住啊，继续伪装你们的身份，你们都是将军，不要露馅了！”
<br/>　　四人重新变化成之前的模样，起身回去，其中那两个妖怪之前现出本尊的时候，把一身衣服和盔甲都撑坏了，只好勉强穿上，提着头盔出来了。
<br/>　　小马坐在御座前的台阶上，头上戴着皇冠，早就左顾右盼的，突然看到四个人出来，眼睛都是红红的，有两个还衣衫不整，顿时大叫：“卧槽，小叶子，你是不是怕他们都爆了啊！”
<br/>　　“滚！”
<br/>　　叶少阳对四人使了个眼色，道：“你们出去之后，立刻调兵到京城来，等我的吩咐，另外看看你们的手下，有没有中抽魂术的，全送过来。”
<br/>　　四人领命里去。
<br/>　　这时候有太监来报，叶少阳要的那些物资都运到了，都在殿外堆着，叶少阳于是让皇帝下令，吩咐文武百官，还有那些锦衣卫和亲兵，全到外面列队站好。
<br/>　　小马扶着皇帝，跟他们一起出去。
<br/>　　叶少阳让他们搬来一口方鼎，摆在殿门外，然后找了十几个太监，帮他把那些法药倒进鼎中，兑水搅拌，之后叶少阳直接用桃木剑在里头搅拌，然后化符水、点慈航，调配出了一大堆法药，自己坐在边上，让那些文武百官排好队，轮流到自己面前来，挑一口法药塞到嘴里，再用灵符封口，让他们半柱香的时间再吐掉。
<br/>　　起初那几个都不干，生怕是毒药，叶少阳就让小马吴嘉伟帮忙按住，来硬的，后面的看前面的没事，又有皇帝下令，不得不从，也就不反抗了。
<br/>　　几百号人，这样轮下来，用了差不多一个小时，搞定之后，叶少阳也累瘫了，这时候那四个“将军”过来复命，报告情况：他们把手下兵士都集结在皇城外面了，也检查过了，只有极少数几个军官中了抽魂术，都在殿外候命。
<br/>　　叶少阳让他们把人都带过来，逐一看过，给他们也用了法药。
<br/>　　他坚持这件事自己一个人干，是想趁机看看每个人的脸——如果是人类的话，就像腾永清之前，虽然身份变了，但容貌并没有变化。
<br/>　　他猜测四宝应该也是这样，想从人群里找他，结果没找到。
<br/>　　他想从这群人里寻找四宝，结果没找到。
<br/>　　“别的将领呢，来了没有？你再催催？”叶少阳问皇帝。
<br/>　　皇帝刚要下令，突然有太监从殿门外进来，急匆匆走到皇帝身边：“禀皇上，杨将军和刘将军各自率领本部兵马来到，前来……前来勤王。”
<br/>　　皇帝偷偷看了叶少阳一眼，对那太监道：“勤什么王，快叫他们进来啊！”
<br/>　　太监为难道：“我也这般说了，但二位将军说……城中有叛军，他们要带兵前来，驱逐叛军！”
<br/>　　“瞎胡闹！”
<br/>　　皇帝一听就急了，自己这边局势好不容易稳定了，他们这来勤王，不管结果怎样，自己可是在人家手上啊，到时候自己第一个没命，于是用力一拍扶手，呵斥太监出去传旨，让两位将军务必火速来见。
<br/>　　太监蹬蹬蹬跑出去，只一炷香时间就回来了，禀告皇帝，那两位将军声称要保存实力，无论如何都不愿觐见。
<br/>　　“而且……”这太监偷偷看了叶少阳一眼，对皇帝道：“两位将军让我代为警告叛军，若一炷香时间不退，他们就要杀入皇城……”
<br/>　　“让他们来！千万别兜着，让他们全都过来！”小马匪气十足地喊道。
<br/>　　皇帝气得捶腿，“反了反了，他们就不怕治罪吗！”
<br/>　　一个白胡子的大臣走上前，低声对皇帝道：“皇上，他们怕是真要反了。”
<br/>　　“何意？”
<br/>　　这大臣眼珠一转，道：“皇上，恕老臣直言，这两位将军是真的反了……他们名义上是来勤王的，实际上……却是要浑水摸鱼……老臣罪该万死，说句不该说的，就算他们弑君，到时候只要一口咬定皇上是在叛军中没了的，又有谁能分辨真假？”
<br/>　　皇帝一听，吓的说不出话来。
<br/>　　叶少阳也觉得这老臣很有见地，多嘴问了一句：“你是什么官？”
<br/>　　“内阁首辅严嵩。”
<br/>　　严嵩！！
<br/>　　叶少阳几个人一下子就懵了，居然是他！
<br/>　　“我靠，你居然是严嵩！”小马先叫起来。
<br/>　　明朝历史上最有名的奸臣，史书上说他长年把持朝政、干了很多黑心事。虽然明知道面前这个是假的，但也是参照真的来设计的，叶少阳还是忍不住多看了他几眼。
<br/>　　这边正在商量对策，皇城外突然传来一阵打斗声，皇帝连忙差人去看，不一会工夫前来回报，说是杨、刘二位将军率兵打过来了，皇帝一听，立刻就吓呆了。
<br/>　　小马再也坐不住，没等皇帝有反应，立刻就起身出去，扬言要把这两个家伙抓过来当坐骑。
