
<br/>　　<br/>　　    华夏上下几千年历史，一共神秘消失了二十二个古老王国，其中最广为人知的就是楼兰国，历史上一直都有传说楼兰活跃繁荣了几百年的时间，但却在一夜之间突然消失了，直到八零年的时候一个外国人在罗布泊探险时偶然发现一神秘墓葬这才解开了关于楼兰消失的面纱。<br/>　　<br/>　　    只不过考古发掘和历史研究，一直到现在都没有把楼兰文明还有消失之谜给研究明白了。<br/>　　<br/>　　    但是在二十多年前的一个小道消息传出，有人去过楼兰古国的地宫，那就是向缺和王玄真，不过后来有人曾经向他们询问过这个事，但两人却对其三缄其口，一个字都不提，这就让楼兰的神秘似乎彻底埋葬在了历史的长河里。<br/>　　<br/>　　    除了楼兰古国之谜，历史中还有巴蜀三星堆，古徐王国和古滇国等等，统共有二十二个古国几乎都探查不出是如何消失的，也许就此永远也查不出了。<br/>　　<br/>　　    王令歌语不惊人死不休的说完，王惊蛰的眼珠子就朝着四周瞄了好几眼，确定没人后，他才舔了舔干裂的嘴唇，低声问道：“你有点感觉啊？”<br/>　　<br/>　　    王令歌斜了着眼睛，说道：“你别告诉我你没有看出来，那个湖底的古城，其实是根据一座风水阵建起来的……”<br/>　　<br/>　　    “嘶！”王惊蛰倒吸了一口冷气，说道：“你小点声，隔墙有耳，让人听到了咋办，消息不露出去了么，这事得私下里谈”<br/>　　<br/>　　    王令歌点头说道：“嗯，那得小心呢，这地方人太杂了”<br/>　　<br/>　　    陈三岁顿时无语的说道：“你俩说这话啥意思啊，合着就我一个外人，那他么就是防着我呢呗？”<br/>　　<br/>　　    “啪”王惊蛰伸手揽上他的肩头，意味深长的说道：“就你觉得，以你的本事能不能扛得过我们五个杀你灭口吧？”<br/>　　<br/>　　    “我退出行不行？”陈三岁连忙举起手说道。<br/>　　<br/>　　    “晚了，你现在趁早赶紧进入组织吧，效忠你这几位大哥，不然我们要是觉得你有什么不确定和不安全因素的话，真整不好背地里就把你给灭口了”<br/>　　<br/>　　    陈三岁崩溃的说道：“你那意思是，我现在就是想退出也晚了，来不及了呗？”<br/>　　<br/>　　    “呵呵，你想多了！”<br/>　　<br/>　　    王惊蛰转头跟王令歌说道：“回去再说这个，我心里有数”<br/>　　<br/>　　    在王令歌点出来之前，那一天抚仙湖上下起大雨，王惊蛰在北岸的山头上看到湖中心出现一片城池之后，他就和陈三岁连夜靠着脑袋里的记忆，把那座古城大概给复原出了一半左右，那时他就差不多品出来了，这古城是依据一个风水大阵而建成的。<br/>　　<br/>　　    回到驻地，王惊蛰将他和陈三岁描绘出的古城图样拿出来递给了王令歌：“这里大概描绘出了一多半，你看看”<br/>　　<br/>　　    王令歌接过图纸，谨慎的端详了片刻，他“唰”的一下抬头说道：“要不要给向叔和我爸说下，甚至让他们赶过来一趟也行”<br/>　　<br/>　　    王惊蛰当即摆手说道：“不用，我之前也曾想过这个，不过我没打算这么干，因为向叔和你爸一旦赶过来，那傻子都知道，这抚仙湖下的古城要有大事了，他俩都退隐多少年了，无缘无故出山那就太此地无银三百两了，并且当年他俩在楼兰地宫中被逆天改命，是没闹得人尽皆知，但小范围里还是有人知晓的……”<br/>　　<br/>　　    “你说，每次变天后，才会引发水下尸体的异象？平时都是没有任何征兆的？这是怎么回事”<br/>　　<br/>　　    王惊蛰叹了口气，说道：“这点我也搞不清楚了，异样是陈三岁在偶然之间发觉的，特别是最后一次在古城里祭坛中卫兵的突然杀手，更确定了这一点，要不是当初我留意了下的话，恐怕那天我能不能安然脱身都不好说了，就算我能回来，其他人估计就够呛了”<br/>　　<br/>　　    抚仙湖从这天起，连续被戒严了一个星期，全方位封锁，不但水面严禁任何人接近，就连沿岸也被隔离了出去。<br/>　　<br/>　　    而在这一个星期中，有一整队蛙人带着减压舱也进驻到了抚仙湖，并且开始对要进入水下的几人进行深潜指导，一个多星期前出现的变故，让谁都知道再次下水深入古城，搞不好就是九死一生了。<br/>　　<br/>　　    几天来，午桥和王令歌等人在练习深潜的时候，王惊蛰和耿明曾经有两次下到了古城中，不过这两次他俩都没有深入，只是在外围观察着，天气也没发生什么变化，所以古城下的尸体，一如往常一样，依旧在沉睡着。<br/>　　<br/>　　    此时，到抚仙湖已经将近一个月的时间快到二月末了。<br/>　　<br/>　　    澄江县的天气，艳阳高照，一连多天都是日头高挂。<br/>　　<br/>　　    过去了一个星期，差不多能准备的都已经准备完了，就算是在往后拖，基本也不会有什么效果了，所以这两天开始，午桥，菜刀文和王令歌还有林汶骐都试探着往湖底靠近，每次都尽量耗上两个小时左右，不过这回因为提前准备了减压舱，他们在返回水面时也都安全了不少。<br/>　　<br/>　　    这天清晨，日出东方，两艘船开到了抚仙湖中间水域，连续几天的试探性下潜之后，差不多已经可以进入古城内部了。<br/>　　<br/>　　    一艘船上，是王惊蛰等人，正在穿着潜水服，做着下潜准备工作。<br/>　　<br/>　　    “我这个以前晕水晕的，喝水有时都得闭着眼睛的人，愣是被你们给操练成了一个潜水高手，真是世事无常啊”午桥穿戴完衣服后，靠在栏杆上点了根烟，惆怅的叹了口气。<br/>　　<br/>　　    “啥也别说了，都是兄弟感情……”王惊蛰拱了拱手，说道：“等这边事完了以后，咱们把酒言欢的时候，我跟你讲讲关于王冬至的个人生活问题，比如她有啥软肋什么的”<br/>　　<br/>　　    午桥顿时斜了着眼睛，点头道：“妥，你这话说我心里确实有点热乎了”<br/>　　<br/>　　    王令歌扣着裤裆，舔着嘴唇说道：“那不得带我一个么，毕竟咱也出力了啊”<br/>　　<br/>　　    王惊蛰说道：“那都不是事”<br/>　　<br/>　　    “我擦，也不能剩下我吧？”菜刀文急头白脸的说道。<br/>　　<br/>　　    “哎呀我去……”午桥顿时一懵，瞅着王令歌和菜刀文说道：“我这么一看，你俩也是奔着想当他姐夫去的啊？”<br/>　　<br/>　　    林汶骐“唰，唰”的眨了几下眼睛，瞬间通透了：“我好像闻到了一股阴谋的味道呢，他这明显是用裙带关系，把你们都给套死了啊，毕竟小舅子不能轻易得罪了”<br/>　　<br/>　　    “唉，吉时到了，正好下水，走吧哥几个，回来在研究这事，走咯……”王惊蛰一看话头有点不对，人直接跳起来踩上栏杆就跃进了湖水中。
