        天际上，阁老和魔腾凌空而立。
<br/>
<br/>        在他们各自的上方，一株古树和一张血色巨嘴对峙着，两股恐怖的吸扯之力，不断的自其中弥漫而开，将天地间的无数能量都给吞噬而入。
<br/>
<br/>        而秦逸尘感觉得到，这两股恐怖的吞噬之力，似乎是在争夺着什么一般！
<br/>
<br/>        “那东西……”
<br/>
<br/>        旋即，秦逸尘的目光一扫，便是落在了两者的中央，在那里，悬浮着一团金色的液体。
<br/>
<br/>        而当看到那团液体之际，秦逸尘现，自己体内不知道生了什么变化的血脉之力，竟然是出了一种极为强烈的感觉。
<br/>
<br/>        “快点离开这里！”
<br/>
<br/>        见到秦逸尘居然苏醒了过来，虽然感觉到后者身上似乎有些异样的变化，但是，在这个时候艾伊洁根本来不及去询问具体情况，而是连忙喝道。
<br/>
<br/>        在话音落下之后，艾伊洁的身形便是化为一道流光对着远处暴掠而去，为了救助秦逸尘，她损耗了太大的力量，如果阁老和魔腾交手的余波倾泻下来，那她肯定毫无抵抗之力！
<br/>
<br/>        不过，让得艾伊洁有些意外的是，秦逸尘却并未随着她离开，虽然站了起来，但是，他却依旧站在原地，根本没有半点离开的模样。
<br/>
<br/>        “这个家伙，难道被一掌拍成傻子了吗？”
<br/>
<br/>        见到秦逸尘的模样，艾伊洁心中不由暗叹道。
<br/>
<br/>        “嗡……”
<br/>
<br/>        然而，在这个时候，天际上那种恐怖的吞噬之力交织间，一股恐怖的波动已经充斥在那片天地间，艾伊洁想要回身却已经是不太可能了！
<br/>
<br/>        “老东西，竟然敢阴我……”
<br/>
<br/>        对于下方的动静，魔腾似乎根本没有察觉到一般，他的眼中有着一抹凶戾的光芒闪烁着，虽然心中很是气愤，但是，在损耗了太多力量的他，与阁老相比起来，显然是有些不支。
<br/>
<br/>        “魔腾，在我面前，可轮不到你来造次！”
<br/>
<br/>        阁老则是冷哼一声，浩瀚的能量不断的自其身躯之中涌荡而出，那株大树上的光芒没有半点减弱，那股可怕的吞噬之力，更是逐渐的占据着上风，隐隐的有着要将那团金色液体抢夺过来的趋势。
<br/>
<br/>        见到阁老的模样，魔腾眼眸深处闪过一抹狡黠之色，在其嘴角也是勾起一抹狰狞的笑意。
<br/>
<br/>        “既然你这么想要这东西，那我就送给你吧！”
<br/>
<br/>        随着一道冷笑之声的响起，魔腾身后那张血色的巨口陡然闭拢，那种恐怖的吞噬之力竟然是突然消失。
<br/>
<br/>        “嗯？！”
<br/>
<br/>        这般突然的变故，让得阁老心中一惊，但是，此时他想要停手已经有些太晚了，随着古树摇曳，一股恐怖的吞噬之力瞬间便是将那团金色的液体吸扯而来。
<br/>
<br/>        “轰！”
<br/>
<br/>        金光一闪，竟然是直接将那株古老的大树给震得狠狠一颤，而阁老更是面色一变，身上的气息竟然出现了一些迟缓。
<br/>
<br/>        “好恐怖的能量，这团液体，究竟是什么？！”
<br/>
<br/>        阁老心中虽然惊讶，但是还是很快的反应了过来，他想要用自己的能量去将这团金色液体抓住，然而，却是惊愕的现，自己的真元，却无法靠近其周围！
<br/>
<br/>        “哈哈，老东西，我给过你机会了，这是你自己没本事！”
<br/>
<br/>        而在这个时候，魔腾的大笑之声再次响起，趁着阁老身上的气息出现短暂的停滞之际，那张巨嘴再次张开，恐怖的吞噬之力，瞬间便是将那团金色的液体拉扯了过去。
<br/>
<br/>        “嗡……”
<br/>
<br/>        随着金色光芒的闪过，那张血色巨嘴顿时闭上，原本充斥在天地间的那种恐怖波动，直接消散而空。
<br/>
<br/>        “被他得逞了？！”
<br/>
<br/>        见到这幕，在远处天际上的艾伊洁身形不由一顿，美眸之中更有着一抹惊骇之色涌现而出。
<br/>
<br/>        虽然她同样不知晓那团金色液体有什么作用，但是从梦魇一族的态度便可以看出此物的重要性，这件事情，对于他们而言，显然并非什么好事！
<br/>
<br/>        此时，阁老的面色也是一片阴沉，他显然没有料到刚才魔腾会做出如此举动，一个不慎，却是被金色液体中的恐怖能量所伤，从而让得魔腾抓住了机会！
<br/>
<br/>        而且，魔腾似乎早就料到会如此一般，他的手段也极为特殊，直接将那团金色液体吞噬而入，并没有半点受伤的迹象。
<br/>
<br/>        “桀桀，老家伙，等着吧，很快我就会让你在我脚下臣服！”
<br/>
<br/>        一举得逞，魔腾的狞笑之声，顿时在天地间响彻而起。
<br/>
<br/>        他感觉得到，只要将这团神血炼化，他就有望突破到天境！
<br/>
<br/>        如今的他，只是触碰到一丝天境的力量，便是知晓天境是何等的可怕！一旦登临那个境界，这些阻拦梦魇一族的土著，再也无法对他造成半点威胁！

<br/>        “那东西，我也想要呢……”
<br/>
<br/>        而就在魔腾正欲控制巨嘴收回之际，一道低喃之声，突然从下方响起。
<br/>
<br/>        “嗯？这小子，竟然没死？！”
<br/>
<br/>        听到这道声音，魔腾微微一愣，旋即，他的目光对着下方扫视而去，旋即，他眼瞳一缩，这才是现，不知何时，秦逸尘已经苏醒了过来。
<br/>
<br/>        而且，魔腾敏锐的察觉到，秦逸尘身上似乎出现了异样的变化，但是，具体是什么不同，一时间他也说不出来。
<br/>
<br/>        对于这个小辈竟然还能站起来，魔腾显然是有些意外，毕竟，他可是知晓，那尊对秦逸尘出手的存在，可是和他们站在同一层次的，在他看来，秦逸尘硬受了那尊存在一掌，应该绝无半点生还的可能才对。
<br/>
<br/>        “小辈，不用着急，你很快便会成为本皇的一部分！”
<br/>
<br/>        魔腾狞笑一声，并未急着对秦逸尘出手，后者虽然吸引他的注意，不过，在这个时候更为关键的，显然还是将那团金色的神血给炼化！

<br/>        “那东西可不是你这邪恶的天外邪魔所能染指的。”

<br/>        秦逸尘仿若没有听见魔腾语音中的杀意一般，他仰着头，脸上一片淡然，他的眸子，似乎变得有些空洞了起来。
<br/>
<br/>        “哼，无知的小辈！”
<br/>
<br/>        见到秦逸尘居然敢挑衅自己，魔腾眼中不由的闪过一抹戾色，不过，他依旧没有出手，毕竟，在其对面的阁老，身上的气息已经平稳了下来。
