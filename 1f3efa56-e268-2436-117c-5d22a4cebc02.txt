　　修炼过后，慕清霄则悄无声息的回到了萧家。
<br/>　　在接下来的日子里，古薰儿奇怪的举动让他有些惊诧，因为他发现，后者貌似有意无意的躲着他，让他有些莫名的尴尬，自己有这么可怕？
<br/>　　时间过的很快，半个月后，慕清霄来到萧炎居住的房间，他斗之气已经提升到了六段，有药尘在，这种提升的速度实属正常。
<br/>　　察觉到慕清霄的到来，药尘自戒指中飘了出来，道：“小娃娃，你这里有什么事情吗？”
<br/>　　“药老，我来这里，其实是请你帮我鉴定一种灵药。”
<br/>　　慕清霄微微一笑，药尘可是大陆第一炼药师，人称药尊者，药圣，让他帮鉴定灵药准没错。
<br/>　　若是连他都无法鉴定，恐怕自己摘取的花朵，真是普通花朵也说不定。
<br/>　　“灵药？拿出来让我看看。”
<br/>　　慕清霄点了点头，开启容器，自其中取出一朵蓝色的鲜花，鲜花散发出淡淡的清香，让人觉得极其舒服，这种怪异的感觉和奇妙。
<br/>　　鲜花刚出现，药尘瞳孔就是一缩，用灵魂力量将鲜花拿到面前，声音都有些颤抖，道：“这花你是从何处得到了，还有没有？”
<br/>　　慕清霄一愣，望着药尘激动的神色，看来这鲜花果然不普通啊。
<br/>　　“药老鉴定出这是何种灵药了吗？”
<br/>　　药尘摇了摇头，语气凝重，道：“我未曾看到过这种灵药，但这灵药的效果却是惊世骇俗，若是让某些老怪物知道，绝对会找上你。”
<br/>　　“这灵药真有如此珍贵吗？”
<br/>　　药尘微微点头，表情严肃，道：“若是我猜到不错，这灵药能够提高人的灵魂力量，对于任何人来说，都是至宝一样的存在，你最好不要随意取出来给外人看。”
<br/>　　只是嗅着此灵药的清香，他的灵魂力量便不再虚弱，且隐隐有恢复的趋势，可见对灵魂的效果高到了何种程度。
<br/>　　药尘如此一说，慕清霄也清楚了这灵药的珍贵，能够增强灵魂力量，这绝对是至宝啊，恐怕整个斗气大陆也找不出这种灵药吧。
<br/>　　想到此处，慕清霄道：“药老可知道这灵药如何使用？”
<br/>　　药尘看了一眼手中的鲜花，有些不舍，但还是递给了慕清霄，道：“有没有其他方法我不知道，但既然是灵药，自然是直接服用最好。”
<br/>　　慕清霄微微点头，不愧是斗气大陆第一炼药师，但他却没有却接药尘递过来的灵药，反而又从容器中取出八朵不同颜色的递了过去。
<br/>　　“既然这灵药对药老来说极致珍贵，那这些就当作您为我鉴药的报酬吧。”
<br/>　　望着慕清霄递过来的灵药，药尘目瞪口呆起来。
<br/>　　增强灵魂的天地至宝，哪怕只有一朵都算是奢侈的，可这小子在知道效果后，却依旧能面不改色的拿出这么多赠予他！
<br/>　　妖孽之才，还如此尊老爱幼，不能收为自己的弟子，药尘当真觉得可惜了。
<br/>　　“这些灵药真的送给我吗？”
<br/>　　药尘神色激动无比，有了这些灵药，他相信自己的灵魂在五年内，绝对能够恢复巅峰状态，甚至还能够增强不少。
<br/>　　慕清霄认真的点了点头，能够鉴定出灵药的效果，药尘可谓是功不可没，若非他鉴定出来，这些能增强灵魂力量的天地至宝，恐怕在过不久便要化作灰烬了。
<br/>　　更何况，这些灵药虽然珍贵，但对他来说真算不得什么，他容器中就算没有一万，却也有八千。
<br/>　　再次确认后，药尘也不再啰嗦，直接将灵药小心翼翼的收进空间戒指中，同时也更加确定，慕清霄绝对是个有原则的人，距离自己复活也不会太远了。
<br/>　　听两人的对话，萧炎知道灵药肯定极其宝贵，慕清霄拿来送人连眼睛都不眨一下，顿时佩服起他的气度来。
<br/>　　见药尘将灵药收下，慕清霄微微行了一礼，道：“药老，我还有一事相求。”
<br/>　　获得灵药，药尘心中非常喜悦，毫不介意，道：“有什么事，你尽管说吧。”
<br/>　　“药老，先前多有得罪，还请见谅。”慕清霄颇为歉意的说道。
<br/>　　见慕清霄态度诚恳，药尘微微一笑，道：“无妨，你收集异火，想必也有你的难处，而且你何来得罪一说，先前是一场公平的交易。”
<br/>　　“不错，异火关乎着我的生命。”
<br/>　　沉吟一会，慕清霄严肃的开口说道，他并没有说谎，若是不能获得异火榜上二十三种异火，他就会死亡。
<br/>　　“嗯，你有什么事情，尽管说吧。”
<br/>　　“恳请药老教我炼丹。”
<br/>　　话语刚落，药尘以为自己听错了，萧炎也楞了一下。
<br/>　　片刻后，药尘神色认真的盯着慕清霄，颇为遗憾的说道：“你的资质是我平生所见之最，但我的炼丹术只传给门下弟子......”
<br/>　　闻言，慕清霄嘴角抽了抽，双膝跪地，态度恭敬，道：“师尊在上，请受徒儿一拜。”
<br/>　　药尘眼中满是欣慰，心中更是激动无比。
<br/>　　慕清霄能在如此年龄成为斗皇，而且还是在没人教导的情况下，用妖孽来形容他一点都不为过，能收这样的弟子，药尘自然是一万个愿意。
<br/>　　激动之下，药尘连忙用灵魂力量将他托起，道：“好好好，以后你和萧炎都是我门下弟子，记住拜入我门下，必须谨记......”
<br/>　　经过简单的拜师礼后，他就顺利成为了药尘的弟子。
<br/>　　原本，慕清霄并不打算拜师，正所谓技多不压身，所以学习炼丹术，对他来说绝对是有利无弊。
<br/>　　更何况药尘，无论是炼丹术上，或者资历上都有资格成为他的师傅。
<br/>　　慕清霄朝萧炎微微一笑，道：“既然如此，往后我便叫你萧师兄了？”
<br/>　　萧炎脸上肌肉抽了抽，斗皇强者叫他师兄，虽然心里很爽，逼格也很高，但总觉得很怪。
<br/>　　“斗气大陆强者为尊，你为师兄，我为师弟便好。”
<br/>　　药尘见两人互相谦让，没有一丝作假，心中更是欣慰，比起欺师灭祖的韩风，面前的两个徒弟简直是宝啊，而且还是白白捡来的宝贝！
<br/>　　“既然如此，那我就占你一些便宜。”
<br/>　　其实，不管师兄或者师弟都无所谓，慕清霄想学炼丹术才是真的。
