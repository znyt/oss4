双双的确是一个胖姑娘，不过她胖的十分的可爱，大家对她都充满善意的。
<br/>
<br/>        她知道赵立英不喜欢自己，但是她没有想到赵立英这么厌恶自己，还说自己一身肉很呕心。
<br/>
<br/>        毕竟是小姑娘，双双的自尊心大受打击，眼眶迅速红了。
<br/>
<br/>        “赵立英，把手机还给我！”双双想起身，但是刚才赵立英推她的时候太用力了，她的膝盖都磕破了，血都流了下来。
<br/>
<br/>        赵立英冷笑一声，眼里是算计的光芒，“你的手机我需要借用一下，现在还不能还给你，明天就有好戏看了！”
<br/>
<br/>        说完赵立英上了豪车，他踩下油门疾驰而去，真的将双双给丢在高速公路上了。
<br/>
<br/>        双双挣扎着站起了身，她的膝盖还在流血，每走一步都觉得很疼。
<br/>
<br/>        这里离家太远了，就算用跑的也要八个小时。
<br/>
<br/>        天色越来越暗，这时突然下起了大雨，很快双双就被淋成了落汤鸡。
<br/>
<br/>        她在大雨里一步步的往家走去，倾盆大雨打的她都睁不开眼，身上又冷又疼，最后的时候她抱紧自己，齿关都在打颤。
<br/>
<br/>        眼里热热的，眼泪刷一下全下来了。
<br/>
<br/>        双双觉得这绝对是她人生里最黑暗最毁灭性的一天，所有的骄傲和尊严都被赵立英给狠狠的踩碎践踏了。
<br/>
<br/>        双双还很担心夏夕绾，很明显赵立英抢走她的手机是冲着夏夕绾去的，他最后说明天就有好戏看了又是什么意思？
<br/>
<br/>        ……
<br/>
<br/>        夏家。
<br/>
<br/>        夏妍妍收到了赵立英的微信，那张照片发送了过来。
<br/>
<br/>        夏妍妍将照片狠狠的来回看了好几遍，双眼里又妒又恨。
<br/>
<br/>        在她的印象里，陆寒霆英俊而成熟，举手投足都蔓延着上流成功男士的优雅矜贵，他在海城只手遮天，又低调神秘，作为最年轻的一代商界大佬他始终披着一层面纱，清冷又禁欲。
<br/>
<br/>        夏妍妍自六年前看到他的第一眼，就被他给深深的迷住了。
<br/>
<br/>        他的私生活相当干净，身边从来没有过女人，她自认为他眼光甚高，不是那些肤浅的男人，他应该更倾心有家世有学识的千金名媛，从不纵欲。
<br/>
<br/>        但是，现在的他已经令她大跌眼镜。
<br/>
<br/>        他将自己的黑金卡送给夏夕绾去刷，他给夏夕绾买only-love那样的钻石珠宝，他在1949酒吧里百般纵容她，就连自己是小白脸的话都信手捏来，他甚至在陆氏集团的大厅里，当着这么多人的面用这样的姿势抱夏夕绾，让她的双腿缠在他精硕的腰身上…
<br/>
<br/>        这样的陆寒霆简直跟那些肤浅的男人毫无区别，他已经被夏夕绾给迷得团团转了。
<br/>
<br/>        夏妍妍又想到了一向清寒暗黑的顾夜瑾，他也一样，叶翎往他面前一站，他的目光总是流连在叶翎那张如花似玉的柔媚小脸还有尤物般的身段上。
<br/>
<br/>        果然男人没有一个好东西，他们骨子里都肤浅，喜欢漂亮的女人，越漂亮越好。
