　　“峰哥，那我就先走了，这几天幸存者太多了，各个地方都需要人来盯着，我就去忙了。对了，这是峰哥您加入我们战士会的第一个月的福利，一百晶石，您点点，另外你们的临时身份证都已经升级为正式身份证了，这个身份证你们要拿好，买房子买水买电或者干什么的时候都需要登记身份证的，若是丢了的话，补办非常困难。”
<br/>　　麻子把东西给了林峰后他就离去了，这几天加入战士会的人太多了，他一直在忙着招揽。
<br/>　　林峰把自己的身份证拿了后，其余狗子他们的身份证都分发了下去。
<br/>　　身份证上依然没有头像生日籍贯地之类的常规资料，唯独一串编号，林峰还是六个八。
<br/>　　除此之外，比临时身份证多了一点的就是，上面出现了一个磁条区，不知道干什么用的。
<br/>　　“老师，这百来个晶石就放你这吧。”
<br/>　　林峰把战士会给的福利丢给了苏云。
<br/>　　“给我干嘛。”
<br/>　　苏云道。
<br/>　　“你拿着就是，用来交水电费和买一些生活用品。”林峰笑道。
<br/>　　“把我当什么了，家庭主妇吗？”苏云翻了个白眼，不过略一犹豫还是收下了。
<br/>　　“放我这也好，免得你胡乱用了。”
<br/>　　林峰笑道：“老师，你是怕我去找昨天看到的，站在街边的那些长腿小姐姐吗？不过话说回来，一百晶石还真的可以找几个好姐姐爽一爽了。”
<br/>　　苏云顿时眼中露出杀气。
<br/>　　“当然了，老师，我是个正经的孩子，我才不喜欢那些小姐姐的长腿，我只喜欢老师你的腿。”林峰认真道。
<br/>　　“不正经。”苏云翻了个白眼就走一边去了。
<br/>　　这几天下来，整个基地内已经不是充满活力这么简单了，而是一个字，乱。三个字，乱哄哄。
<br/>　　许多大街上已经出现了就地而睡的人，甚至刚才还有三个尸体被军警抬走的，这些都是活生生病死的。
<br/>　　当然，他们不是因为丧尸病毒，而是因为其余各式各样的病毒，比如说感冒病毒等等一系列。
<br/>　　尸体抬走后就会有许多穿着白色防化服的官兵在那里喷消毒药水。以免病毒大面积传播。
<br/>　　“老师，狗子，你们就自由活动吧，尽快熟悉熟悉这基地内的情况。另外虽然说这个地方看起了安全，但依然不能懈怠。”
<br/>　　林峰之前接触过战狂，五阶六阶的在寻常人眼中，那是非常厉害的大佬，但林峰从来没有这样觉得过。
<br/>　　“放心吧，我们心里有数。”陈勾道。
<br/>　　“嗯。”
<br/>　　林峰说完就离开了他准备出去找空投。
<br/>　　平民区非常的大。
<br/>　　大街上，一堆堆幸存者都是好奇害怕或者担心的看着这个陌生的环境。
<br/>　　“妈的，这什么破地方，老子堂堂XXX公司董事长居然要住这个地方？”
<br/>　　一个成功人士破口大骂。
<br/>　　“基地负责人呢？我要找他！”
<br/>　　然而四周的人都是看白痴一样的看他。
<br/>　　林峰在平民区内逛了大一圈，发现果然如小王所说，在这个地方的黑铁战士大部分都是些三阶以下的，偶然也会出现一些四阶的。
<br/>　　“咦，居然还有悬赏告示？”
<br/>　　林峰要出城时，在门口一个巨大的告示栏上有许多任务，各式各样类型的任务都有，其中发布者大部分都是军部，也有一些诸如战士会这样的大组织。
<br/>　　任务类型也是千奇八怪，有帮找儿子的，有帮找妈妈的，也有帮猎杀什么丧尸的，也有要求去什么地方救人的。
<br/>　　甚至还有人求某件物品的，其中大部分都是求各式各样的药剂，退烧药、生理盐水酒精纱布外加抗生素这些医疗用品最为广泛。
<br/>　　此时此刻，许多人都聚集在这里。
<br/>　　这些人看热闹的比较多，林峰走了过去，扫了眼，发现倒是有几个任务不错。
<br/>　　“援助军部猎豹小队清理XX区域，在XXX整装待发集合，推荐三阶以上的黑铁战士接取！任务成功奖励，任务进行中所猎杀的丧尸所得，都归个人所有，除此之外，军部还会奖赏十颗三阶晶核。”
<br/>　　“这个任务不错啊。”
<br/>　　林峰直接就接取了这个任务。
<br/>　　“看有白痴居然真的接取这个任务了。”
<br/>　　“这傻子估计以为奖励多想都没想就接取了。没看到推荐等级要足足三阶！”
<br/>　　“有去无回哦！”
<br/>　　“他怕是不知道这个地方很危险的吧。”
<br/>　　一群人小声议论道。
<br/>　　林峰懒得搭理他们，一群一阶二阶的对真正的力量一无所知，懂个毛他们。
<br/>　　看了眼旁边的任务告示，上面说了一些接取任务和完成任务时候的具体流程。
<br/>　　林峰扫了眼，他这边任务一旦接取后，直接离开了基地。
<br/>　　他决定先去找空投，再然后去这个什么白云镇。
<br/>　　“飞哥，您这为什么不去军部反而去这什么战士会啊。”
<br/>　　此时此刻。
<br/>　　平民区来了几个人。
<br/>　　其中一个年轻人走在最前面，后面跟着几个汉子。
<br/>　　“你懂个屁，军部那群人条件虽然给的好，但是规矩多，这战士会不一样，最主要的是，美女多。军部那个地方，都是些抠脚大汉，没有女的。”
<br/>　　飞哥道。
<br/>　　“原来如此。”几个汉子都嘿嘿笑道。
<br/>　　“飞哥，刚才进基地，那街边站着的两排小姐姐，其中不乏一些高级货色，价格也不贵，要不要今天晚上我们爽爽。”一个汉子猥琐笑道。
<br/>　　飞哥闻言也是嘿嘿一笑，“自然是要爽，到时候你去挑五个最好看的，我两个，你们一人一个。”
<br/>　　“得嘞，多谢飞哥。”
<br/>　　几个小弟都兴奋道。
<br/>　　“我去。”
<br/>　　忽然飞哥走的好好的，是忽然不动了。
<br/>　　他的眼睛都直了，盯着一个方向看了半天没有动静。
<br/>　　一群人顺着他们的目光扫去，就看到一个大白腿美女。
<br/>　　“极品啊！”
<br/>　　飞哥舔了舔嘴巴眼中露出一抹淫邪道。
<br/>　　“这腿这胸部，简直极品。”
<br/>　　几个小弟也是才回过神。
<br/>　　“飞哥，这妞好像也是战士会的人啊。”
<br/>　　飞哥点了点头，“就是战士会的人，这里是战士会的宿舍区，没想到刚来就遇到这么极品的小妞，到时候你们几个都给我注意点仪态。”
<br/>　　“放心吧，绝对不耽误飞哥泡妞。”
<br/>　　飞哥整理了下衣服，然后就大步走了过去。
<br/>　　陈勾李广吴三他们就出去猎杀丧尸了。苏云就在屋内清点着物资。
