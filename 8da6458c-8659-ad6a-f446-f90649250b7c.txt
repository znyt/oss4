<br/>　　“是啊，本来我还觉得蓝队晋级决赛肯定是非常轻松的呢，不过今天看了TM战队的表现，霄神他们还真不能掉以轻心。”

<br/>　　在前期两方都是在那里安稳的发育着，只不过到了中期的时候，god这边突然发现了情况貌似有些不对劲了，对面的中路根本就没有想要游走的想法，那个样子看起来仿佛是在牵制自己一般。
<br/>　　而在12分钟，当雷霆战队这边的打野蜘蛛配合着辅助牛头来了一波上路越塔强杀之后，god这种不好的预感越来越强烈了起来。
<br/>　　当时间到达15分钟的时候，在无数观众的注视下，此刻雷霆战队这边的辅助牛头已经是站在了对方f4这个区域，下一刻牛头直接双手砸地，紧接着白光一闪，牛头已经是出现在了god的身前。
<br/>　　没给他任何反应的时间，他手中的机械先驱便是被高高的击飞在了空中，下一刻牛头w技能直接将机械先驱朝着陆伟的辛德拉那边顶去。
<br/>　　同一刻他们这边的打野蜘蛛也是从河道草丛中走了出来，一个e技能精准的命中了还在空中飞舞的机械先驱的身上，在配合上辛德拉的伤害，机械先驱瞬间被秒杀。
<br/>　　这一刻god这边终于明白是哪里不对了，因为这一刻小刀已经不再是和中路来打联动了，而是和辅助配合了起来，这突如其来的变化，一时之间让他们没有想出来，所以才着了道。
<br/>　　只不过这个时候他们这边的劣势已经是有些大了，所以god也是没有丝毫的办法。
<br/>　　“这一局雷霆战队打的是野辅联动，就是应该如此嘛，中野被对方给针对了，我们可以打野辅，雷霆战队还有机会。”米粒大声的喊着。
<br/>　　最终这一场比赛在雷霆战队这边前期建立起的巨大优势下，TM战队这边没有丝毫翻身的机会，在时间到达33分钟的时候便是被雷霆战队推破了基地。
<br/>　　TM战队比赛室，
<br/>　　“下一场比赛你们要重点防守住对方的打野和辅助，不要让他们在打出这一局的联动来，只要针对住了他们两个，我们就能赢了。”教练抓紧在那里安排战术。
<br/>　　很快双方的第四局对决再次开始，这一场比赛的选人，雷霆战队依旧是没什么变化，和上一局的阵容完全一样。
<br/>　　而在这一场的比赛开始后，TM战队这边便是将雷霆战队的辅助看的死死的，根本就不给他一点动弹的机会。

<br/>　　只不过在这一局的比赛中，雷霆战队再次打起了中野联动，再一次打了TM战队这边一个绰手不及。

<br/>　　10分钟，小刀和陆伟来上，配合着自家的上单直接将对方的上路抓死。
<br/>　　这一场两人再次打出了属于他们的联动，虽然中间TM战队有几波反击，但是在陆伟手中的辛德拉已经发育的无限好的情况下，最终他们依旧是没有掀起什么浪花来。

<br/>　　“牛逼，这两场比赛打的真是太过瘾了，节奏始终在我们的手中，TM那边根本就没有反击的机会啊。”
