        他傅夜沉的表情可谓是要真实多了。
<br/>
<br/>        连华生越想这事，越觉得有趣儿。
<br/>
<br/>        话题到这里，连华生突然想起那件最重要的事来，便问道：“夜沉，你昨晚派人给我打包带回来的活体试验品，是作为我救了你家‘小谢’的谢礼吗？”
<br/>
<br/>        “随你怎么想。”傅夜沉淡漠地应了声。
<br/>
<br/>        连华生剑眉微挑，坏坏地笑了笑：“不承认就算了！反正，你放心。我不会给你闹出人命来的！”
<br/>
<br/>        傅夜沉抬眸瞥了连华生一眼，默不吭声。
<br/>
<br/>        就在两人尬聊的时候，刘婶领着童筱颖从楼上下来了。
<br/>
<br/>        连华生放眼望去的时候，看着童筱颖身上那件淡紫色露肩雪纺连衣裙，不禁会意地回眸看了傅夜沉一眼，意味不明地勾唇一笑。
<br/>
<br/>        还说不喜欢小谢，连人家的三围都摸清楚了！
<br/>
<br/>        今早，连华生可是亲耳听到，他傅夜沉在给某个品牌服装店的老板娘打电话，让那老板娘挑一套齐全的里里外外的衣服送过来。
<br/>
<br/>        傅夜沉可是把人家的三围和身高，一字不差地报给那位老板娘听的。
<br/>
<br/>        这不，衣服穿上了，如此合身，还穿得明艳动人。
<br/>
<br/>        要说这男人对那女人没动心，怎么都说不过去！
<br/>
<br/>        傅夜沉也只是瞄了童筱颖一眼，便刻意地将目光给挪开了。
<br/>
<br/>        童筱颖仍旧戴着那副白羽假面眼镜，一路从盘梯上下来时，看到那边餐厅里还坐着两个男人，正犹豫要不要过去打声招呼。
<br/>
<br/>        刘婶此时从她身边经过，迎面朝那两个男人走去，毕恭毕敬地对其中一个男人说道：“大少爷，小谢小姐说她要回去了！”
<br/>
<br/>        “正好我要去公司，那就一起出去吧！”傅夜沉随即放下了手里的刀叉，拿起碟子里的手帕，擦了擦嘴唇。
<br/>
<br/>        刘婶欠身应了声：“是！我去叫小陈把车开出来！”
<br/>
<br/>        连华生再次睐了一眼傅夜沉，端起餐桌上的果汁，笑意不明地喝了一口。
<br/>
<br/>        豪车内。
<br/>
<br/>        傅夜沉把驾驶座后最安全的位置，让给了童筱颖坐。
<br/>
<br/>        车内的气氛有点儿暧昧。
<br/>
<br/>        傅夜沉自己上了车，才发现，他把那天季雅姗留给他的手工珍珠包给落这车里了。
<br/>
<br/>        眼看着“小谢”就坐在自己的身边，傅夜沉拿起那个手工珍珠包，一边递给“小谢”，一边坐直了身子。
<br/>
<br/>        童筱颖一脸疑惑不解地看着傅夜沉递过来的这个手提包，愣了半天也未接手。
<br/>
<br/>        傅夜沉索性就将手里的珍珠包，塞到了童筱颖的怀里，淡淡地说道：“一个小女生送的，我拿着没用，给你！”
<br/>
<br/>        童筱颖这才提起怀里的这个珍珠包，仔细地看了看。
<br/>
<br/>        包包做工很精致，上面的珍珠莹润光滑，透着淡淡的光彩。
<br/>
<br/>        “好漂亮的包啊！”
<br/>
<br/>        童筱颖看着这包，爱不释手地赞叹。
<br/>
<br/>        傅夜沉瞥了“小谢”一眼，看到她嘴角微微上扬的那抹弧度，是她那发自内心的微笑，而心里也跟着莫名其妙地尝到了一丝甜头。
<br/>
<br/>        忽然间，他脑海里有种感情错觉，认为只要她喜欢，他就会很开心。
<br/>
<br/>        童筱颖欣赏完这个手提包，瞬间意识到自己的失态，连忙又将手提包给傅夜沉递了过去：“好像挺贵重的，你还是留着吧！”
<br/>
<br/>        “你不要的话，我就把它扔了！”傅夜沉冷哼了一声，一脸嫌弃的样子。
<br/>
<br/>        他刚准备伸手拿回那包，童筱颖又下意识地抓着包护在了怀里。
<br/>
<br/>        “那你还是给我吧！这么好看的包，不能被你给糟蹋了！”
<br/>
<br/>        童筱颖抿了抿唇。
<br/>
<br/>        傅夜沉嘴角微扬，似笑非笑地转头望向了车窗外。
<br/>
<br/>        此时此刻，他完全忘记了，这个手工珍珠包，他应该要留给苏漫雪才对。
<br/>
<br/>        无意间，童筱颖的目光落在了这个男人的右手腕上。
<br/>
<br/>        一条精致的青丝手链，吸引了她的注意力。
<br/>
<br/>        那是……
<br/>
<br/>        童筱颖微微皱了下眉头。
<br/>
<br/>        回想起，那天，她在给傅氏集团的未来继承人设计总裁办的装修图时，他突然出现，强吻了她后，还拿匕首割断了她的一绺长发。
<br/>
<br/>        难道……
<br/>
<br/>        他手腕上的青丝手链，是用她的长发编织的吗？
<br/>
<br/>        或许是她多心了！
<br/>
<br/>        童筱颖若有所思地沉默了。
<br/>
<br/>        两人之间没再继续任何话题，车内也格外安静。
<br/>
<br/>        车子驶入市中心后，童筱颖便要求下车。
<br/>
<br/>        傅夜沉也不挽留，她说要下车，便让司机小陈靠边停车，放她下去了。
<br/>
<br/>        童筱颖下车后，傅夜沉的目光这才敢落在她的背影上。
<br/>
<br/>        他目送她的背影，走入人群之中，直到人来人往，彻底将她的背影给淹没，他才将目光收回，淡淡地说了声：“开车。”
<br/>
<br/>        童筱颖回到家门口后，才想起自己把手机和钥匙落会所了。
<br/>
<br/>        不过幸好，家门还可以用指纹和密码打开。
<br/>
<br/>        童筱颖开了门后，一进家门，就开始翻箱倒柜的找东西。
<br/>
<br/>        她找了老半天，终于在自己的行李箱内拉链袋里找出了那条陨石项链。
<br/>
<br/>        这条项链，也是那个男人送给她的。
<br/>
<br/>        而且，她笃定，那个男人并未认出她来，纯粹只是认为她是“夜惑”里的一个歌女而已。
<br/>
<br/>        这样一来的话，童筱颖忽然觉得那个男人就是一个名副其实的花花公子。
<br/>
<br/>        所以，他送的东西，也没什么好留恋的了。
<br/>
<br/>        其实，她就从未留恋过。

<br/>        童筱颖拿起这条陨石项链，和那个手工珍珠包，出了门去了“夜惑”。

<br/>        当她把这两样东西递到老板费尔南面前时，费尔南却只是好奇，这是哪个男人私底下送给她的。
<br/>
<br/>        “费老板，我知道夜惑有地下拍卖场，你帮我把这两样东西的价格给炒起来！我分三成给你，如何？”童筱颖柳眉扬了扬。
<br/>
<br/>        “你这两样东西值不了几个钱！”费尔南故作淡定地白了童筱颖一眼。
<br/>
<br/>        童筱颖却拿着陨石项链，吊在半空中，在费尔南眼前晃了晃：“那个包包可能也就一万块钱的样子，但是这条陨石项链，一定价值不菲！”
<br/>
<br/>        她现在真的很缺钱啊！
<br/>
<br/>        费尔南勾唇一笑，发现自己忽悠不了这小姑娘后，便开始正经八经地跟童筱颖谈口头生意：“五五分，我就帮你把这两样东西拿去‘夜惑’的地下拍卖场炒价！”
