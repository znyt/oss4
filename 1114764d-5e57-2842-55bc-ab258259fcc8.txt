　　那位‘雷叔’接着又道：
<br/>　　“四公子，此人能以区区金丹境，对抗五阶妖兽，就算是在那些高级星域，那也算的是极为逆天的存在，如果可以的话，我们务必要和此人结交一二！”
<br/>　　那青年男子听到这话，顿时两眼一亮，
<br/>　　“雷叔您言之有理！先不说此等人物，背景必定很不简单，单单是他具备化神境的实力，只要能为我所用，我必定可以一举压过二哥，到时候成为万宝阁的继承人，也就轻而易举了！哈哈！”
<br/>　　就在这位‘四公子’和‘雷叔’谈话之时，另外一边的战局，也已经进入了白热化状态。
<br/>　　轰！
<br/>　　那吞天蟒再次甩出巨尾，叶尘当即举起紫琼剑抵挡。
<br/>　　铿！
<br/>　　刺啦！！
<br/>　　那吞天蟒的皮甲之硬，竟然堪比圣器，以叶尘如今的功力，即便手持紫琼剑这等准仙器，竟然也无法破开它的防御！
<br/>　　而在没有施展逆生百转的情况下，叶尘也只能凭借身法优势，只守不攻，方才能勉强抵挡吞天蟒的攻势。
<br/>　　“虽然这头吞天蟒，比起真正的化神天君，还是要弱上一些，可是防御变态，又耐打，刚好是我的克星，倘若我施展出全部底牌，依旧无法干掉这畜生，那我可就凶多吉少了！”
<br/>　　叶尘一边躲闪着吞天蟒的攻击，心中思绪纷飞，一时间也不知道要不要施展逆生百转。
<br/>　　而就在叶尘犹豫之时，忽然一道惊天剑气，自远处爆射而来！
<br/>　　轰！
<br/>　　那道剑气，快如迅雷，竟然直奔吞天蟒的眼睛！
<br/>　　“嗷！！”
<br/>　　吞天蟒被那道剑气刺中之后，顿时发出一道凄厉的怒吼声，庞大的身躯撞在一座上峰之上，后者顿时轰然倒塌，连同整个大地都跟着一阵晃动。
<br/>　　叶尘这才发现，不知道何时，远处天空之上，出现了一名须发尽白的老者，而且气息浑厚，手持一把青色长剑。
<br/>　　而在那老者的后方，竟然还有数十名气息不弱的修真者，纷纷呼啸而来。
<br/>　　“布阵！”
<br/>　　随着那白发老者一声令下，那数十名高手，当即按照某种阵列，对吞天蟒形成合击之势。
<br/>　　轰轰轰！
<br/>　　那吞天蟒先是被白发老者偷袭得手，然后又被众人合击围攻，一时间竟然落入了下风。
<br/>　　叶尘反应过来之后，顾不得去思考，这些人为什么要帮他，也当即加入战团之中，而且专门攻击吞天蟒最脆弱的地方。
<br/>　　“嗷嗷嗷！！”
<br/>　　吞天蟒体型庞大，如果是一对一，自然具有体型上的优势，可是被众人围攻，优势就变成了劣势，身上开始频频受伤，发出一阵阵惨叫。
<br/>　　“嗷！！”
<br/>　　吞天蟒的七寸位置，再次被叶尘刺出了一个血洞，顿时哀嚎了一声，突然冲出人群，钻入下方的深谷之中，竟然选择了逃跑！
<br/>　　“别追！凭我们这些人，根本杀不死这畜生，见好就收吧！”
<br/>　　白发老者一声令下，众人无有不遵，纷纷停止了追击。
<br/>　　“多谢诸位出手相助！”
<br/>　　赶跑了吞天蟒，叶尘走上前，冲着那白发老者一拱手，开口道。
<br/>　　虽然他隐约觉得，这些人不会无缘无故出手，不过到底也算是帮了自己。
<br/>　　白发老者哈哈一笑，十分客气道：
<br/>　　“其实无需我等出手，小友应该也能全身而退，反倒是我等贸然出手，还望小友莫怪！”
<br/>　　白发老者说完，叶尘还没有开口，旁边一名相貌颇为英俊的青年男子，跟着开口笑道：
<br/>　　“相逢即是有缘！鄙人万宝阁张凌云，阁下若是不嫌弃，不如到我船上一叙可好？”
<br/>　　叶尘眉头微微一皱，
<br/>　　万宝阁？
<br/>　　他隐约有些印象，可毕竟时间太过遥远，一时半会却又想不起来。
<br/>　　而且他着急赶往天玄城，哪里有时间在这儿耽搁？
<br/>　　“抱歉！我有急事要前往天玄城，实在没时间！”
<br/>　　叶尘说完这话，便准备转身离开。
<br/>　　不料，
<br/>　　一道残影在空中划过，
<br/>　　一名元婴境初期的老者，陡然拦在了叶尘的身前，喝道：
<br/>　　“大胆！我家公子是何等人物？屈尊降贵邀请于你，你竟然还敢拒绝？你这是不给我万宝阁面子吗？”
<br/>　　叶尘见状，眉头顿时皱的更紧了，
<br/>　　若非对方之前帮过自己，早就一巴掌将此人拍成肉饼了！
<br/>　　“让开！”
<br/>　　叶尘也懒得和他废话，直接一声低喝，同时夹杂着精神攻击。
<br/>　　蹬蹬蹬！
<br/>　　那老者顿时如遭雷击，一连倒退了十几步，方才站稳身形，体内气血翻滚，脸色煞白。
<br/>　　其余众人见状，也顿时个个脸色大变，当即排开阵势，将叶尘围在了中央。
<br/>　　“都给我住手！”
<br/>　　关键时刻，那名白发老者，再次站了出来，缓缓走到叶尘跟前，微微一笑道：
<br/>　　“手下人不懂事，小友莫怪！”
<br/>　　“小友既然有急事，我等自然不便强留！”
<br/>　　“不过，我等也刚好要返回天玄城，而且我们的追风穿云舟，可日行五十万里，小友不如和我们同行，也可早日抵挡，不知你意下如何？”
<br/>　　白发老者说完这话的同时，冲着后方一指。
<br/>　　只见远处一艘庞大的金色飞船，转眼之间，便已经飞驰而至。
<br/>　　叶尘不由得眉头微微一挑，
<br/>　　眼前的这艘飞舟，应该是圣器级别的飞行法宝，比普通的灵器飞舟，速度要快上十倍不止，而且只要灵石充足，便可以日夜兼程，中间不需要停歇，的确比乘坐九翅紫电鹫要方便多了。
<br/>　　而且，他到天玄城之后，还要去寻找神魂类的高阶灵草。
<br/>　　这些人一看就知道来头不小，说不定可以帮上忙。
<br/>　　当然了，对方如此讨好于他，必然有所图谋。
<br/>　　但是以眼前这些人的修为，实力最高的那位白发老者，也不过才元婴巅峰境而已。
<br/>　　只要他小心行事，应该问题不大。
<br/>　　一念及此，叶尘也不矫情，当即点头道：
<br/>　　“如此，那我就恭敬不容从命了！多谢！”
<br/>　　白发老者和青年男子，见叶尘答应了下来，顿时纷纷露出一抹喜色。
<br/>　　很快，
<br/>　　叶尘将幻化成人形的九翅紫电鹫和北山薇儿，也召了过来，一同登上了那艘金色飞舟。
