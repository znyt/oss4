        连华生见傅夜沉不爱听他这话，于是又主动岔开了话题，打趣地说：“我今日是从临海城赶过来的，除了替上级带资料给你，还去了一趟傅家大庄园探望冷爷爷，无意间听到冷爷爷跟林贵叔说，他当太爷爷了心里其实还是挺高兴的，那小曾孙也有两岁多了，长得很可爱。”
<br/>
<br/>        他虽是打趣，实则是想打消傅夜沉对童筱颖仅存的最后一丝幻想。
<br/>
<br/>        爷爷傅炎廷就他傅夜沉和弟弟傅昼景两个亲孙子，能让爷爷傅炎廷高兴自己当了太爷爷的事情，自然是他自己的直系亲孙有了儿子。
<br/>
<br/>        傅夜沉心口顿时闷得发慌，隐隐地痛感，再次袭击他全身的每处敏感神经。
<br/>
<br/>        筱颖给阿景生了个儿子，他跟着也当了大伯。
<br/>
<br/>        很显然，得知这件事情的傅夜沉，脸色看起来并不大好。
<br/>
<br/>        连华生顿时觉得，自己这心直口快，是不是把傅夜沉的心给伤了。
<br/>
<br/>        都三年过去了，他的心怎么还是会痛？
<br/>
<br/>        爱得有多深，痛得就有多厉害……
<br/>
<br/>        傅夜沉恍神间，想拿起办公桌上的茶杯喝水，结果连杯子都未拿稳，杯子里的水全洒在了文件上。
<br/>
<br/>        “阿沉，你……没事吧？”连华生上前一步，连忙从桌上抽出几张纸巾，一边替傅夜沉擦拭着文件，一边关问道。
<br/>
<br/>        傅夜沉摆摆手，故作淡定地转开话题：“上级的文件呢？拿给我看看。”
<br/>
<br/>        “好！”连华生放下手里的活，将打湿的纸巾扔进了一旁的垃圾桶中，随手从自己的上衣内兜里，掏出一个小小的黑色笔记本递给了傅夜沉。
<br/>
<br/>        傅夜沉拿起这小本子，认真地翻看了一番后，深沉地说道：“这件事，很难办！”
<br/>
<br/>        “难办，咱们也得去办。”连华生一脸无奈地耸了耸肩。
<br/>
<br/>        澳洲，悉尼大学附近的联排别墅区。
<br/>
<br/>        童筱颖抱着小爱，盘腿坐在了客厅里的茶几前的地毯上，拿着笔记本电脑开始投简历。
<br/>
<br/>        其实，她可以免试直接进入傅昼景所在的建设公司工作的，但是因为出了季思妍的这件事情，她忽然间不想再天天面对傅昼景了。
<br/>
<br/>        以前，她总想着如何把两个人之间的距离拉近，现在，她却开始在逃避两个人之间的亲密。
<br/>
<br/>        童筱颖对傅昼景产生了心理隔阂，而这道隔阂却在今天再一次登堂入室。
<br/>
<br/>        门铃响起来后，童筱颖透过猫眼看到是季思妍，很不想开门。
<br/>
<br/>        季思妍却不依不饶地在外大声喊门，哀求道：“筱颖，我知道你在家。求求你开个门，好不好？我真的有事情想求你！”
<br/>
<br/>        她来求她，无非是让她生个孩子救她的孩子的事情。
<br/>
<br/>        童筱颖充耳不闻，抱着小爱，起身上了楼。
<br/>
<br/>        过了许久，门外没了动静。
<br/>
<br/>        童筱颖从楼上下来，打算出门去买些食材回来做饭吃。
<br/>
<br/>        当她打开门后，一个稚嫩的声音，从眼皮子底下传来。
<br/>
<br/>        “妈咪，抱抱！”
<br/>
<br/>        季小景坐在门口，仰着小脑袋，可爱地向童筱颖张开了双臂。
<br/>
<br/>        童筱颖看着这个私生子，一脸不悦地皱起了眉头。
<br/>
<br/>        而季小景的身边，还放着一个天蓝色的长方形的笔记本。
<br/>
<br/>        童筱颖随之四下看了看，并未发现季思妍的身影，于是拿起那个天蓝色的笔记本翻看了起来。
<br/>
<br/>        筱颖，我现在澳洲找到工作了，但是需要去总部培训一个星期，这人生地不熟，我只能将小景托付给你照顾一个星期。是我教小景叫你“妈咪”的，在我看来，小景是阿景的儿子，你又是阿景的妻子，自然也是小景的半个妈妈。小景的生活日常与饮食习惯，还有一些急救措施，全在本子里，你可以看看。谢谢你，筱颖。
<br/>
<br/>        这一翻开笔记本，就看到了这张留言条，让童筱颖很是无语。
<br/>
<br/>        季思妍的心到底有多大？还是她有多相信她童筱颖的为人。
<br/>
<br/>        一个小三，竟然让原配照看她的孩子，难道不是笑话吗？
<br/>
<br/>        童筱颖恼火地站起身来，随手扔了手中的笔记本，转身进屋，“乓”地一声，重重地关了门。
<br/>
<br/>        “呜——哇！”或许是这关门声，惊吓到了门外的那个孩子，稚嫩响亮的啼哭声，从门外一阵又一阵地传来。
<br/>
<br/>        孩子哭得撕心裂肺，童筱颖听得也十分揪心。
<br/>
<br/>        而现在，面对这种状况，到底要她如何抉择？
<br/>
<br/>        季小景从地上站起来，走到大门边，一边拍打着门板，一边用稚嫩的声音哭喊着：“妈咪，开门门！呜呜——哇哇——”
<br/>
<br/>        隔着一张门板，童筱颖也站在门边无动于衷，却在潸然泪下。
