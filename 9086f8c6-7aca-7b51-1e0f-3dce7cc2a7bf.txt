　　望着闭目中的云霆，慕清霄点在他眉心的中指与食指不断下移，最终停留在腹部位置。
<br/>　　“磁磁……”
<br/>　　正当他准备调动灵力时，云霆身上冒出一抹抹雷弧，好似在抵抗一般，后者仿若未觉，雷弧也没有损伤他一丝一毫。
<br/>　　见状，慕清霄目光一凝，雷灵珠似乎在云霆体内待习惯了，并不希望脱离宿体。
<br/>　　当然，脱不脱离可由不得它。
<br/>　　只要雷弧不会伤害到云霆，如此一来，慕清霄就没有后顾之忧可言了。
<br/>　　星眸中闪过一抹黑茫，玉额上黑色神纹浮现，慕清霄指尖隔空一点，一股强大的吸力自手心中产生。
<br/>　　“轰轰……”
<br/>　　伴随着震耳欲聋的雷鸣，一道道蓝色闪电自云霆体内掠出，后院各处皆是被轰成粉末。
<br/>　　其中一道击在结界上，令得结界上荡起一抹涟漪。
<br/>　　慕清霄脸上牵起一抹弧度，好在雷灵珠不蠢，知晓云霆身体能承受的极限，若是肆无忌惮的释放雷霆，后者的身体恐怕会因此奔溃。
<br/>　　“轰！”
<br/>　　伴随着雷鸣声，慕清霄眼神一眯，脸色瞬间变得苍白一片，甚至嘴角溢出一丝金色的血液。
<br/>　　只见慕清霄胸口的位置，此刻正通着一个拳头大小的洞口，洞口周围已经焦黑一片，甚至通过胸前能看见身后的小湖。
<br/>　　明显，刚才那一道恐怖的雷霆，直接贯穿了他的身体。
<br/>　　然而，慕清霄手中的吸力并没有因此而下降，反倒不减反增，而云霆的身体几乎已经被雷弧包裹其中。
<br/>　　“噼噼啪……”
<br/>　　雷光闪烁间，慕清霄的胸口处，陡然冒出一团黑色火焰。
<br/>　　在黑色神火冒出的同时，雷灵珠似乎察觉到某些气息，雷弧也渐渐变的弱小起来。
<br/>　　只见，在黑色火焰的焚烧下，伤口焦黑部位脱落，细胞组织瞬间重生，狰狞可怖的洞口也以肉眼可见的速度恢复。
<br/>　　浴火重生！
<br/>　　待火焰消散，洞口已经消散，取而代之的是粉嫩如同新生婴儿一般的肌肤。
<br/>　　“嗡嗡……”
<br/>　　此时，云霆腹部所在的空间轻微一阵扭曲，旋即一颗深蓝色的圆珠就呈现而出，其上隐隐有雷弧闪过。
<br/>　　雷灵珠。
<br/>　　望着悬浮在空中的深蓝色圆珠，慕清霄瞥了眼自己的胸口，若有所思。
<br/>　　轻轻一卷，便将雷灵珠握在手中，奇怪的是，雷灵珠似乎放弃了挣扎，静悄悄的躺在他的手心当中。
<br/>　　取珠出乎意料的顺利，旋即，魂剑出现将雷灵珠带进了阴阳玉。
<br/>　　随后，慕清霄双手结印，布置在四周的空间结界也消失不见……
<br/>　　“雷灵珠已经取出，不会在妨碍到你的正常生活。”
<br/>　　云霆渐渐睁开双眼，眼底闪过一抹黯然，他能清晰的感觉出，寄宿在体内的雷霆力量确实已经消失的无影无踪。
<br/>　　“你答应过，保雷州百年无忧。”
<br/>　　慕清霄微微一笑，道：“我承诺过的事情，自然不会食言，我会在雷州居住一段时间，将后院暂时让给我如何？”
<br/>　　“请随意。”
<br/>　　云霆话语刚落，云垂野就快步走了过来，打量了自己儿子一圈，道：“云儿，你没事吧？”
<br/>　　“父亲，孩儿没事。”
<br/>　　“那就好，那就好啊！”
<br/>　　说罢，云垂野看向慕清霄，道：“多谢公子，云某这就命人给公子收拾客房……”
<br/>　　不多时，待云府侍女将后院的住宅打扫干净后，慕清霄也没有不磨叽，当下命人找来九柄玄铁剑，二十七柄精钢剑。
<br/>　　以云府在雷州的地位与产业，寻找这些不错的凡剑也是花费了不少的心思。
<br/>　　跟云霆父子商量过后，终于决定将阵法地点定在了云府后院。
<br/>　　站在院落中央，望着堆积成一堆的玄铁剑，慕清霄隔空一点，地上的玄铁剑与精钢剑如同有灵一般，发出一阵清脆的剑鸣。
<br/>　　旋即，以慕清霄为中心，三十六柄长剑悬浮而起，每一柄长剑都散发着锐利至极的气息。
<br/>　　“嗡嗡……”
<br/>　　将无上剑意贯入每一柄长剑中，九柄玄铁剑为阵眼，另外二十七柄精钢剑为剑阵。
<br/>　　剑阵阵眼形成，三十六柄长剑也隐没与空中，凡眼根本看不出一丝一毫的端倪。
<br/>　　蜀山派的《三十六天罡剑阵》，需三十六个蜀山精锐弟子才能完整发动，威力强大到能够彻底消灭魔尊。
<br/>　　当然，魔尊代表的是强大的魔头，并非魔尊重楼那种等级。
<br/>　　而且，由三十六柄凡剑组成的剑阵，水份自然不小，但威力也不可小觑，若是剑阵发动，足以在顷刻间抹杀化神期以下的妖魔鬼怪。
<br/>　　现在，雷州没有的雷灵珠，此处就再也没有任何能够吸引妖魔的东西。
<br/>　　所以，由凡剑组成的剑阵，足以保雷州百年无忧，至于百年后，便是修仙门派的事情了，与他无关。
<br/>　　将剑阵布置好后，慕清霄瞳孔变得猩红妖异，三颗勾玉浮现，在阵眼上施加了一道普通的幻术，以免有人破坏。
<br/>　　事后，将阵法布置好的事情告知云霆父子后，慕清霄并没有就此离开。
<br/>　　得知慕清霄会在云府住上半年，云垂野自然非常的乐意，并且告知，想住多久都不是问题，其中颇有讨好的意味。
<br/>　　对此，慕清霄当然清楚云垂野心思，将当初古藤老人给他的蜀山卷轴中的仙术通通传给了云霆，并且加以指点。
<br/>　　云霆的资质有限，慕清霄所做已经是仁至义尽，接下来就只能看他自己的造化了。
<br/>　　将琐事处理完毕，慕清霄盘坐在后院的小湖泊旁，双手结印，结界向四周扩展而出，准备开始炼化雷灵珠。
<br/>　　有了土灵珠的前车之鉴，他也已经做好被雷霆蹂躏的心理准备。
<br/>　　同时也有些期待，雷灵珠中的雷霆力量正好能起到锻体的作用，甚至会比土灵珠的重力效果更佳也说不定。
<br/>　　外物防御根本没有任何用处，慕清霄也就没有在这方面花心思。
<br/>　　想要炼化灵珠，就必须承受的住灵珠中能量性质带来的实质性伤害，否则都是白搭。
<br/>　　这也是某些人族或妖族在获得灵珠后，不敢炼化的原因，比如古藤老怪，又比如极乐世界中的火鬼王。
