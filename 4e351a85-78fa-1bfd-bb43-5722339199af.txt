　　第316章
<br/>　　以程大雷多年玩游戏的经验，对这种状况倒也是并不陌生。比如在战略游戏中，某个基地可以开采出铁矿，某个基地可以开采出铜矿，想要都开采出来，就只能开分基地。
<br/>　　这样有好处也有坏处，好处是只要自己多开几个三级基地，就能买到各种各样的商品。坏处是，目前买不到可以大幅度提升实力的火药。
<br/>　　是不是也要将落玉寨生到三级。
<br/>　　还有一种可能，就是三级山寨只能买到同一等级的商品，四级山寨可以买到更高等级的商品。
<br/>　　开三级本，对程大雷来说已经相当困难，目前他还没有开四级山寨的打算，当然，也没有这个条件。
<br/>　　既然进入商店一次，就不要白来，反正一万的进店费已经交了。程大雷干脆买了一千件皮甲，即便一套需要一万恐惧值，以他现在拥有的恐惧值，也是买得起的。
<br/>　　毕竟是加三围的装备，正好也给蛤蟆寨换上统一制服。
<br/>　　然后，程大雷屋里就堆满了各种尺码的皮甲。程大雷愣了愣，突然出现这么多，自己该如何给大家交待。
<br/>　　恰在这时候，徐神机走进来，看到这一幕也吓着了。
<br/>　　“大当家，这是哪里来的？”
<br/>　　“呃……哈哈，我让云中龙订做的，刚送来，给弟兄们换上吧。”
<br/>　　“可是我今天没看见有人过来呐？”
<br/>　　“呃……”程大雷无语，需要这么刨根问底么。
<br/>　　徐神机看出了程大雷的难堪，微微倾身，压低声音道：“莫非是仙人送来的。”
<br/>　　“呃……这个……哈哈”
<br/>　　“放心，大当家我明白，不会说出去的。”徐神机一副了然于胸的态度。
<br/>　　程大雷轻轻点头，不置可否。心道这算无一策还真是算无一策呐，永远和正确答案无缘。
<br/>　　让人进来，把皮甲都抬出去，蛤蟆寨进行了统一的换装仪式。
<br/>　　大家对换上新衣服，还是蛮感兴趣的，在临近年关时，山寨一副欢声笑语之声。
<br/>　　各人取了各自尺码的皮甲，迫不及待的穿在身上。很快程大雷面前就黑压压聚集了一大片人，这些皮甲是统一的黑色，和帝国的制式军装类似，穿上后能护住胸口，肩膀手臂，且又活动自如。
<br/>　　而在皮甲的背部，纹着统一的图案……一只四蹄张开的大蛤蟆。
<br/>　　程大雷的头又开始疼了，蛤蟆，为什么又是蛤蟆，难道自己这辈子就摆不脱蛤蟆了。
<br/>　　大家穿上统一的制服后，皆感觉动作比以前更灵活，力气好像比以前更大。程大雷知道这并非是诸人的错觉，毕竟这衣服是加防御、体力、敏捷的。
<br/>　　而统一的黑色制服，更带来一种无形的威压，在程大雷面前整整齐齐站成队伍……像一支精兵了。
<br/>　　只不过诸人腿上的裤子，脚上的靴子还是千奇百怪，有布鞋，有皮靴，也有草鞋……程大雷回头准备去城里定制一批，要换就干脆都换了，来个一黑到底。
<br/>　　进入冬天，天气越来越冷，河面已经开始结冰。冬天，本来该是个休养生息的日子，但程大雷不敢闲着。
<br/>　　随着天气转寒，河面会结上坚冰，程大雷隐隐有种感觉，等河面结冰，人马可行的时候，莫鸣米绝对不会放过自己。
<br/>　　准备工作必须提前进行，队伍的训练仍在继续，而关于指挥者的素质培养，也不能放过。
<br/>　　大家白日训练，夜里，程大雷开起了课堂。
<br/>　　首先是统一教大家识字，这一项由李婉儿负责，然后是开拓诸人的眼界，这一项由李行哉负责，他去过很多地方，称得上见多识广。
<br/>　　至于军事课，由程大雷亲自负责。程大雷本来是个门外汉，倒也不敢自居领兵打仗的高手，他授课时，还是让大家分析讨论每一次战斗，那些优点可以吸取，那些缺点应该避免。不过程大雷毕竟是一路杀过来的，还是有些见地的。
<br/>　　开始授课后，就可以看出每个人的不同。有些不求上进的，让他们提刀比提笔容易，而有些人则是每次都到，在其他时间脑袋里想的也是这些事。
<br/>　　对于懒惰的，程大雷自然也会提醒惩罚，但程大雷却也知道，外力总不能赶着他走。如果他不想往前走，只能被后来者取代。
<br/>　　而在授课的时候，程大雷的思路也被打开，以前没有想过的问题，现在却需要认真想一想。
<br/>　　这一日黄昏，程大雷在演武场上授课，今天程大雷倒是不聊军事方面的事，他想聊一聊企业文化。正义教都有个还天地光明的信仰呢，为什么蛤蟆寨不可以有。
<br/>　　一个共同的目标，可以把大家凝聚在一起，让队伍变得更团结。但至于是如何的目标，程大雷自己也没想过。
<br/>　　是解救苍生，还世间太平，还是抢钱抢粮抢婆娘，专心培养小loli……说来说去，程大雷说出口的还是四个字：好好活着。
<br/>　　“呔，姓程的，你给我滚出来！”
<br/>　　程大雷正和大家侃互帮互助，团结友爱之类的话，突然外面传来一阵骂声。
<br/>　　大家看程大雷的眼神有些不对了，这明显是睡了人家大姑娘的态度么，刚才大当家不还说要不欺妇孺，不抢老弱么……
<br/>　　程大雷已经听出了这声音是谁，跑到外面一看，果然不假：樊梨花提着绣绒刀，骑着胭脂马和初见时一模一样。
<br/>　　“樊寨主，今天怎么有时间过来了，我们里面说话？”
<br/>　　“呔，程大雷你住了吧，你自己做过的事情，你忘了么！”
<br/>　　众人都看着程大雷：大当家，你就承认吧，看这态度，肯定是已经有了。
<br/>　　“这个……我做过什么事情了？”程大雷一脸懵圈。
<br/>　　“大家都知道落玉寨也是蛤蟆寨的弟兄，为何蛤蟆寨都换了新衣服，我落玉寨却没有。”樊梨花气鼓鼓道：“程大雷，你手心手背分得够清。”
<br/>　　程大雷一楞，倒是疏忽了这件事。他将樊梨花恭恭敬敬的请到自己办公室，呃……也就是书房，避免了被人围观。
<br/>　　可看樊梨花的意思，也不是单纯奔着衣服来的，那眼神中淡淡的幽怨……程大雷一激灵，莫非是奔着自己。
<br/>　　都说挥剑斩情丝，能斩断的也就不是情丝了。
