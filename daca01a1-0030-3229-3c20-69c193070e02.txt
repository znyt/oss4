　　第290章
<br/>　　牧长庚恭恭敬敬给徐神机倒了杯茶，撑着疲惫的身躯，脸上陪着笑意。
<br/>　　“徐军师，我就是奔着大当家来的，可大当家有多大本事，您好好和我讲讲。”
<br/>　　“嘿，若说大当家的本事可多了，其他我也不和你聊。”徐神机突然压低声音：“你可知道大当家一身本事是从何处来的？”
<br/>　　“还要向军师请教？”
<br/>　　“那可是神仙梦中传艺。”徐神机声音更低，像在诉说天大的秘密。
<br/>　　“这个倒是也听人说起过，只是，这世上真有神仙么。”
<br/>　　“嘿，别说你，我一开始也不信呐。可后来的事情，却是让人不得不信，我从小看着大当家长大，大当家各方面都挺普通，平平无奇。可就在老寨主死后，大当家像换了一个人似的，多知多懂。”徐神机侃侃言道：“天上地下，无有不知，无有不晓，这倒也罢了，大当家还会神仙法术。”
<br/>　　“这是真的？”牧长庚睁大眼睛。
<br/>　　徐神机很享受对方此时脸上的表情，道：“何止是真的，比真金还真，五雷法，呼风唤雨，撒豆成兵，这都是我亲眼见过的，难道还能诓你不成。”
<br/>　　牧长庚有些含糊了，的确，也听说这程大雷有些诡异之术。但青天白日，朗朗乾坤，哪有那么多神仙鬼怪。只不过，徐神机说得绘声绘色，斩钉截铁，也不由得他不信。
<br/>　　“呵呵，这么说谁也攻不破我们山寨了，这当真是再好没有。”牧长庚笑着，笑容有些苦涩。
<br/>　　“没有法子那是别人，老夫可不是没有法子。”徐神机吹得有些刹不住车，一时间精神头上来了：“老夫身为大当家的军师，什么不得替他想到。我这些日子苦思冥想，想的就是破寨的法子，破寨的法子想到了，自然就有了守的法子。”
<br/>　　牧长庚眼睛一亮，这个法子高啊。换位思考，自然更容易知道山寨防守的破绽。都说徐神机神机妙算，平常倒也没有看出来出奇之处，今日方才知道真人不露相，露相不真人。
<br/>　　他忙陪着笑容：“那军师已经想到法子了？”
<br/>　　“这个自然，只要……”徐神机一警，方才想到眼前这人可是个二五仔，自己苦心想来的法子，怎么能与他讲。他摆摆手道：“嘿，我和你聊这个干嘛，睡觉睡觉。”
<br/>　　“随便说说么，我也向军师学习学习。”
<br/>　　“你学不到的，某些事情是讲天赋的。”徐神机打了声哈欠，歪在枕头上睡着了。
<br/>　　徐神机打着呼噜，牧长庚却是久久难以入睡。知己知彼，方才能百战百胜，若要找破寨之法，还能有什么比徐神机脑海中的法子更高么。如果自己能把他心里的话骗出来，倒也可以回去交差了。
<br/>　　只是这个人城府深沉，外表看上去粗枝大叶，内心却很警醒，刚才自己一直追问，已经惹得他警惕，却是不可再鲁莽，如何步步为营，获取他的信任后，再骗他吐口。
<br/>　　漫漫长夜，如此过去。
<br/>　　第二天天还没亮，牧长庚就去铁匠铺打铁。他心里有了目标，做事有了精神头，铁匠铺沉重的工作他竟然撑了下来，而且表现很好。
<br/>　　程大雷甚至还亲口表扬过他，牧长庚雄赳赳气昂昂，觉得自己已经成功潜伏下来，获取了程大雷的信任。
<br/>　　当然，更重要的事情，是如何骗开徐神机的嘴。
<br/>　　程大雷也没有太多时间关注牧长庚，他每天有很多事情要忙，牧长庚的重要性要排在序列号后几位。
<br/>　　通过一战，程大雷在琴川站稳了脚跟，十八家头领表面上已接受他的存在，甚至会给程大雷支付一定的保护费。
<br/>　　而练兵的工作也没有停下来，目前已经由场地训练转为实战演练。五支队伍之间展开竞争，每日奔波在大河两岸，所操练的自然是抢劫。在这个过程中，各人的组织能力，指挥能力也在稳步提升，现在他们能组织一百人，不远的未来，他们就能指挥更多的人。
<br/>　　这一日，程大雷亲自参加了抢劫活动。
<br/>　　他加入的是豹字队，由高飞豹率领，他们这次的目标，是琴川关附近的一处马场，算是给莫鸣米养马的。
<br/>　　抢劫军队，虽然只是杂牌军，难度也不小。程大雷这次没打算亲自上阵，他主要是为看看，尽管这么长时间的训练，这支队伍配合得如何。高飞豹有没有从曾经的莽汉，成长为一名合格的指挥官。如果高飞豹都可以合格，其他人就更加可以放心了。
<br/>　　“大当家，你就放心吧，消息我们都打听清楚了。对方只不过有一二百人，几百匹马，当天去，当天回，出不了意外。”
<br/>　　接连几场胜仗，高飞豹也抖了起来，骑在马上趾高气扬。
<br/>　　“嗯，但也不得不小心，对方都是军人，装备要比普通的山贼强。你若到时候没抢了别人，反而被别人抢了，那可就丢人了。”程大雷佩剑骑马，与高飞豹并肩而行。
<br/>　　“行啦，行啦，大当家你不用哪壶不开提哪壶，今时不同往日，今天你就看我表演吧。”
<br/>　　“好，我就看你表演了。知道我为什么今天别的队不看，只看你的对伍么，还不是因为觉得你比他们强。”
<br/>　　“当然了，当然了。”高飞豹更加得意。
<br/>　　程大雷忽然腹内一阵绞痛，道：“等我一下，我去旁边方便一下。”
<br/>　　“那你快点，别误了时辰。”高飞豹时间观念还是很强的。
<br/>　　程大雷下马走进路边的小林子中，刚方便完，还没回过神来，忽然感觉头顶风声响动，他猛抬头，一根套索从天而降，将他吊在空中。
<br/>　　程大雷心内一惊，身子在空中一折，已经挣开套索，双脚稳稳落地。手摁在剑柄，目光警惕着周围。
<br/>　　“什么人，敢来对付你程爷爷，出门没看黄历么？”
<br/>　　“嘿！我在这里。”
<br/>　　身后忽想起一个清脆的声音，程大雷回过头，看到一个身着戎装的女人站在自己身后，笑若梨花。
<br/>　　樊梨花。
<br/>　　程大雷楞了楞，也就这一愣的光景，一块黑布从天而降，将他罩在其中。
<br/>(本章完)
