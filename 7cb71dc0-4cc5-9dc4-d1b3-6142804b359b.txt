　　轰隆隆！
<br/>　　四座巨大的道门将四柄凶剑崩飞，杀气磨散。四剑恢复了本来的面目，散落在地，剑身之上，光芒暗淡。
<br/>　　又过片刻时间，通天教主的三尸自身亦被元始分身打败，浑身是伤。
<br/>　　“如何？”元始目光平静地盯着眼前的通天。
<br/>　　通天不语，闭上双眼，好似在感受着什么，不过一时三刻，通天便再次睁开双眼，与此同时，他全身上下爆发出一股更加强烈的气势，轰破周边空间，令虚空为之一颤。
<br/>　　“嗯？”元始皱了皱眉，一挥袖便将周边禁锢，气息丝毫没有外泄，外人并不知道阵内情况。
<br/>　　通天突破了，成了圣人四重天，自前期步入中期，在诸多圣人之中，也只有太上到此境界，就连接引道人也差上一筹。
<br/>　　“恭喜三弟突破！”元始微微一笑，道。
<br/>　　通天平息体内气息浮动，将三尸与四剑收起，对着元始拜道：“这还得多谢大兄相助！”
<br/>　　通天自然明白这是怎么回事，他若是自己突破，还需花费上千年时间，而此次大战是元始借机在自己面前展示各种大道，清晰如实质般的道理展现在他面前，让他对道的领悟迅速加快，这才导致了他此刻突破。如若不然，元始要对付他，也不需要如此多手段。
<br/>　　“不关本座之事，是你自己的悟性。”元始淡淡道，目光平静，看不出任何问题。
<br/>　　见此，通天也是微微一笑，不再多语。
<br/>　　“有些人将手伸得太长了，竟然将主意打到你我二教身上，借此机会终于将他们引出。”元始的眼神突然变得冷厉起来，连周边的温度都好似下降了许多。
<br/>　　“是啊，竟敢度化我的弟子！”通天教主的眼神此刻也变得冰冷无比。
<br/>　　“正好，封神榜上还差几个名额，便用他门下弟子吧。”元始淡淡道。
<br/>　　阵外，准提与接引见元始两人在阵中大战，战斗动静颇大，甚至他们都能感受到其中的凶险。看看眼前这些两教弟子，他们顿时起了心思。东方大地人杰地灵，不似他西方贫瘠，此刻正是一个壮大西方教的好机会，两位教主大战，必定是两败俱伤，到时候就便宜他们了。
<br/>　　心思一动，接引道人道了一声：“善哉！杀劫已至，可怜这万仙遭此劫难，我西方教慈悲为怀，可助此万仙脱离劫难，早登极乐，尔等俱皆与我西方有缘，可悟我西方大道！”
<br/>　　说着，他将乾坤袋打开，将截教三千红尘客，有缘之人，渡往西方极乐世界，毗卢仙、金箍仙、长耳定光仙、龟灵圣母等数千群仙俱皆被收入袋中，有些能够料算先机之仙，如凌虚子等，见势不妙，急忙躲开。其他如无当圣母，多宝真人却是被太上庇护了。
<br/>　　就在此时，四柄凶剑一颤，自四方虚空封锁而来，将这片虚空禁锢，诛仙剑阵再次布出，不过，此刻却是针对准提与接引的。
<br/>　　“两位道兄，尔等此乃何意？”通天教主手中青萍剑横胸，剑芒吞吐，目光冷冷地盯着接引与准提。
<br/>　　一旁，元始目光冷漠，淡漠地打量着两人。
<br/>　　准提与接引见此，心中深感不妙，这哪里是两败俱伤啊，两人都好了不能再好了。无奈，接引只好说道：“善哉，此乃红尘之地，苦海之中，他们身犯杀劫却还不自知，贫道这是为了助他们超脱苦海，早登极乐！”
<br/>　　“哦？是么，两位教主倒是好心肠，尔等连自己都还没有超脱，又何来超脱他人？”通天冷笑一声，目光却是愈发的冰冷了。
<br/>　　“我看两位教主与我东方有缘，不如加入我阐教，本座可许尔等副教主之位。”一旁元始那淡漠的声音传来，让两人心神一惊。
<br/>　　“道友说笑了。”准提脸上愁苦更甚了。
<br/>　　“我可不会说笑，你看，道友身形如枯槁，瘦如干柴，明显是营养不良的体现，显然，在西方两位都是经常吃不饱，如今更是来我东方化缘。不如两位直接入我东方之教，岂不更好，至少不会为伙食问题而发愁。”元始淡淡道。
<br/>　　“呃？”
<br/>　　两人闻言，都说不出话来，他们的确是没有元始等人壮实，但那是他们本来的面目，平时谁会无聊关注他们的外表，只会敬畏他们的实力，此刻却被元始提了出来，他们的脸色顿时变得难看起来。
<br/>　　“善哉！道友，你欲如何？”接引深吸一口气，道。
<br/>　　“放出吾之弟子，各自留下一宝，作为赎命所用。”通天手中长剑一挑，冷冷道。
<br/>　　“阿弥陀佛，两位真要如此才肯罢休？”
<br/>　　接引道。他心中知晓，这是针对他们的一个阴谋，今日就算是放出那些弟子，元始两人也不会任由他们安然离开。再说，西方本来法宝便稀少，若在此地留下两件，那西方岂不是更穷了。
<br/>　　说到底，还是要做过一场，洪荒之中，一切以实力说话。
<br/>　　“混账！安敢如此？”
<br/>　　突然，接引大声骂道。与此同时，准提也是面色难看无比。
<br/>　　却说为何两人会有如此反应，原来是刚刚两人感受到自己西方出现巨大危机。根据两人三尸所报，太华山劫魔殿趁此机会进攻他西方灵山本部。
<br/>　　西方教本来就没有多少弟子，此刻却正在被劫魔殿与冥河老祖的阿修罗族联手进攻。他们两人之中，只有接引斩去三尸，而准提只斩去了两尸，便以功德成圣，此刻接引三尸与准提两尸都被万劫魔尊与冥河老祖缠住了，而他们的弟子们，却是劫魔大肆屠杀，此战过后，就算他们不死，西方教也会名存实亡，威严全无。
<br/>　　“啊！”
<br/>　　突然，准提惨叫一声，他的面色立刻变得苍白无比，原来是他的一尊善尸被万劫魔尊给灭了。虽然善尸被灭了，还可以重新斩出，但却会给本尊造成重大的消耗，而且三尸的陨落，更会危及本尊，让本尊实力大降。
<br/>　　此时，元始与通天也算出了两人的情况，顿时心中舒畅不已，你西方教也有今天。
<br/>　　“两位到底要如何，此事才算了结？”接引二人此刻心中焦急不已，欲立刻回到西方教，但此刻却被通天二人困住，无法脱身，心中糟糕透了。
<br/>　　“哈哈哈，莫急，吾等可以慢慢谈，我时间很多。”通天教主哈哈一笑，缓缓道。既然知道了西方教被劫魔殿进攻，两人自然乐见其成。
<br/>　　“噗——”
<br/>　　突然，接引的嘴角亦流出一缕鲜血，显然，他的一尸也被灭了。
<br/>　　“道友，看来还是要做过一场，方可谈条件！”接引抹去嘴角鲜血，渐渐恢复平静，双眼平淡地看着通天道。
<br/>　　“如此甚好！”
<br/>　　通天一愣，不过还是说道。接着，他手中四剑悬空，剑气爆发，杀气聚空，展现出无尽凶威。
<br/>　　“今日我定要剑染圣血，一扬诛仙之锋芒！”
