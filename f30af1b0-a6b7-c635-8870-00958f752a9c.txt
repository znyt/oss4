　　小魔主出言，并没有跟那剑苍穹争辩，而是直接看向楚风眠，冷笑这开口道。
<br/>　　“我们九魔城的威严，不容挑衅，如今当然是要先前来，杀了这小子再说！”
<br/>　　小魔主开口，便是一句话，确定是他所来的目的。
<br/>　　杀楚风眠。
<br/>　　“这小子之前的大话，说的实在是太大了，现在让九魔城的人，抓到了把柄了。”
<br/>　　“小魔主此人，极为阴险，如今趁着这个机会，他绝不可能会放过楚风眠。”
<br/>　　一些武者，纷纷议论开口道，从这小魔主的前来，他们便是明白，这到底是为何而来。
<br/>　　楚风眠如今展现出来的天资，实在是有些恐怖至极，可以说都是超越了九域年轻这一代武者的天资。
<br/>　　没有任何人，愿意让楚风眠成长起来。
<br/>　　这小魔主前来，如今便是表明态度，要来对付楚风眠。
<br/>　　如今单打独斗之下，谁跟楚风眠交手，都占不到任何好处。
<br/>　　“不错，小魔主，这小子的确是太过狂傲，根本不将我们七大宗门，放在眼中。”
<br/>　　站在一旁的圣王宗盘纯，也是冷哼一声开口道。
<br/>　　“对付这样狂妄的小子，根本不需要在乎什么规矩，我们一起出手，直接杀了他！”
<br/>　　“你叫楚风眠？九域之中，倒是很久没有出现过如此狂妄的小子了，敢跟我们七大宗门挑衅的人，已经是数百年没有出现过了。”
<br/>　　小魔主看着楚风眠，冷笑连连起来道。
<br/>　　“也别怪我们七大宗门，不给你机会，现在我个给你一个机会，自废修为，我们可以饶你一命。”
<br/>　　小魔主站在剑苍穹跟盘纯的身边，看着楚风眠冷笑这开口道。
<br/>　　刚刚楚风眠跟剑苍穹，盘纯交手的时候，他在一旁，也是看的一清二楚。
<br/>　　楚风眠的天资，令他的心中，都是起了杀心，所以他才会来现身，围攻楚风眠。
<br/>　　不过，就算是楚风眠的实力再强，楚风眠也只是一个人。
<br/>　　一个人，难道会是他们三人联手的对手？
<br/>　　更何况在后面，还有着天剑宗，圣王宗，九魔城三方势力的弟子。
<br/>　　这种围攻之下，楚风眠，必死无疑。
<br/>　　这三人，已经是隐隐约约，将楚风眠围困起来，随时准备要出手，围攻楚风眠。
<br/>　　“可笑，难道你们以为，你们如今联合起来，就是我的对手了？”
<br/>　　楚风眠扫视了周围一眼，听到那小魔主的话，不由的大笑起来道。
<br/>　　“既然我说过，你们七大宗门的人，都要死，今日不管是你们来多少人，都一样得死！战龙武苍穹，给我破！”
<br/>　　楚风眠一边长啸起来，一边他的身躯便是在空中，化为了巨大无比的远古战龙的模样，凌空舞动。
<br/>　　楚风眠要先下手为强。
<br/>　　七大宗门中，如今古月峡谷的弟子，全部都被楚风眠轰杀。
<br/>　　但是还有荒古门，药王谷，天枢殿这三方的武者，没有到来，楚风眠也不打算在拖延下去，至少要将这三人，轰杀再说。
<br/>　　盘纯还逊色几分，但是这剑苍穹，小魔主，都绝对是年轻一代的武者巅峰，不好对付。
<br/>　　楚风眠猛然出手，庞大的力量，便是从这凌空之中，疯狂的轰击出去。
<br/>　　“圣王神拳！”
<br/>　　“天地剑影！”
<br/>　　“魔狱大手印！”
<br/>　　盘纯，剑苍穹，小魔主三人直接出手，他们早已经是将灵力积蓄完成，在楚风眠出手的一瞬间，他们便是也迫不及待的，直接出手。
<br/>　　三股庞大的力量，全部便是冲着天空之上那远古战龙身影，轰击而去。
<br/>　　不管是盘纯，还是剑苍穹，小魔主，他们三人放在九域上，在年轻一代在都是无敌人物。
<br/>　　如今他们三人联合起来，这爆发出来的力量，也是恐怖到了极点的地步。
<br/>　　三人联合起来的力量，只怕是让那些超越生死的强者前来，都要避其锋芒。
<br/>　　“轰！轰！轰！”
<br/>　　无数轰鸣声，突然爆发而起，无数力量从空中，疯狂交锋。
<br/>　　只见楚风眠的力量，在这三人的联手之下，突然之间便是被直接压制住了。
<br/>　　他的战龙血脉就算是在强大，但是同时面对着这样三尊天才联手之下，都是有些吃力。
<br/>　　“剑！”
<br/>　　楚风眠冷喝一声，祖龙至尊剑猛然出窍，一道剑光，在他面前的无数力量，都被直接斩断。
<br/>　　楚风眠最为强大的，依然是他的剑道，他的龙族武技，不过是在血脉之中的本能罢了。
<br/>　　他的剑道，才是他的终生所向。
<br/>　　“天临！”
<br/>　　楚风眠手中剑锋，猛然闪耀起来，这一道耀眼光芒，几乎是刺的人，睁不开眼睛。
<br/>　　无数剑锋，同时向着下方三人，直接斩杀过去。
<br/>　　“轰隆！”
<br/>　　几乎就在这同时，一声巨响，突然响彻而起。
<br/>　　这一声巨响，仿佛是如同一座许久封闭的大门，如今突然打开一样。
<br/>　　“这是？”
<br/>　　“云羽天宫开启了！”
<br/>　　“走，快走！”
<br/>　　“云羽天宫，无穷无尽的宝藏，快去！”
<br/>　　一些武者，已经是反应过来，猛然向着天羽城中心飞了过去。
<br/>　　就在这天羽城中心，那一座阵法，如今却是突然之间，已经开启。
<br/>　　云羽天宫，开启了。
<br/>　　而距离云羽天宫最近的，便是楚风眠，跟盘纯，剑苍穹，小魔主三人。
<br/>　　“走！”
<br/>　　楚风眠察觉到这一刻，便是猛然之间收回剑锋，他的脚下一动，身形分化幻影连连，直接一道遁光，向着云羽天宫中，冲了进去。
<br/>　　跟着三人交手相比，这云羽天宫，显然是要重要的多得多。
<br/>　　曾经九域第一宗门的宝藏，比起杀一百个剑苍穹得到的好处，都要多得多。
<br/>　　他没有丝毫犹豫，直接遁光急速变化，一下子冲入到了这传送法阵之中去。
<br/>　　“不好！追！”
<br/>　　“这小子肯定是受了伤，不能让他逃走！”
<br/>　　剑苍穹，小魔主看到楚风眠直接向着传送阵法方向飞去，也是急忙调动灵力，化为遁光，追了上去。
<br/>　　如今围杀楚风眠，本来是在他们看来，这是最好的机会，不趁着这个机会杀了楚风眠，以后必成大换。
<br/>　　那其他圣王宗，天剑宗，九魔城的弟子，也是接连跟上，一道道身影，全部都是进入到了传送阵法之中去，消失不见。
