夏妍妍这嗓门一喊，当即有两个保安走了过来，“小姐，请问这里发生什么事情了？”
<br/>
<br/>        夏妍妍指着夏夕绾，“这个女人没有烫金邀请函还想混进宴会厅，还扯谎是什么校长邀请她过来的，我看她就是想闹事，你们快点将她给赶出去！”
<br/>
<br/>        夏妍妍不想在这里看到夏夕绾，这种聚会夏夕绾是没有资格来参加的，而且夏夕绾还穿的这么漂亮，这身明裸色的长裙就像是为主角而设定的，她不希望夏夕绾吸引走任何的目光。
<br/>
<br/>        那两个保安迅速来到夏夕绾的面前，“这位小姐，这里不能滋扰生事，请你立刻离开，要不然我们就动手了！”
<br/>
<br/>        看着夏夕绾吃瘪，夏妍妍心里畅快淋漓，她趾高气扬道，“绾绾，快走吧！”
<br/>
<br/>        话音刚落下，一道严厉的嗓音突然传了过来，“你们在这里闹什么？”
<br/>
<br/>        夏妍妍回头一看，校长陆茵茵来了！
<br/>
<br/>        陆茵茵身后还跟着钱教授。
<br/>
<br/>        夏妍妍迅速收敛了眼里的心机和戾气，娇柔的上前解释道，“校长，我给你介绍一下吧，这位是我的妹妹夏夕绾，事情是这样的，绾绾没有邀请函但想进去，还说什么是校长你邀请她来的，这种谎言我当然是不会信的，绾绾才高中毕业，怎么可能认识校长你呢，所以我好言好语的劝绾绾离开这里，正好保安也过来了。”
<br/>
<br/>        陆茵茵用一种很奇怪的眼神看了一眼夏妍妍，“你说什么，你说夏夕绾才高中毕业？”
<br/>
<br/>        夏妍妍挺直了脊背，明明嘴角已经勾出了得意的弧线但是表面叹息道，“是啊校长，绾绾从小就被送到了乡下，所以没有读什么书，学历不高，你千万不要看不起她。”
<br/>
<br/>        钱教授看到夏夕绾来了也很诧异，“夏夕绾，你怎么来了，今天是医学名人的聚会，你就不要来闹事了，小姑娘家家的，我看生的也不错，有点自尊，不要将自己闹得太难看。”
<br/>
<br/>        钱教授因为先入为主的原因对夏夕绾有偏见，不过夏夕绾一身长裙亭亭玉立，小姑娘模样挺好的。
<br/>
<br/>        夏妍妍觉得目的达到了，她得意的看着夏夕绾，“绾绾，你快点走吧，不然保安真的要赶你出去了。”
<br/>
<br/>        这时陆茵茵走上前宣布道，“夏夕绾是我邀请过来的，有谁敢赶她出去？”
<br/>
<br/>        什么？
<br/>
<br/>        夏妍妍整个一僵，她瞪大了双眼不可置信的看着陆茵茵，刚才她…她说了什么？
<br/>
<br/>        夏夕绾真的是…是她邀请过来的？
<br/>
<br/>        陆茵茵来到了夏夕绾的面前，“夕绾，你来的时候怎么不打个电话给我，我好来迎接你。”
<br/>
<br/>        夏夕绾澄亮的翦瞳望着陆茵茵，“校长，我也刚到，正好碰上了熟人，聊了两句。”
<br/>
<br/>        这个熟人当然是夏妍妍了。
<br/>
<br/>        陆茵茵冷淡不悦的目光从夏妍妍僵硬的脸上一掠而过，然后看向那两个保安还有侍者，“这是我邀请过来的贵客，你们千万不要狗眼看人低，都下去吧。”
<br/>
<br/>        “是。”大家都退下了。
<br/>
<br/>        夏妍妍觉得自己好像被扇了一下耳光，因为陆茵茵这句“狗眼看人低”好像是说给她听的。
<br/>
<br/>        陆茵茵亲昵的拉住了夏夕绾的小手，喜爱的笑道，“夕绾，我已经等你好久了，真的好怕你不来，这身明裸色的长裙是我一眼看中的，我当时就觉得适合你清丽纤尘的气质，夕绾，今晚你真的好美啊。”
