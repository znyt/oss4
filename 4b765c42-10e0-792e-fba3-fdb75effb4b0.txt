　　第647章
<br/>　　三水关已经开城门，戴白只能选择退兵。他不能攻击三水关的守军，一旦动手性子就变了。当然，如果打的话也未必打得过。
<br/>　　而百里胜的八百残兵已随着百里无常涌入三水关，之后，城门紧闭，三水关前一个人影也无，有夜风轻轻吹过，摇晃芦苇，此地像是什么事都没有发生过。
<br/>　　过去很长时间后，从芦苇丛中探出个脑袋，他左右看看，确认面前无人后，向后挥了挥手。
<br/>　　有几个人凑到他身边，诸人鬼鬼祟祟，聚在一起窃窃私语。
<br/>　　这伙人，正是以混江龙为首的黑风寨山贼。黑风寨被程大雷一把火烧掉之后，混江龙便成了无家可归的流浪犬。他倒是有找程大雷报仇的心思，奈何实在没这个本事，于是就带着手下四处逃窜，想着去京州占个山头，继续当自己的山大王。
<br/>　　“大当家，他们在做什么？”绿头蝇问道。
<br/>　　“废话，老子怎么知道，咱们可千万不能和他们碰上。”混江龙骂了一声，他是山贼，仗着地势险要与官兵周旋，但正面冲突，山贼的战斗力是绝对比不上官兵的。
<br/>　　一行人小心谨慎穿过芦苇荡，想着涉山而过，进入京州。京州是天子脚下，论繁华程度远远胜过凉州。若是能进了京州，先找个村镇抢了，好好打打牙祭。这几天东躲西藏，混江龙已好几日没碰过女人，正好找几个香喷喷的美娇娘暖暖被窝。
<br/>　　混江龙心中如此想着，脚下便快了些。三水关他不敢走，只能选择翻过大山，夜里赶路也是怕被官兵发觉。
<br/>　　混江龙也算是熟悉附近地形，知道前面有条小路，可以绕过三水关进入京州。
<br/>　　“大当家，前面大树下有个大汉，靠在大石头上睡觉。”
<br/>　　快走到山前时，有喽啰报道。
<br/>　　“大汉，什么来路？”
<br/>　　混江龙皱起眉头，握着自己的鱼叉靠近，还未走近便听到一阵一阵，打雷一样的鼾声。他擦了擦眼睛，探着脑袋往前看，只见这大汉硕大一颗蓝脑袋，胡须血红，身边靠着一匹骏马，整个人宛若头怪物。
<br/>　　混江龙一激灵：是他！
<br/>　　他与单雄信照过面，此刻哪里还认不出来。只不过他想不明白，单雄信怎么会出现在这里，这周围还有没有他的同伙。
<br/>　　左右看看，除了如雷般的鼾声外并没有其他声响。虽然想不明白眼前的情形，但混江龙却是已经动了杀心。
<br/>　　这贼在黑风寨乱杀乱砍一番，如果双方动手自己不是他的对手，可此刻他酣睡不醒，岂不正是杀死他的好机会。
<br/>　　他挥手示意手下不要发出动静，手握住鱼叉，小心翼翼的靠近。只要走得近了，朝着单雄信的胸口一叉，单雄信就会被捅个透心凉。
<br/>　　正这时候，单雄信的座骑忽然唿律律发出一声嘶鸣，四蹄跺地十分的焦躁。
<br/>　　单雄信从睡梦中惊醒，与混江龙大眼瞪小眼，混江龙心里一发狠，鱼叉朝着单雄信扎过来。
<br/>　　“狗贼，与我去死吧。”
<br/>　　单雄信从大石头上一滚，堪堪避过这一叉，狼狈从地上站起，刚才也是十分凶险。
<br/>　　他擦掉额头的汗珠，忽然放声大笑。
<br/>　　“狗贼，我在这里等你多时了，快快纳命来。”
<br/>　　话说，单雄信如何来了此处？混江龙从他手中逃脱，程大雷令他去守城门，他心里迈不过这道坎儿，就想着自己可以杀了混江龙，提他的脑袋去见程大雷。也算在蛤蟆城挣回自己的脸面。
<br/>　　但混江龙又岂是好寻的，寻了几日没发现半点踪影，还要担心被蛤蟆城的探子发现。
<br/>　　出来混了几日，没有半点收获，单雄信心中郁闷，就在这大树下喝了几口随身带着的水酒，迷迷糊糊睡了过去。
<br/>　　谁料想，醒来后混江龙便出现在面前。这才是踏破铁鞋无觅处，得来全不费功夫。
<br/>　　他抓起马背上的大槊，照着混江龙脑袋就劈了过来，这一招如果劈实了，非得把混江龙整个人劈成两半。
<br/>　　混江龙就地一滚，没想到单雄信如此大的杀威，吓得亡魂皆冒。这时候他的手下涌过来，将他从地上扶起。
<br/>　　“大当家，大当家……”
<br/>　　混江龙惊魂未定，忽然转念一想，自己现在手下有五百多弟兄，这单雄信再有本事，也是好汉难敌四手。今日情形，谁杀谁还说不定呢。
<br/>　　他桀桀冷笑一声：“天堂有路你不走，地狱无门你闯进来，弟兄们，与我杀！”
<br/>　　这时候，单雄信的酒意也已经醒过来。一看眼前情况，的确对自己不利。但他本就是粗枝大叶的性格，虽然敌众我寡但又何惧之有。手中大槊一甩，口中喝道：“来！”
<br/>　　一帮小喽啰一拥而上，口中叫骂着，挥舞着各种兵器攻向单雄信。单雄信当真如砍瓜切菜相仿，手中鲜血淋漓，不多时地上就多出了数具尸体。
<br/>　　那混江龙本想仗着人多势众在此地将单雄信杀了，谁料想他将如此了得。一看眼前情形不对，再打下去自己的性命也得报销。
<br/>　　于是三十六计走为上计，口中喊着『弟兄们与我杀』，脚步却在慢慢后退，刚刚与单雄信拉开距离，便转过身向后逃。
<br/>　　单雄信的目的本就不是杀几个小喽啰，一看混江龙想走，那能容他脱身。他一扯马僵绳，翻身上马就去追。
<br/>　　混江龙逃得气喘吁吁，两条腿再快也逃不过四条腿。正走投无路的时候，前面出现一条大河。混江龙一激灵，快走几步一头扎进水里。
<br/>　　单雄信急忙勒马停住，看着面前的河水顿时没了主意。他本身水性不行，此刻也不敢下河去追。
<br/>　　那混江龙本以为自己必死无疑，他在水中伏了半天，却没听见任何动静。大着胆子探出头来，就见单雄信骑在马上一副束手无策的样子。
<br/>　　混江龙瞬间便明白过来，口中哈哈笑道：“你敢下来么？”
<br/>　　单雄信也是福至心灵，脱口而出道：“你敢上来么？”
<br/>　　“你敢下来么！”
<br/>　　二人隔着水叫骂半天，混江龙踩着水慢慢去了对岸，单雄信眼睁睁看着这一幕，也是束手无策。
