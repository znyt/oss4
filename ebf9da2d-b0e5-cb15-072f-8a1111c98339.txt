>    “妖晶是做什么的？”夜阑问。<br/>    “妖师修炼，靠的就是妖晶。这东西对普通人无用，但对妖师来说却是不可多得的东西。”龙文牧道，“而且在妖师与妖师之间，妖晶也等同于钱财。”<br/>    夜阑略微点头，对普通人而言，金银铜是货币，但是在妖师之间，妖晶却取而代之。<br/>    夜阑静静看着，差不多半个时辰之后，整头云隐兽都已经四分五裂，而在其中寻出来的妖晶，足足近百块之多。<br/>    “啧啧，还真不少。”龙文牧似笑非笑。<br/>    一般小妖体内也就五六十块，而这头云隐兽毕竟是龙种，居然孕育了近百块，就小妖而言，这数量已经不少了。<br/>    又过了一阵，只见一个柳家护卫废了九牛二虎之力，总算是敲开了云隐兽的头，兴奋的找到一枚鸡蛋大小的珠子，惊呼连连。<br/>    “那个是内丹。”龙文牧对夜阑说，“内丹用处也不小，对妖师而言，这东西比妖晶还要珍贵。”<br/>    找到内丹的护卫兴奋的跑到柳长风跟前，献宝一样的献上，柳长风脸上也洋溢着笑容。<br/>    龙种体内的内丹，这可是比一般小妖的内丹还要来得珍贵。<br/>    “找到了……我找到了！”就在柳长风打量着内丹的时候，另一个兴奋的声音传来。<br/>    所有人不约而同的望去，只见某个护卫找出了一个拳头大小的赤红晶石。<br/>    晶石形状并不规则，通体散发着诡异的光。<br/>    晶石被护卫拿在手上，上面竟隐隐传出龙吟声。<br/>    在找到妖晶和内丹的时候龙文牧还不动于衷，但当这枚晶体出现的那一刻，他的眼眸也逐渐变得狭长。<br/>    “源石，这是妖兽体内最重要的东西。”龙文牧缓缓开口。<br/>    “源石是什么？”夜阑疑惑问。<br/>    “丫头，你还记不记得什么是妖师？”龙文牧忽然问了个莫名其妙的问题。<br/>    “妖师……不就是对付妖兽的人吗？”夜阑小声说。<br/>    “那你知不知道妖师是怎么来的？”龙文牧又问。<br/>    夜阑微微踌躇，然后摇了摇头。<br/>    “一个普通人，不管再怎么努力也不会具备妖力。凡人终究是凡人。”龙文牧脸上流露着认真，“上古时候，妖兽肆虐，凡人之躯无论如何也挡不住妖兽。后来便有大能之人想出办法，将妖兽的源石融入人的身体，让人拥有妖兽之力，那就是妖师的由来。”<br/>    夜阑猛然惊骇。<br/>    “少爷，你是说融入人的身体？”<br/>    “一个人一生能接纳四块源石的力量。接受一块，就入了小妖师的门槛，从此便是妖师。等到长年累月的修炼到上品小妖师巅峰的时候，便能接受第二块，如果成功吸收，这人便是大妖师……”<br/>    “那……接受三块就是地妖师，接受四块就是天妖师？”夜阑隐隐明白龙文牧的意思了。<br/>    龙文牧点头：“吸收源石并不是易事，也伴随风险，吸收的源石越强，也就越危险。”<br/>    “可是……不同妖兽体内的源石，岂不是不相同？”夜阑问。<br/>    “不同妖兽的源石都不相同，所以一个妖师的强弱，除了取决于自身修为，也取决于吸收的源石的力量。”龙文牧认真说，“两个相同修为的小妖师，如果其中一个吸收的是大妖的源石，那他就一定比吸收小妖源石的人要强。”<br/>    夜阑听得惊骇连连，龙文牧说着有关妖师的种种，其中的玄奥，在她脑海里挥之不去。<br/>    一个妖师的强与弱除了跟境界有关，也跟源石有关。那如果一个人吸收的是大妖的源石，又或者吸收的是地妖的源石，那该有多强？<br/>    天妖也有源石吗？传闻中，与妖兽抗争的历史中，也有天妖被斩杀过的记录。那是不是就是说，也曾经有人吸收过天妖的源石？那又该是什么样的实力？<br/>    空地中央，柳长风已经将源石接了过来，喜悦的心情溢于言表。<br/>    “恭喜大人得到云隐兽的源石。”方奇在一旁道贺，“大人如今是上品小妖师，恐怕再过不久就有望冲击大妖师之境。到时大人吸收了这块源石，成就定然超过寻常的大妖师。这次能够得到龙种的源石，连上天都眷顾大人。”<br/>    柳长风把玩着源石，简直爱不释手。<br/>    大妖的源石他不指望，大妖这种存在除非出动数名大妖师或者出动地妖师才有望猎杀，而且大妖可遇不可求，大妖的源石更是妄想。<br/>    一般的妖师晋升，能以小妖的源石晋升就已经不易了。<br/>    就比如方奇，现在虽是大妖师之境，在他修炼过程中曾吸收过两次源石，而两次源石皆是来自小妖。<br/>    对妖师而言，这也是最普遍的情况。<br/>    现在他手上的这枚是龙种的源石，在小妖里绝对是佼佼者。<br/>    他能够想象的到，如果自己能以这枚源石来冲击大妖师之境，那自己的实力肯定比方奇都还要高出一筹。<br/>    既然成了妖师，谁会不追求实力，就算没有大妖的源石，有龙种的源石也是不易的，他岂能不乐。<br/>    说到底，来雾山本来只是碰碰运气，能狩猎到云隐兽实在是意外之喜。<br/>    含笑着将源石收好，至于妖兽的内丹和妖晶则是让下人携带。比起源石来说，其它东西已经不重要了。<br/>    “大人，整头妖兽已经找遍了。”一个护卫上前抱拳，“找到妖晶一百零四块，敢问大人，剩余的残骸要怎么处理？”<br/>    “暂且放下吧，再往高处走走看，等回程的时候再运回柳家。”柳长风想了想说。<br/>    方奇闻言：“大人还要往高处去？”<br/>    “此处连半山腰都不到，再走走又有何妨？”柳长风长袖一拂，“这座山云雾缭绕，兴许妖兽不止一头，如果能再碰到一头，那岂不美哉？”<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>
