李玉兰来到了病床前，夏老爷子正躺在病床上。
<br/>
<br/>        李玉兰的目光里露出了怨毒，这个老不死的，竟然这么命大，毒都毒不死他。
<br/>
<br/>        她快速的拿出了针管，将针管扎进了老爷子的手臂里。
<br/>
<br/>        但是很快她就发现了不对劲，因为老爷子的手臂冰冷又僵硬，好像是一个…死人！
<br/>
<br/>        怎么回事？
<br/>
<br/>        李玉兰将手指伸到了老爷子的鼻下，那里已经没有了呼吸。
<br/>
<br/>        老爷子早已经死了！
<br/>
<br/>        糟糕，她中计了！
<br/>
<br/>        这时病房门突然被打开，刺目的白盏灯投射了过来，一批穿着制服的警察冲上前，直接将李玉兰给制住了，那个针管也放在了密封的袋子里。
<br/>
<br/>        “李玉兰，现在怀疑你跟一宗故意杀人案有关，现在证据确凿，我们要将你带回去审查！”
<br/>
<br/>        李玉兰瞪大了双眼，满脸的惊恐和不安，她迅速挣扎，“你们放开我，我没有，我不是，你们误会了。”
<br/>
<br/>        这时有两个人走了进来，是夏夕绾和夏振国。
<br/>
<br/>        当看到夏夕绾的那一瞬间，李玉兰都明白了，老爷子早死了，这不过是夏夕绾跟夏振国联合的一场戏，她栽进去了。
<br/>
<br/>        李玉兰浑身冰凉，她整个人如坠深渊，“夏夕绾，是你，又是你，是你在害我！”
<br/>
<br/>        夏夕绾一双澄亮的翦瞳冷冷的看着李玉兰，嗓音清丽的一字一句道，“究竟是谁在害谁，我相信法律会给予公道。”
<br/>
<br/>        “你！”
<br/>
<br/>        这时夏振国冲上前，抬手就狠狠的扇了李玉兰一耳光，他虽然配合夏夕绾演戏，但是他怎么也不相信这个跟他同床共枕了十多年的女人竟然早就将罪恶的双手伸向了他的父亲，而他被蒙骗了这么多年！
<br/>
<br/>        “李玉兰，你这个蛇蝎毒妇，我本来还不相信是你害死了我爸，我对你还不好吗，你还有什么不知足的，你为什么要害死我爸？”夏振国眼眶通红的瞪着李玉兰。
<br/>
<br/>        李玉兰的脸上迅速浮现出了一个鲜红的巴掌印，她知道自己完了，现在她被当场抓获，根本无从抵赖，一切都瞒不住了。
<br/>
<br/>        “夏振国，你怎么有脸说你对我好的，当年我做你的外室，妍妍都给你生了，但是这个老不死的就是不让我进门，还不肯承认妍妍的身份，你窝囊愚孝的连屁都不敢放一个，这个老不死的就是我的绊脚石，我当然要将他除掉了！”
<br/>
<br/>        “夏振国，你不要怨别人，这些都是因为你蠢，你不喜欢夏夕绾，我也不喜欢夏夕绾，所以我正好设计推了老爷子嫁祸在她的头上，但是将她送到乡下可是你的选择，也是你让我成为夏夫人的，给了我再次毒害这个老不死的机会，这一切都是你咎由自取！”
<br/>
<br/>        夏振国浑身颤抖，恨不得再给李玉兰一耳光，但是李玉兰说的又是实话，字字敲打在他的心上，让他觉得难堪。
<br/>
<br/>        从未有过的难堪。
<br/>
<br/>        “把她带走！”
<br/>
<br/>        这时李玉兰被带走了。
