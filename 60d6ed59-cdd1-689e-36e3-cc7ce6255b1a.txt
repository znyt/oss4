　　穿过了射击队，很快两人就同时皱起眉，在他们眼前有约莫七八百人的队伍，一个个都拥挤着睡的很熟，甚至很多人因为位置不够，都叠着睡。
<br/>　　同时在这片地带，穿过射击队区域时变淡的腐臭味，在这里又浓郁了起来，并且还刺鼻无比，让人都难以呼吸。
<br/>　　“这是三战队吧？”洛冥已经猜出了眼前这群人的身份。
<br/>　　段江伟看了一圈后点头：“这是几万人中排在最前方的位置，应当为三战队专属，数量也很接近，但估计死了一百人上下……”
<br/>　　洛冥沉默了几秒，随后问道：“三战队在这次大战中负责的是什么？”
<br/>　　段江伟想了想回答：“我并不清楚这场战役的具体战略，但通过鲁初雪每日传达的战报来看，三战队应该是负责了七阶以上的全部高阶丧尸。”
<br/>　　“七阶以上？”洛冥震惊不已。
<br/>　　段江伟点头，随后指着南署基地的外圈高墙：“我们绕了那一圈的高墙上，那些体型巨大的丧尸都是七阶以上的高阶丧尸，包括了城墙外下方，因为尸体挤不下而掉下去，堆成小山的那些，也是。”
<br/>　　回想着之前绕一圈时看到的这些丧尸数量，洛冥的神情充满了震撼：“那是多少高阶丧尸，三战队人数总共还没超过一千吧？”
<br/>　　“对，经过我段氏基地参谋部的粗略分析统计，在尸潮失控冲进南署基地之后，七阶以上的高阶丧尸数量……”说着段江伟声音一顿，抬眼看向洛冥眼神尤其复杂：“有一百五十万左右。”
<br/>　　“一百五十……万？！”洛冥这下子彻底震惊到无以复加，声音都忍不住提高了一个八度：“一百五十万七阶以上的丧尸，全部都是这些才一千不到的年轻人解决的？”
<br/>　　就在洛冥震惊喊出之时，不远处一个呼噜声震耳欲聋的人忽然翻了个身，‘啪’的一巴掌拍在了旁边的同伴身上，还迷迷糊糊的开口道：“别吵老子睡觉，别……以为你是队长……我就不敢揍你。”
<br/>　　而旁边的人也动了动，抬起腿就一脚将此人踢开，同样迷迷糊糊的回应道：“顶撞上级，去……军法处领罚。”
<br/>　　突如其来的对话雷的洛冥和段江伟双双外焦里嫩，两人呆滞的望着那两名三战队成员的互动，对其睡觉还能对话的神级技能深表佩服。
<br/>　　在两声迷糊的对话后，很快就再次回归安静，四周也只剩下了一连串的呼噜声。
<br/>　　“那是徐锋吧！”洛冥仔细看了好几眼后确认，又指着旁边的那个大汉问道：“这个呼噜声最响，声音跟打雷似得人是谁，也是杀羽战队成员？他喊徐锋队长呢！”
<br/>　　看到洛冥这个老顽童双眼放光的好奇样子，段江伟也忍不住想笑，回答道：“这是杀羽战队的副队长李必疯少将，很出名的一位将领，战斗风格比徐锋更贴近杀羽战队，是这个战队不可或缺的核心人物之一。”
<br/>　　“李必疯，我记得。”洛冥面上露出一抹深意：“当初我和姓牧的那老头，塞给楚涵的第一批人马，那时候的李必疯还是个不好驯服的尉级军官，现在竟然都已经是少将了啊，想必战绩很漂亮吧？”
<br/>　　段江伟笑容古怪了起来，摇头道：“如果不是李必疯总是触犯狼牙军法条例，以他的战绩早就能荣升中将了。”
<br/>　　“恩？”洛冥有些惊讶。
<br/>　　段江伟咧开嘴，难得露出了一个灿烂笑容，好似是在打趣：“杀羽战队很有意思，明明是公认狼牙最强战力的战队，却同样也是狼牙内部触犯军法最频繁的队伍，据说杀羽战队之内每个人都被刘玉定惩罚过，次数最少的也是两次，李必疯次数最多，百来次吧！”
<br/>　　洛冥又被震惊了一把，瞪大了眼睛满是惊疑：“竟然还有这种事！”
<br/>　　段江伟点头，恢复了一板一眼的正经神情：“我猜如果不是军法处的牵制，刘玉定也一向强势，这杀羽战队估计早就把天都捅破了，这是非常激进的一个战队。”
<br/>　　洛冥点点头，往旁边没走多久就看到了风格显眼的神隐战队。
<br/>　　“三战队中最不好认的就是神隐战队成员，神之隐匿并没有打出多响的名声，但对狼牙战团一直调查的人都清楚，三战队里神隐战队相当可怕，他们常常隐藏在人群中，除了他们自己其他人压根认不出来。”跟上的段江伟笑道：“但神隐又是最好认的，尤其是在三战队的人在一起的时候。”
<br/>　　洛冥深以为然的点头：“因为杀羽和黑芒一个锋芒在外一个暗芒汹涌，反而将神隐给突显了出来。”
<br/>　　“洛老你知道哪个家伙是路冰泽么？”段江伟指着地上的一群人发问。
<br/>　　洛冥笑着没再逗留继续往前走：“你都认不出，更何况我？”
<br/>　　段江伟跟上洛冥的脚步，颇有些吐槽的说道：“楚涵藏最深的一个人就是路冰泽，到现在还没几个人真正见过神隐战队的队长究竟长什么样，听说那家伙现在很可怕，带着神隐把狼牙各部门挑战了一个遍，闯下了好几连胜。”
<br/>　　两人随意走动，不断在三战队中找人挑出来讨论一翻，似乎对这种事情感到很有意思，这些人的战绩、性格及在狼牙中的位置，说起来每个人都充满了精彩故事，能讨论个三天三夜。
<br/>　　大战已经结束，这一恐怖又让人沸腾的胜利无声的荡漾在这个南署基地，很快爆炸性的消息就将传遍大江南北。
<br/>　　越是如此，洛冥和段江伟反而越不着急，所有人都对狼牙战团很好奇，能看到毫无防备的狼牙战团全体成员，还有机会对他们指指点点说三道四，这机会可遇不可求啊！
<br/>　　就这么一路走动中，两人有说有笑的认出了很多人，段江伟更是滔滔不绝的将自己暗地里调查出的情报说出，就像是对明星的八卦一样聊着天。
<br/>　　毕竟狼牙三战队的成员，还真有不少是末世纪元的明星人物！
<br/>　　忽的洛冥停下脚步，有些意外的看着旁边躺着的一名三战队成员。
<br/>　　段江伟走近后也顿了顿，最终迟疑道：“这位，之前是您老培养起来的吧？”
