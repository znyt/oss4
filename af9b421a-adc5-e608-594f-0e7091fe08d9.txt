<br/>				    听到这青年男子爽快应下，周围的众人神色有一瞬间的诧异。这样明摆着是坑的事情，居然也有傻子往里面跳？这男子该不会是脑子不正常吧？<br/>　　<br/>　　    “哎，我也许久没吃肉了，馋得紧，走吧！既然要去，那就别耽误时间了。”凤九神色轻松的说着，脸上不惊不惧。<br/>　　<br/>　　    两名魔修也愣了一下，狐疑的看了眼前这青年一眼，最后才道：“那走吧！跟着我们走。”说话间，他们御剑而行，往那凶兽林的方向而去。<br/>　　<br/>　　    见此，凤九便也跟着御着剑跟了上去。待三人一走，有的修士便议论了起来。<br/>　　<br/>　　    “嗤！也不知哪来的傻小子，竟也敢跟着那两人去兽林？这是嫌命太长自寻死路。”<br/>　　<br/>　　    “估计是饿惨了，你看那小子身无三两肉，应该是许久没闻到肉香味了。”<br/>　　<br/>　　    “别说那小子许久没闻到肉香味，我也有两三个月没吃到肉食子，这段时间都是林中摘着一些野果子充饥。”<br/>　　<br/>　　    “可惜，我们散修能找到食物的地方是越来越少了，那八座峰头的人也太狠了，占着山头不说，还跟我们抢东西，抢地盘，这是想要逼死我们！”<br/>　　<br/>　　    “想吃肉？也不是没有，去加入那八座峰头的任何一头，自然也是跟着他们吃香喝辣的。”<br/>　　<br/>　　    “吃香喝辣？哈哈哈，你别笑死人了，就算是那八座峰头的人肉食也是分着吃的，你不知道吧！上回我可听说了，这八座峰头的人的肉食是按着实力来区别，就算是认了他们为主，也别想可以得到公平对待。”<br/>　　<br/>　　    听到这话，议论着的的众人都沉默了下来。若换成以前还没被弄到这里来时，他们何曾为食物的问题烦恼过？何至于像现在为了一顿肉食而拼了命去那凶兽林中捕杀兽类？<br/>　　<br/>　　    以他们的实力，若是在外面，随便走到哪个地方都是被人所仰视敬畏的人物，然，今时不同往日，一切已经不一样了。<br/>　　<br/>　　    另一边，凤九跟着两名魔修往那兽林而去，御剑飞行着，不紧不慢的跟在两人身后。看着前面的两人，她询问着：“那八座山峰的人没占了那兽林吗？怎么会放着这样一个地方不要？”<br/>　　<br/>　　    “呵！占？想占也得看他们有没那个本事。”<br/>　　<br/>　　    一名魔修回头瞥了凤九一眼：“你以为那凶兽林是什么样一个地方？在这里面，这凶兽林也被称为死亡之地，可没几人能进得去，就算是有的人能进得去，也没人能侥幸活着出来的。”<br/>　　<br/>　　    闻言，凤九勾唇一笑：“哦？既然这样，那你们还敢带我去那地方？”<br/>　　<br/>　　    “那凶兽林是由五大仙帝布下的阵法和结界所隔绝的，一般人进不去，就算进去了也会迷失在里面出不来，不过，恰巧，我们知道有一个地方可以过去。”<br/>　　<br/>　　    也许是觉得最后这青年也是落得个死的下场，因此，说起话来倒也没顾忌什么，就连他们所知道的秘密通道也都跟凤九说了。<br/>　　<br/>　　    而听到他们的话，凤九则饶有兴致的笑了。<br/>
