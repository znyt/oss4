<br/>　　“铮！”

<br/>　　同时一股恐怖的煞气，顺着慕清霄的手臂试图入侵他的身躯，守护心神，《御女心经》运转，无上剑意释放而出，直接逼退煞气！
<br/>　　“真是恐怖的煞气，不知道有多少生灵死在这柄剑下！”慕清霄微不可查的摇头。
<br/>　　逼退诛仙剑上的煞气，慕清霄在四周查找起来，确实没有发现天书的痕迹，于是目光就放在了剑身上。
<br/>　　望着手中锈迹斑斑的诛仙剑，天书绝对不是诛仙剑本身，否则得手后，系统早就提示了。
<br/>　　想到这里，慕清霄释放着剑意，压制煞气的同时，心神开始入侵诛仙剑，仔仔细细的查探起来。
<br/>　　心神越是入侵其内部，浓郁的煞气就扑面而来，如同厉鬼一般，不断的撕扯无上剑意所形成的保护屏障，试图着想要将心神泯灭掉。
<br/>　　慕清霄剑眉紧皱在一起，如此浓郁的煞气，恐怕剑灵早已魔化，就在这时，心神周围出现一缕金色光芒，越来越多，向心神汇聚而来。
<br/>　　“咻咻——”
<br/>　　紧接着，金色光芒向外射去，慕清霄一愣，心神紧追，当回过神来，紫云密布的苍穹上已经出现一排巨大的金色字体。
<br/>　　“天地不仁，以万物为刍狗！”
<br/>　　望着苍穹上的一排金色字体，慕清霄嘴角牵起一丝笑容，确实是第五卷天书无疑，心神直上，轰的一声，脑海一片空白，天书已融入其中。
<br/>　　“叮，成功收集第五卷天书，任务完成1/5！”
<br/>　　听到系统的提示声后，慕清霄再也不看诛仙剑一眼，直接归回原位，苍穹上空的雷弧也恢复正常。
<br/>　　他需要的是天书，若非信息中提到，诛仙剑就是第五卷天书，他根本不会来幻月洞府，自己阴阳玉中的魂剑，可不是诛仙剑能相比的。
<br/>　　“咦，速度真是够快的。”慕清霄微微一笑，目光看向幻月洞府的入口处。
<br/>　　万剑一已经来到其中，显然已经破开了永恒万花筒的幻术，虽然慕清霄并没有用全力，但如此迅速就从月读空间脱离，确实值得称赞。
<br/>　　此时，万剑一脸色前所未有的凝重，目光瞥了一眼慕清霄的眼睛，随后移开，刚才那双邪恶的眼眸，他终生都难以忘怀，想到刚才的幻境，心中后怕不已。
<br/>　　“阁下难道是魔教之人，混入青云门的目的便是诛仙剑？”万剑一语气凝重，手中握着一柄木剑，已经准备开始进攻。
<br/>　　慕清霄觉得有些好笑，自己要是魔教的人，刚刚岂会轻易放过他，直接宰了不是更好，至于诛仙剑，他还真是看不上。
<br/>　　“第一，我不是混入青云，而是各脉首座争着让我拜入青云门，第二，我来青云门本就抱着目的，并非诛仙剑，而是第五卷天书，第三，你觉得我能瞧上这柄破剑？”
<br/>　　慕清霄眼中闪过一丝轻蔑，道：“万剑一，难道你也变得与道玄一样迂腐？”
<br/>　　刚刚听到第五卷天书的时候，万剑一脸色就微变，听到慕清霄叫破他的名字后，瞳孔一缩，面前的青年居然知道他的身份......
<br/>　　在整个青云门，知道他没有死去的，也只有掌门道玄，他实在不明白，面前的青年为何知道如此多的秘辛！
<br/>　　“你到底是什么人？”
<br/>　　“我是谁很重要吗？最起码我对青云门并没有恶意，想要的东西也都拿到了，不如给我行个方便，让我安安稳稳住上个半月，到时候好给道玄一个交代。”
<br/>　　万剑一剑眉紧皱，紧绷的心弦稍稍松了一些，他知道面前的青年修为恐怖，哪怕他用尽全力，想必也不是对手，强者不屑说谎，但他不得不防。
<br/>　　“既然你没有恶意，为何晚上鬼鬼祟祟的来到幻月洞府，而且还知道开启洞府的方式，你说自己不是魔教中人，又是如何得知，让我如何相信你？”
