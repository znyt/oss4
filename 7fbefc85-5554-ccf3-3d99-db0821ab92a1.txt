        连华生被夏玄灵逼得别无他选，只能给裴金澈开安眠药，保证他一觉睡到天亮，让那个所谓的“第二人格”没法出来。
<br/>
<br/>        其实，他明里暗里、间接或者直接地询问过自己父亲，为什么要研究这种改变基因的药剂。
<br/>
<br/>        只可惜，他父亲只回他一句“别多管闲事”。
<br/>
<br/>        彻底被父亲孤立这种滋味，其实一点都不好受。
<br/>
<br/>        傅夜沉见连华生叹气连连，一副愁眉苦脸的样子，又关问道：“你跟你父亲吵架了？”
<br/>
<br/>        “那倒没有，只是……”连华生欲言又止，顿了顿后，才感慨道，“我感觉，我老爸很提防着我。”
<br/>
<br/>        “你从小就不听他的话，什么事情都有自己的主见，更何况，你站我这边，连叔不提防你才怪。”傅夜沉云淡风轻地说道。
<br/>
<br/>        连华生不禁就想不明白了，纳闷地反问：“你说吧！我老爸当年跟你老爸可是铁哥们，怎么突然就把关系闹僵了呢？而且还跟夏局站到了一队！”
<br/>
<br/>        “这种事情，你得去问当事人，比如说你父亲，或者是夏局。”傅夜沉漫不经心地回答。
<br/>
<br/>        连华生见傅夜沉没什么心情跟他闲聊父辈们的话题，只好识趣地接着说：“行了，我就不打扰你工作了。对了，童筱颖的行踪，我已经派人轮流二十小时密切关注着，一有什么不对劲，会立即通知我们。”
<br/>
<br/>        “你把监听器关了吧！”傅夜沉淡淡地应了声。
<br/>
<br/>        “为什么？”连华生顿时感到不可思议，虽然说，给童筱颖植入纳米监听定位器是他连华生个人的主意，但他也是为了他傅夜沉好。
<br/>
<br/>        “因为，她为了这件事情，有些不开心。”傅夜沉凝眉道，随后，他放下了手中的签字笔，怀抱双臂，纳闷起来，“我总觉得，有人事先给筱颖进行了催眠，然后在潜意识里给了她某种指令。”
<br/>
<br/>        “你怎么突然想到了这个？”连华生诧异。
<br/>
<br/>        傅夜沉随之从书桌前起身，转身爬上书架的楼梯，从最顶端那层的书架上抽出了一本厚厚的书籍，侧身向连华生扔去。
<br/>
<br/>        连华生敏捷地接住这本书，看到书的封面后，顿时恍然大悟：“潜意识催眠术！”
<br/>
<br/>        “你是专业，你比我懂。我只是无意间看到这本书，才联想到一块的。”傅夜沉一边说着，一边从书架上下来。
<br/>
<br/>        这本书里有描述到，催眠师可以通过特定地一些物品，对催眠者进行潜意识的催眠。
<br/>
<br/>        比如说，给催眠者下达某种命令，让催眠者平常时与常人无异，当这个催眠者看到某个东西的时候，潜意识里就会开始执行催眠师的命令。
<br/>
<br/>        连华生先是翻到目录，找到与童筱颖类似的情况后，认真地看了起来，顿时惊讶不已：“莫非，童筱颖在看到你后，同时看到可以用来杀人的器具，就会在潜意识里去执行催眠师的命令杀了你！”
<br/>
<br/>        “我想，有可能，但是……”傅夜沉不禁微微蹙起眉头，“筱颖似乎对我下不了手。”
<br/>
<br/>        “她如果对你下不了手，就证明她的主人格还在。”连华生合上手里的书，睐了傅夜沉，接着说道，“我觉得，她可能是真的很爱你。”
<br/>
<br/>        听连华生说这话时，傅夜沉身心一怔。
<br/>
<br/>        连华生又接着滔滔不绝起来：“认识童筱颖至今，我发现她对待你的方式，与对待傅昼景的方式不大一样。她曾经跟傅昼景在一起的时候，于她而言，她对傅昼景除了心存感激，还有就是盲目崇拜与体谅，毕竟他俩是同行，还有就是傅昼景的身世可怜，她跟傅昼景在一起可以说是同命相怜。后来她跟你在一起的时候，我真觉得她似乎对你并不用心。直到她选择自杀，来保你平安后，我才设想，她真正爱的那个人应该是你傅夜沉，而不是傅昼景。现在，你说她对你无法痛下杀手，估计也是这个原因。她的内心世界，应该有两个潜意识在打架。一个潜意识护着你，一个潜意识在执行命令。可喜可贺的是，护着你的那个潜意识赢了。”
<br/>
<br/>        “你向来不喜欢筱颖，但作为旁观者，你倒是看得很透。”傅夜沉欣慰地笑了笑。
<br/>
<br/>        他和他是最好的“兄弟”，话匣子都是敞开了来讲，所以，为了筱颖的事情，其实他和连华生私底下没少吵过架。
<br/>
<br/>        连华生微微耸了耸肩，双手一摊，一脸无奈地回答道：“谁叫你死活不肯接受落以柔？”
<br/>
<br/>        “你有空，劝劝她，不要再把心思浪费在我身上。”傅夜沉说着，不禁感慨起来，“筱颖已经让我够伤脑筋的了，我没心思再去关注其他女人。”
