　　楚风眠一路思考这，一边注意着周围的一举一动。
<br/>　　他已经是发现了，周围深邃到黑暗，已经是消失了，取而代之的，则是一片片的光幕。
<br/>　　楚风眠试探性的伸出手去碰了一下，发现这光幕的另一边，却是在通往这另外这另一个地方。
<br/>　　“莫非到了这里，就算是初步完成了独桥的考验？这光幕的另一边，应该是通往宝藏的地方，可以得到奖励。”
<br/>　　楚风眠观察了一下，突然凝聚起灵识来，渗透了过去。
<br/>　　这光幕，不止是足以阻挡视线，还足以阻拦灵识，像是被人刻画出了阵法，故意遮挡住了一样。
<br/>　　不过这上面的阵法禁制，已经是经历了无数岁月，上面的威力已经是十不足一，楚风眠催动灵识，一下子便是穿越了这光幕。
<br/>　　只见就在这光幕的背后，是一座宫殿，宫殿之中，琳琅满目摆放着不少的圣药，圣丹。
<br/>　　每一片光幕背后，都摆放着不同的宝物。
<br/>　　只是在这光幕的背后，还有着一道巨大无比的阵法，笼罩在其中，将这光幕跟独桥，隔绝开来。
<br/>　　只要是穿越这光幕，便是就可以踏入到背后的宫殿之中，收取其中的宝物。
<br/>　　可若是这样做的话，那进入光幕之中的武者，也就没有第二次机会，在进入独桥。
<br/>　　这是一个选择，可以进入其中，去收取宝藏，但是一旦是将宝藏收取了，也就代表着再也无法再进一步，只能是离开这大型宝藏了。
<br/>　　每个人只有一次得到其中宝藏的机会。
<br/>　　楚风眠看到这，心中也没有任何的惊讶。
<br/>　　一般这样的大型宝藏，其中的宝藏，并非是只有一个人可以得到。
<br/>　　虽然其中最为主要珍贵的宝藏，肯定是属于一个人的，但是其他进入其中的武者，也并非是没有机会，可以分一碗汤。
<br/>　　像是在九玄仙尊洞府之内，只要是在武道塔之中闯关，就可以得到相对应的奖励。
<br/>　　可九玄仙尊真正的宝藏，九玄仙尊，只有彻底通过武道塔二十一层考验的人，才可以得到。
<br/>　　现在楚风眠只要是身形一闪，就可以通过光幕，去得到宝藏，但楚风眠的目标，可是这大型宝藏之中最深处的宝藏。
<br/>　　这些宝藏，虽然不错，但是楚风眠还远远没有走到尽头，如果有武者，来到这里，正好是没有力气再进一步了，那么去收取一分宝藏，倒是不错。
<br/>　　不过这些宝藏，想要收取，也没有那么容易，如今楚风眠所在的这附近，只怕是八劫巅峰古帝，想要踏足都并不容易。
<br/>　　也就只有九劫古帝，由着自信可以踏入到这里。
<br/>　　没有实力的武者，就算是来到这大型宝藏之中，也就只能够是一无所获。
<br/>　　所以这种大型宝藏，一向都是运气实力，缺一不可，进入其中是运气，但真正想要得到宝藏，凭借的最终还是实力。
<br/>　　楚风眠的遁光继续在独桥之内穿行。
<br/>　　一片片的光幕都被楚风眠甩在了后面，这些光幕背后的许多宝藏，都已经是足以让许多九劫古帝疯狂了。
<br/>　　让一位九劫古帝，得到其中之一，都足以提升两三成的实力，是绝对的至宝。
<br/>　　得到这其中一份至宝，足以抵得上一名九劫古帝足足前数百年上千年的苦修了。
<br/>　　楚风眠却是全然无视，丝毫没有在意，他的目标，只有这独桥尽头的宝藏。
<br/>　　五千里。
<br/>　　足足是花费了两三个时辰的时间，在向前又前进了两千里。
<br/>　　在楚风眠的面前，又是一座宫殿的虚影，缓缓的显现出来，楚风眠一步步靠近过去，这宫殿才凝实出来。
<br/>　　眼前这一座宫殿，跟楚风眠之前看见的那两座宫殿之中的模样，一模一样，上面的禁制，也是被腐朽的很厉害。
<br/>　　楚风眠已经是轻车熟路，一股灵力凝聚在了楚风眠手心之中，当即一掌轰击而去。
<br/>　　轰隆！
<br/>　　这宫殿封闭的大门，被当场轰击而碎，在这宫殿内部的宝物显现了出来。
<br/>　　只是这一次，既不是诸天奇物，也不再是材料，而是一本古籍。
<br/>　　“武技？”
<br/>　　楚风眠眼神一亮。
<br/>　　他现在不缺少灵器，对楚风眠现在有着最大用处的，不是像是建木树枝那样的诸天奇物，也不是用来提升实力的圣丹，而是剑术。
<br/>　　楚风眠的一身实力之中最为重要的一点，便是楚风眠自己开创出来的九域剑术。
<br/>　　如果有强大的剑术被楚风眠得到，融入到九域剑术之中，楚风眠的实力也将会疯狂提升。
<br/>　　尤其是这大型宝藏之中，留下的宝物都是对九劫古帝有着大用的至宝，这种级别强者留下的剑术，绝对非同一般。
<br/>　　这武技，如果是一道剑术，对楚风眠来说可就有着大用了。
<br/>　　楚风眠一边期待着，一边一步上前，抓向了这一本古籍。
<br/>　　这古籍之上，还残留有着一股灵力，感觉到楚风眠想要抓取，顿时一股灵力浮现了出来，化为了一道屏障，准备阻止楚风眠。
<br/>　　“一本古籍，也想要反抗？”
<br/>　　楚风眠眼神一寒，脸色一怒，一股龙威便是笼罩而下。
<br/>　　在楚风眠将十二万九千六百滴远古战龙血脉全部凝聚出来之后，他现在身上的龙威，精粹无比。
<br/>　　尤其是楚风眠的身上可还是有着万龙之祖，始祖天龙血脉的，他身上的龙威，比起真正的一尊远古战龙身上的龙威，还要恐怖。
<br/>　　镇压下去，这古籍之上浮现出的一层灵力，被当场轰碎。
<br/>　　旋即楚风眠伸手一抓，这一本古籍便是落到了楚风眠的手心之中。
<br/>　　这一本古籍被楚风眠抓在了手心之中，一股纯粹的武意，便是被楚风眠感觉到。
<br/>　　“不是剑术。”
<br/>　　感觉到这武意，楚风眠稍微有些失望的摇了摇头。
<br/>　　既然是武意，而并非是剑意的话，那这一本古籍，多半的一本拳谱之类的。
<br/>　　对楚风眠来说，并没有多少的用处，不过他依然是翻开了这一本古籍。
<br/>　　翻开这古籍的一刻，这一股武意，突然化为了一头雄狮一般，冲天而起。
<br/>　　这一股威势，强横到了极致。
