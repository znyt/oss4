<br/>　　楚风眠站在原地，只是缓缓的抬起手，用手指向着天空一点。
<br/>　　一指点出，这一股威压，便是全然破碎。
<br/>　　任由屠雄的攻击多么凶猛，楚风眠便是站在原地，一动不动，毫不受到影响。
<br/>　　“此人修炼的，到底是什么功法？怎么会如此诡异？可以如此轻易的化解我的灵力？”
<br/>　　屠雄眉头一皱，他的连续两拳，在楚风眠的面前，都被轻松化解。
<br/>　　楚风眠的境界，只有御风境的境界，所以在他看来，楚风眠能够化解他的攻击，多半是因为功法的原因。
<br/>　　想到这里，屠雄猛然收敛起了身上的灵力，灵力全部被他融入到了身躯的肌肉之中去，来增添他的力量。
<br/>　　“哼，就算你功法特殊又如何？可以化解的了我的灵力，那倒是要看看，你能不能接下我的力量！”
<br/>　　屠雄脚下一动，猛然向着楚风眠冲了过来，这一次屠雄便是放弃了灵力，打算直接以肉身力量，来对付楚风眠。
<br/>　　以屠雄的实力，就算是肉身，都足以抗衡许多圣者了。
<br/>　　更何况他还刚刚吞噬了无数圣者精血，练就了鲜血魔体，力量更是倍增。
<br/>　　一瞬之间，屠雄便是冲到了楚风眠的面前来，舍弃所有灵力，化为了极为庞大的肉身力量，冲着楚风眠，直接一拳砸了过来。
<br/>　　“来的好。”
<br/>　　楚风眠目光闪烁，冷笑一声。
<br/>　　他也正好想要试试自己的肉身力量，达到了什么境地。
<br/>　　楚风眠本来修炼战龙决，他的肉身就要比起其他武者的要坚韧许多，更何况楚风眠还得到过雷霆炼体，最近更是又以天晶凝聚。
<br/>　　楚风眠正是打算有着试试肉身的意思，看到屠雄居然是舍弃灵力，以肉身打来。
<br/>　　楚风眠便是也将一身的灵力，全部收敛起来，同样以肉身，一拳之间轰了过去。
<br/>　　“疯了，此人是彻底疯了。”
<br/>　　“以肉身去抗衡屠雄？此人简直是在自己找死。”
<br/>　　很多人看到这一幕，都是不由的摇了摇头。
<br/>　　一尊神力境武者，就算是以肉身，也要轻易碾压御风境的武者。
<br/>　　肉身抗衡，可就是真正力量的对抗了，没有任何技巧可言。
<br/>　　在有着如此境界差距的情况之下，这力量差距，只怕是一眼就可以看出来。
<br/>　　“轰！”
<br/>　　就在众人议论之际，楚风眠跟屠雄的双拳，已经是撞在了一起。
<br/>　　“咔嚓。”
<br/>　　一声骨折的声音，突然响起。
<br/>　　“怎么可能？”
<br/>　　在场无数众人的眼神中，皆是目瞪口呆。
<br/>　　满眼不敢相信。
<br/>　　这骨折的声音，居然不是楚风眠，而是屠雄。
<br/>　　只见屠雄的右臂，便是以一种极为诡异的角度弯曲这，直接便是被一拳相对的力量，震到骨折。
<br/>　　“好恐怖的力量！这家伙莫非是一尊妖兽吗？”
<br/>　　屠雄目光闪烁，满脸不可置信，这一拳的力量，相对之下，他的力量居然是被死死的压制住了。
<br/>　　他的力量，居然被死死的压制住了。
<br/>　　楚风眠可是一个御风境的蝼蚁，居然是足以在力量上，压制住他。
<br/>　　“该死！我就不信这小子真的是无敌的不成！鲜血魔体！恢复！”
<br/>　　屠雄怒喝一声，一股血气便是围绕在他的身躯周围，顿时便是将他刚刚折断的右臂，彻底恢复过来。
<br/>　　这一次屠雄的气息，可是要比起刚刚又要强大了几分，真正的动了真怒，全力出手。
<br/>　　屠雄双手同时握拳，说话之时，便是冲着楚风眠轰击过去。
<br/>　　“我看你还能恢复多少次！都给我断！”
<br/>　　楚风眠大步上前，猛然五指张开，一爪抓了过去。
<br/>　　“咔嚓！”
<br/>　　屠雄的双臂，再一次在楚风眠的手中，直接是被扭断。
<br/>　　同时楚风眠的身躯，也已经是冲入到了屠雄的面前来，一脚对着屠雄的胸口踢了过去。
<br/>　　“轰！”
<br/>　　屠雄的身躯，足足是被踢飞数百米去，撞到擂台的墙壁上，才停了下来。
<br/>　　“可恶！一个御风境的小子，怎么会有着如此可怕的实力！”
<br/>　　屠雄再次站起来，他目光中本来的轻视之色，已经是消失的无影无踪。
<br/>　　他如今也看得出来，楚风眠绝对不是看起来的那般简单的人物，绝对是他的劲敌。
<br/>　　“我的全部实力，本来是打算进入到北邙学院在展现出来的，小子，你已经足以让我动用全力了。”
<br/>　　屠雄看着楚风眠，眼神中满是杀意道。
<br/>　　“我会让你知道，我的全力，到底是有着多么强横！”
<br/>　　说话之间，屠雄的身躯再次动了。
<br/>　　这一次，他的速度却是比起刚刚，要快上了数倍。
<br/>　　屠雄身上的力量，仿佛便是打破了一道封印一样，完全是展现了出来。
<br/>　　这屠雄，为了压制生死大劫，一直以来隐藏了自己大部分的力量，如今碰到楚风眠，他的全力，终于是爆发了出来。

<br/>　　跟这样的人物一同参加考核，绝对是不幸中的不幸。

<br/>　　楚风眠不愿意直接杀了他，也不过是为了表现的没有那般耀眼。
<br/>　　不然的话，就这小小考核，擂台上的所有人，楚风眠一息之间，都可以击败。
