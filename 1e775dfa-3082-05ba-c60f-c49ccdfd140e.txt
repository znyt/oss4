　　曹氏研究院的内部动荡让所有人都始料未及，曹春辉忽然的倒台毫无预兆，胡梦浩的上·位更一下子让整个研究院都气氛微妙。
<br/>　　连胡梦浩自己都一下子没能反应过来，他们辛苦了这么长时间的准备，竟然仅靠了一次在罗杰面前的胡言乱语，就直接让这座研究院易主。
<br/>　　可哪怕胡梦浩当上了新院长，能够彻底掌控所有的研究项目，却还是发现无法离开这里。
<br/>　　罗杰并没有将这座研究院的全部限权交给胡梦浩，而是用了另一种方式将这些研究人员，从上到下的全体软禁在这里。
<br/>　　虽说比之前自由很多，可依然与世隔绝。
<br/>　　这座研究院的所有出入口，都被神秘家族的人监控着，所有的研究物品来源也被切断，又罗杰派来的人掌管。
<br/>　　眼看希望出现又快速破灭，胡梦浩有些无奈：“还是出不去，外面的情况究竟如何也不知道。”
<br/>　　几名狼牙暗部的成员聚集在一起，相互望了一眼后也都沉默不语。
<br/>　　一旁的楚云天自罗杰等人离开后就一直没出声，此时他看着眼前的这些人，想了想有些紧张的问道：“我有一件事想确认。”
<br/>　　胡梦浩回过神，朝着楚云天点头：“之前多亏了你帮忙，有什么想问的你就说吧，我知道的一定如实告知。”
<br/>　　“我也是顺着你的话忽然想到而已。”楚云天摆摆手，而后神色认真道：“我想知道，狼牙的那个楚涵……”
<br/>　　胡梦浩看了几名狼牙暗部的成员一眼，然后朝着楚云天慎重点头：“就是你的儿子。”
<br/>　　当猜测变成了现实，楚云天猛地身躯一阵，一股狂喜的心情油然而生。
<br/>　　他被曹春辉控制了两年，一直担忧妻儿的状况，连是死是活都不知道，他这么长时间之所以能坚持下来，就是心中还有着一丝希望。
<br/>　　可他没想到自己的儿子不但没有在末世里死去，甚至还是一方霸主，从罗杰及曹春辉的口气中不难听出，楚涵已经强大到让这些人头疼不已。
<br/>　　“我儿子他？”楚云天激动的开口，有无数的问题想问，却一时半会儿不知如何发问。
<br/>　　“暗部的人说下吧。”胡梦浩扭头看向那几名成员，道：“我也挺想知道之后楚涵又做过什么。”
<br/>　　一名暗部成员中规中矩的走上前，先是敬了一礼，而后道：“我们也被隔绝外界时间不短，只能告知进来前的最近情况。”
<br/>　　“楚涵长官依然是华夏最年轻的上将，狼牙基地已经改造成了纯军事基地，安罗市内的丧尸全部清除后更名为安罗城，幸存者已经分批入住，风花雪月城依然是娱乐城，是狼牙最重要的经济来源地。”
<br/>　　“狼牙战团的发展已经成熟，各队伍风格显明，三战队在世上的名气很大，狼牙战团也被誉为华夏第一战团。”
<br/>　　说着那名暗部成员一顿，忽然想起了什么道：“对了，最重要的一点需要告知楚叔，除了您的儿子是闻名天下的将军之外，您的妻子也一切安好，并且是狼牙研究部的院长，新人类觉醒剂和等阶提升剂，袁熙叶院长都是主要研发者。”
<br/>　　楚云天双眼中的神色带着大喜，激动的满脸通红。
<br/>　　看到楚云天的神色，胡梦浩忍不住翻了个白眼：“不得不说你儿子楚涵真是个狡诈小人，自从知道你被关在这里之后，简直无所不用其极的找各种办法派人混进来，连我都被他忽悠了过来，一呆就是这么长时间，人生都浪费了！”
<br/>　　此番话一出，楚云天有些惊讶：“你是故意被曹春辉囚禁在这里的？”
<br/>　　“是啊！我本来已经逃离了，偏偏在最后一步即将成功的时候，楚涵出现在了我面前。”胡梦浩回忆着与楚涵唯一的一次见面，再次涌·出后悔的情绪，无奈的指着旁边几人道：“这些都是狼牙暗部的成员，专门混进来救你的，说实话我还真是羡慕你们父子之间的关系，能做到这一步的人实在不多了，你也清楚这个地方想混进来有多不容易。”
<br/>　　楚云天震惊的望着眼前的几名年轻人，这些人的面孔他都很熟悉，有研究人员也有清洁人员，负责什么样工作的都有，也不知什么时候起这些人就经常会出现在他周围。
<br/>　　以前楚云天没有想太多，可眼下知道真·相后，便顺便明白了最终原因！
<br/>　　这些人用尽了手段靠近他，时常出现在他附近，都是为了保护！
<br/>　　“真是辛苦你们了！”楚云天感动道。
<br/>　　胡梦浩此时也有些好奇，问道：“楚涵还有什么话让你们告知吗？”
<br/>　　暗部成员用极简的语言开口道：“长官只让两位遇到任何事都以保命为先，其他事情能不管就不管，等待时机。”
<br/>　　胡梦浩毫不意外，点点头后继续道：“你们几个混进来，狼牙暗部付出了不小的代价吧？”
<br/>　　暗部的成员都一愣，那名领头者苦涩一笑，回答道：“曹氏研究院是我们遇到最难攻破的地方，所花的金钱就不说了，根本就是个无底洞，到底花了多少都没去统计，暗部尝试了无数种方法，最终设下了好几个套路才成功。”
<br/>　　虽然知道艰难，但胡梦浩和楚云天也没想到会如此严重。
<br/>　　“有牺牲么？”胡梦浩了解狼牙内部的机制，忍不住问道。
<br/>　　那几名暗部成员的领头点点头：“参与这次任务的暗部成员人数为五百人，最终进来的就我们七个。”
<br/>　　乍然听闻这一数字，胡梦浩和楚云天都震惊无比，身躯不禁僵硬·起来。
<br/>　　“其他人……都？”楚云天声音有些发颤。
<br/>　　“都死了。”
<br/>　　良久的安静在屋内蔓延，一时间都无人说话，沉重的压力更蜂拥而至，也让胡梦浩和楚云天都倍感压抑。
<br/>　　尤其是楚云天，他完全没想到楚涵为了救他出去，不惜动用了如此大的代价。
<br/>　　狼牙暗部是一个什么样的组织他并不了解，但能够混入这密不透风的研究院内，可见这个部门的能力有多强，但却为了救他一人，而一下子牺牲了这么多人！
