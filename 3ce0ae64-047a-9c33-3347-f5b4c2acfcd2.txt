　　第44章
<br/>　　那场战斗造成的余波要在很长一段时间后才能平息。此事过后，程大雷彻底在青牛山方圆叫响了腕儿。人们只传说那蛤蟆大王每顿饭要吃十斤人肉，二斤人血，生得青面獠牙，方口阔面。
<br/>　　这下蛤蟆大王的匪号是叫响了，程大雷还打算找时间给自己想个上档次的外号，如温柔小斧，赛潘安俏宋玉什么的，如今大家都认了蛤蟆大王这四个字，再想改名会掉粉的。
<br/>　　相应的，程大雷也收获了一笔恐惧值，目前已达到4000 ，不过距离10000点恐惧值还有一定距离。
<br/>　　这几天程大雷就烦着这件事，怎样才能迅速提高恐惧值，获得建立点将台权限，看看这点将台究竟是怎么会事。
<br/>　　目前还没想到办法，总是突然跳到别人面前想吓人一跳，显得很滑稽。
<br/>　　事情发生三天后，程大雷在大树下乘凉，今天是个好天气，也就是很热。
<br/>　　“大当家，山下有人来了，看意思是想见你。”徐神机和黄三元过来。
<br/>　　“走，带我去看看。”
<br/>　　三人趴在山寨的墙头上往下看，只见山下有三匹马，为首一人着黑衣，目光若有若无看着蛤蟆寨大门。
<br/>　　“他是干嘛的？”程大雷困惑问。
<br/>　　“我认得他。”黄三元突然道。
<br/>　　“喔，说来听听。”程大雷手搭凉棚，望着山下，只见为首之人板着一张脸，表情肃穆。
<br/>　　“黑石城有一个本地帮会，叫做野狼帮，帮内有一二百喽啰。我认得山下这人，叫做李甲，他就是野狼帮的老大，这个人盘踞黑石城很多年，是个人物。”黄三元道。
<br/>　　“野狼帮……听听这名字，一看就没文化。”程大雷不屑道。
<br/>　　徐神机和黄三元无语，好像蛤蟆寨这三个字也没啥文化。其实山寨和帮会一样，都属于绿林，也就是黑·道，当然，在人们心目中的地位，帮会更高一些。
<br/>　　“我们和野狼帮井水不犯河水，从前也没打过交道，你说他过来干什么呢？”徐神机。
<br/>　　“我觉得……他可能是来给薛半川当说客。”程大雷道：“薛半川想把那笔寿礼赎回去。”
<br/>　　“这……”徐神机难以置信道：“不太可能吧，我们抢了他们的东西，他不攻打我们也就是了，怎么还会和我们合作，我们可是山贼。”
<br/>　　“他不是不想打我们，而是没时间。”程大雷道：“薛半川刚篡了城主的位置，城内很多势力要平息，幽州王的寿宴快近，他没有那么多时间跟我们耗，所以把这笔寿礼赎回去是最好的办法，当然，薛半川如果腾出手来，收拾我们是肯定的。”
<br/>　　“大当家言之有理。”黄三元赞道。
<br/>　　“不过我想不明白的是……”程大雷困惑的看着山下的李甲：“他来谈判也不是不可以，但他在山下杵着干什么？”
<br/>　　“我听说……”黄三元道：“这个李甲架子极大，曾经韩虎踞要见他，两个时辰他一句话不说，最后还是韩虎踞先开口，他才说话。”
<br/>　　“喔，原来是个骄傲的人呐……”程大雷嘴角抿起：“我就是喜欢骄傲的人。”
<br/>　　“要我不下去请他上山？”黄三元道。
<br/>　　“请什么……既然他喜欢等着，那就让他等着吧。”程大雷仰起身看着天上的日头：“今天是个好天气呐。”
<br/>　　今天的确是好天气，好天气的意思是很热，山林中的环境又燥热些，汗水出了一层一层，粗麻布衣服贴在身上，有密林中的小虫往身上扑，包括苍蝇、蚊子，臭虫……习惯空调房的现代人根本想象不到这种痛苦。
<br/>　　当然，这个时代人们忍受酷热的能力还是很强的，李甲也并非养尊处优，所以一开始他还忍得住。
<br/>　　可是，这事情和他想象得不一样。程大雷分析得不错，李甲的确是被薛半川所托，来充当双方的说客。他算是绿林道的，在双方之间说话方便。不过，李甲不大看得起山贼，绿林道也是有鄙视链的，野狼帮这些本地帮会，都是结交达官贵人，做些灰色生意，我们是文明人，怎么能和山林中的土老帽一样。那句话怎么说来着：最讨厌你们这些打劫的，一点技术含量都没有……
<br/>　　的确，程大雷击溃了陆哼，威名……好吧，是臭名远博。但这有什么，无非是用了一些陷阱而已，而陆哼的战斗力……也没有吹得那么玄乎嘛。
<br/>　　所以李甲有一种下乡慰问的意思，来到蛤蟆寨后给他们聊聊城里的见闻，让他们开阔一下眼界。顺便再说一下自己的来意，比如什么『城主很生气，后果很严重』『幸亏我看在绿林道的情分上，给你们压下来了，要不你们就死了』『快点把寿礼拿出来吧，薛城主大恩，赏你们些钱……这可是我好不容易求来的』……
<br/>　　但事情和他想得不一样，很不一样，话说，今天的天气怎么这么热。衣服湿漉漉黏在身体上，似乎有蚂蚁落在自己身上，又痒又痛，自己要是解开衣服看看，会不会破坏自己的气质。
<br/>　　明明看见几个人在墙头鬼鬼祟祟，怎么还不开门迎接，难道是准备欢迎仪式……嗳，不用太麻烦，不用太麻烦……
<br/>　　不过这准备时间也太久了些，一直从上午到日头正中，火一般烧灼的阳光落在地上，树叶和草丛都打蔫了。
<br/>　　李甲晕晕乎乎，在马上一跌一跌，现在他差不多明白，这好像并不是准备欢迎仪式的节奏。
<br/>　　望着头顶紧闭的寨门，他深吸一口气，用最大的声音喊道：“野狼帮李甲……求见……”
<br/>　　程大雷睡过午觉，正用井水洗脸，徐神机走过来：“大当家，下面开始喊了，要不要再耗耗他们……”
<br/>　　“耗什么呀，再耗就中暑了。”程大雷擦了擦脸：“让他们上来，你和黄三元和他谈，看薛半川会出什么价？”
<br/>　　在李甲迷糊的视线中，蛤蟆寨寨门打开，黄三元在他视线中出现，李甲如同见到救星。还想摆一下架子，不过想到刚才发生的事情，还是打消了这个主意。
<br/>　　“黄管家，好久不见！”
<br/>　　
<br/>(本章完)
