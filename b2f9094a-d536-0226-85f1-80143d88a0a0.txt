　　不料，那龙长老微微一愣之后，旋即哈哈大笑，朗声道：
<br/>　　“小丫头，你说的是叶狂仙吗？那我劝你还是死了这条心吧！自古以来，踏入通天路之人，还没有人能够活着回来！”

<br/>　　“不惧？当年我们叶门主，单枪匹马杀入你们蓬莱仙界，五大宗门尽皆俯首，若非叶门主念在上天有好生之德，早就将你们蓬莱仙界连根拔起了，你这老匹夫哪里还有机会，在这里大放厥词？”

<br/>　　果不其然，唐清雅抓住了龙长老的漏洞，当即冷喝道：
<br/>　　“所以，你终于承认自己是蓬莱仙界的人喽？”
<br/>　　“我……”
<br/>　　龙长老顿时脸色巨变，一下子哑口无言。
<br/>　　哗！
<br/>　　周围众人反应过来之后，也顿时一片哗然，而网络直播间里的观众们，听到龙长老这话，顿时也一下子炸开了锅！
<br/>　　“我靠！剧情大反转啊！没想到这青龙派，竟然真的是蓬莱仙界的奸细！”
<br/>　　“我就说嘛，我的女神怎么可能会是十恶不赦的坏人？”
<br/>　　“照这么说来，那沈青山也是蓬莱仙界的人了？这太可怕了！”
<br/>　　“我们需要一个真相！”

<br/>　　蓬莱仙界这十多年来，频频在华夏兴风作浪，在华夏民众心中，就是天外邪魔的代名词，如今听到龙长老，亲口承认自己是蓬莱仙界的人，顿时全都坐不住了，纷纷口诛笔伐起来。
<br/>　　远处的那座山峰之上，蓬莱仙界的那几位强者，看到这个变故之后，同样个个脸色无比难看。
<br/>　　“糟了！龙乾这个蠢货，居然被一个小丫头片子给带沟里去了！”
<br/>　　“这下我们和平征服华夏的计划，怕是要彻底泡汤了！”
<br/>　　“看来最终还是要靠武力来解决，大家准备一下，一旦情况不对，随时出手，先将今日到场这些华夏武者全部灭掉！”
<br/>　　……
<br/>　　另外一边，
<br/>　　而那龙长老经过片刻的呆滞之后，顿时气得双眸通红，身上升起一股惊人的杀意，之前道貌岸然、慈眉善目的形象，顿时一下子荡然无存，死死盯着唐清雅，咬牙切齿道：
<br/>　　“好你个小丫头！你竟然敢套老夫的话！找死！！”
<br/>　　暴怒之下的龙长老，此时哪里还顾忌自己的形象问题，手掌直接猛然一握！
<br/>　　轰隆隆！
<br/>　　随着龙长老这一掌握下，那个巨大的牢笼瞬间爆炸开来！
<br/>　　网络直播间里的观众，看到这一幕，顿时群情激愤起来。
<br/>　　“这老家伙真不是个东西！他这是要杀人灭口了吗？”
<br/>　　“所以唐女神说的都是真的，这老匹夫和整个青龙派，真的都是蓬莱仙界的奸细？”
<br/>　　“唉！可惜了这么一个绝色大美人，就这么香消玉殒了！”
<br/>　　“这个老畜生！不得好死！”
<br/>　　……
<br/>　　然而，
<br/>　　在众人的惊呼声中，那片空间爆炸过后，待一切恢复如初，众人看清楚擂台上的场景，顿时全都惊呆了，
<br/>　　只见，
<br/>　　唐清雅依旧站在擂台中央，竟然毫发无损！
<br/>　　“什么情况？”
<br/>　　“刚刚那种程度的攻击之下，她居然没死？”
<br/>　　“难道是那老匹夫最后手下留情了？”
<br/>　　……
<br/>　　众人纷纷望向青龙派的那位龙长老，却见他同样也是一脸惊愕之色，
<br/>　　“这怎么可能？”

<br/>　　而唐清雅也是一脸迷茫，她刚刚都已经完全放弃抵抗了，以为自己必死无疑，可就在龙长老发出那一击的时候，她猛然感觉周身上下，似乎被一股神秘的力量所包裹，替她挡下了对方那致命的一击！
<br/>　　就在所有人全都愣住，整个天台上，陷入一阵寂静之时，一道淡漠的声音突然响起，

<br/>　　众人全都一惊，纷纷循声望去，
