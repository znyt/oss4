　　“叶天师，你怎么了？”青冥道长以为自己说错了什么，急忙问道。
<br/>　　“没什么。”叶少阳摇了摇头，没有说为什么。青冥道长永远也不会猜到，只是因为他刚才说了自己的道号，青冥子三个字，一瞬间，让叶少阳想到了自己的师父青云子，然后突然就郁郁了。
<br/>　　空界一战，青云子并没有参加，之后也没见过他，他老人家一定也知道自己失踪了，不管怎么样，他一定很为自己担心。
<br/>　　深深吸了口气，叶少阳更是坚定了想要早点回去的决心。
<br/>　　赶尸的事情敲定之后，青冥道长带叶少阳去客厅喝茶，找来一个尸工，叶少阳虽然在法术方面见识极深，但毕竟对当地不熟。青冥道长给叶少阳介绍了一下，这个尸工叫陈三，是个经常赶尸的，穿山越林，对周围一带的地形非常熟悉，可以给叶少阳打下手。
<br/>　　陈三看着有四十来岁，风吹日晒的，一脸褶子，不过眼神很和善，只是听说要给叶少阳打下手，顿时有些吃惊。之前叶少阳为女尸断命的时候，他不在场，虽然是青冥道长安排的，但是看叶少阳才二十出头，细皮嫩肉的，不禁还是有些怀疑他的实力。
<br/>　　“叶天师，我这义庄都是放死人的地方，不太适合住，也没法招待，不然……晚上用过酒饭，今晚就出发？”赶尸都是晚上出发，白天休息，因此青冥道长才会这么说。
<br/>　　叶少阳一口答应。
<br/>　　青冥道长于是让人安排酒饭，然后亲自去为叶少阳做赶尸前的准备，张道长也去帮忙，留下叶少阳和翠云在客厅里喝茶。
<br/>　　人都走后，叶少阳转头看着翠云，笑着问道：“你是不是有点怕？”
<br/>　　翠云本想说不怕，但是想了想，还是老实地点了点头。
<br/>　　“姐，我想过了。你还是不要跟我赶尸。我先送你去县城，然后你找个地方住下，等我回来怎么样？”其实叶少阳自己也不知道为什么要回来找她干什么，但是她毕竟是跟着自己出来的，叶少阳觉得自己得对她负责，而且他是不喜欢离别的人，不想就这么彻底分开。
<br/>　　如果真的要分开，至少也是自己能够离开这个世界的时候，妥善安排好她，然后再来个道别。
<br/>　　翠云咬了咬嘴唇，说道：“你是觉得姐没用了吗，在路上会当你的累赘。”
<br/>　　“怎么可能！”叶少阳差点跳起来，“我是怕你不习惯跟尸体打交道，而且我们一走就是一晚，我怕你吃不消……”
<br/>　　翠云想了一下，说道：“喏，怕尸体是有点，不过有你在，也没啥怕的，至于体力你就别担心啦。你姐我是个寡妇，地里活都是我一个人干，赶路这种事，真没啥辛苦的。”
<br/>　　叶少阳还想劝说，翠云接着说道：“少阳你别说了，姐也不放心你一个人去，姐虽然没什么本事，但是衣食方面可以照顾你，只要你嫌弃姐，姐就一直跟着你，直到你娶媳妇。”
<br/>　　叶少阳望着她，心中满满都是感动，点了点头。
<br/>　　晚上，青冥道长张罗了一桌饭菜，大家吃完之后，一行人来到义庄，青冥道长已经让人把需要赶的尸体都找人立了起来，换上了长长的衣服，这种衣服叫尸袍，是用糯米汁浆过的，可以压制尸体身上的尸气，将尸变的可能降到最小。
<br/>　　除了那具女尸，另外还有五具尸体，靠墙摆着，脑袋都用纱布包着，只露出五官，肉身是用秘法配制的法药涂过，能保证几天之内不腐烂，但是尸体的表皮也都变成了酱紫色，在烛火下看着更是显得阴森森的。
<br/>　　翠云一看到这些尸体，立刻躲在了叶少阳身后。
<br/>　　青冥道长拍着叶少阳的肩膀，说道：“兄弟，这趟走尸就拜托你了，别的事情，我全交代给陈三了，这一路上有几个客栈，他会安排宿头，不必你操心。”
<br/>　　叶少阳一口答应。
<br/>　　青冥道长带着叶少阳在香案前焚香祝祷，叶少阳看神龛上供着的是一个长须的古代男子，叶少阳认得出是上古巫医巫咸，严格来说巫咸不是神，不过却是华夏历史上能追溯到的第一代巫医，曾辅佐过商纣王，创造了就很多巫术。
<br/>　　各行各业都有自己的祖师爷，例如打渔的拜姜子牙，杀猪的拜张飞，小偷拜时迁，都是找个同行业的第一个知名度最大的人来参拜。赶尸属于巫术，起源于道教众阁派，但真正发扬是在湘西巫术，所以拜的不是道门众神，而是巫术的祖先巫咸。
<br/>　　叶少阳身为道门天师，自然不可能参拜任何道教之外的神，只是拱了拱手，算是礼节了。
<br/>　　青冥道长从神龛上取下一方木头做的令牌，交给叶少阳，抱拳说道：“叶天师，此番走尸，就拜托给你了！”
<br/>　　叶少阳接过赶尸令，对他抱了抱拳。
<br/>　　青冥道长让人倒了两杯酒，端给叶少阳和陈三，叶少阳看酒色发红，端起来一闻，是糯米和雄黄的混合味道，立刻就明白了，喝了雄黄酒，能够一定程度上加固自己体内的阳气，邪气不侵。糯米更不用说，是僵尸最怕的东西。赶尸的途中，赶尸匠都要喝这种酒。
<br/>　　叶少阳把酒碗端给身后的翠云，让她喝一些。
<br/>　　“俺不会喝酒。”翠云犹豫着。
<br/>　　“这是糯米酒，没什么酒精，你少喝点。”
<br/>　　翠云接过去喝了一口，果然没什么强烈的刺激，于是多喝了几口。
<br/>　　“抱歉，我把令姐给忘了。来人，再端一碗雄黄酒给叶天师！”青冥道长急忙说道。
<br/>　　“不用了，我不需要喝这个。”
<br/>　　叶少阳这句话，让青冥道长等人有些不解，叶少阳也没好解释，自己是先天灵体，基本上一般的邪气根本没法对自己造成任何伤害。
<br/>　　青冥道长也没坚持，从兜里摸出一块怀表，看了眼说道：“酉时三刻，起尸吉时，叶天师，请！”
<br/>　　叶少阳朝靠墙那排尸体走过去，来到第一具尸体面前，从背包里拉出墨斗，上面挂上一枚铜钱，栓在了尸体的脖子上。
