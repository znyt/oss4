　　第84章
<br/>　　“大哥，你听说了没？”
<br/>　　飞虎寨，高飞豹这样问。
<br/>　　“我还不是听你说的。”高飞虎很无奈的白了自己弟弟一眼。
<br/>　　“那大哥你觉得是真的么？”高飞豹立刻问。
<br/>　　高飞虎沉默下来，消息传得沸沸扬扬，现在他已经知道些。一亩辣椒的产量是其他作物的十几倍，想想就觉得不可思议，所以他潜意识中就觉得这是假的。
<br/>　　可万一呢……
<br/>　　若万一是真的，那之后的日子怕自己都要睡不着觉了。为什么……心疼呐。
<br/>　　“怎么可能是真的……蛤蟆寨竟整些幺蛾子，呵呵……”
<br/>　　“那万一呢……”
<br/>　　“呃……什么万一，哪有那么多万一，说了不可能是真的，就不可能是真的。”
<br/>　　“可万……”见大哥脸上表情不好，高飞豹没有继续说下去，他忽然道：“大哥，咱们山寨也有几百斤辣椒，不如我拉着去看看，如果是真的，自不必说，如果是假的，我非要给蛤蟆寨一个好瞧。”
<br/>　　飞虎寨的确也种了一些辣椒，只是不多，也就是房前屋后，犄角旮旯种了一些，收货后约有四五百斤。平日吃饭时也用了些，不得不说，味道还是很好的。可若说这一斤辣椒能换一斤粮食，高飞虎是不信的。
<br/>　　“你愿意去就去吧，去了也就死心了。我早就说过不让你种，可你偏偏不听，有这次，你就知道大哥为什么是大哥了。”
<br/>　　高飞豹套上马车，用麻袋把辣椒装了，叫了几个山寨的弟兄，一路去往蛤蟆寨。
<br/>　　相距倒也不远，不多时便到了。只见蛤蟆岭下还有几个人，都是其他山寨聚在这里打探消息的。
<br/>　　一见高飞豹这幅架势过来，立刻有人站起来道：“高二爷，你这是干嘛……我记得是你当初不让我们种的，原来你自己竟偷偷拿回去种。”
<br/>　　“什么话！我为什么要种，我就是要拆穿他们。现在我拉着辣椒来了，如果兑不出粮食，我看蛤蟆寨怎么说。”
<br/>　　“二爷，高啊！”有人翘起大拇指。
<br/>　　“一般，也就一般般啦。”高飞豹摆摆手，道：“这样，你们和我一起上山，到时候看看程大雷怎么下台。”
<br/>　　“走，大家陪二爷一起去。”
<br/>　　“走，同去，同去！”
<br/>　　一群人十几个，陪着高飞豹一起上山。这幅架势倒是让守山的林少羽吓了一跳，怎么要来打山。不过，现在蛤蟆寨有人了，凭着十几号人也做不得什么事。况且现在蛤蟆寨外的山坡上，除开一条上山的小道，其他都是陷阱。真如果再来百八十号精兵，蛤蟆寨想阻挡不会再像从前一样放心。
<br/>　　黄三元接待了高飞豹，互相寒暄过，黄三元才知道高飞豹的来意。
<br/>　　“黄总管，我们是来换粮食的，说过的话可不能不算啊。”高飞豹昂着头道。
<br/>　　“您还是第一个到的呢。”黄三元说着检查辣椒，忽然愣住：“高二爷，你这辣椒没有炒过啊？”
<br/>　　“没有炒过怎么了，难道我还要炒熟给你嘛。”高飞豹提高声音。
<br/>　　这幅场面倒是唬住黄三元，他道：“二爷在这里等一会儿，我去喊一下大当家。”
<br/>　　高飞豹心里彻底明白了。他冲周围人道：“大家都不傻，我想都看明白了。蛤蟆寨这是拿不出粮食，要找理由搪塞呐。兄弟辛辛苦苦种了辣椒，他要说辣椒长得小了，颜色不好了，这都是推脱的理由。”
<br/>　　“是呐，对亏当初听了二爷的话，否则咱们兄弟白辛苦一场不说，还瞎了好几亩地。”
<br/>　　“别着急，等程大雷来了，看他怎么说，要是不给个说法，我高飞豹第一个饶不了他。”
<br/>　　“放心，青牛山五十多家山寨，几千号人都支持你。这程大雷不能把我们当猴耍。”
<br/>　　一般人亢奋起来，等程大雷到的时候，却都安静下来，要看看程大雷怎么下台。
<br/>　　程大雷过来后检查下辣椒，的确没有炒过。蛤蟆寨为防止辣椒种子流出去，都要干炒一遍，这件事，飞虎寨可能还不知道。
<br/>　　不过最后，程大雷还是摆摆手，道：“如果别人是不行的，可二爷既然已经来了，就别让他白跑一趟。反正也不多，咱们山寨辛苦辛苦，顺手炒了吧。”
<br/>　　“是。”黄三元道：“卸货，给二爷兑粮食。”
<br/>　　高飞豹却是愣住了，结结巴巴道：“你说什么？”
<br/>　　“给你兑粮食啊，我们都是小麦兑，一斤辣椒兑一斤小麦，二爷没有意见吧？”
<br/>　　“没……”高飞豹忽然觉得莫名压力山大，似乎周围人的目光都落在自己身上。
<br/>　　卸车，过称，把小麦拉出来。人多力量大，这些事情却没有花多少时间。高飞豹一共拉过来五百斤辣椒，不多时就给他换成小麦装到车上。
<br/>　　“高飞豹！”
<br/>　　一个刺耳的声音响起，高飞豹回头，看到刚才还唤自己『二爷』的一人，此刻已经咬起牙。
<br/>　　“当初可是你不让我们种的，结果自己却回去偷偷种，你们飞虎寨就是这么做事的嘛。”
<br/>　　一众人都虎视眈眈的盯着高飞豹，眼中的火都要喷出来。
<br/>　　“干什么！”高飞豹一头瀑布汗：“没听他刚才说吗，其他人是不行，也就是我们飞虎寨面子大，他不敢赖我们，还有我们种的少，他拿几百斤粮食堵我们的嘴，换你们其他人行嘛？”
<br/>　　众人目光犹疑，此刻还拿不准这件事，如高飞豹所说，也未必没有可能。
<br/>　　这时候，又有一队人上山，未见其人，便有香风扑面，莺莺燕燕的声音响起。
<br/>　　整个青牛山有如此大排面的人也就杏花岭了。一众女人拉着马车上山，碰到这帮山贼却也不害羞，互相开着下三路的玩笑。
<br/>　　“白当家来了，快来人，帮忙卸货，给白当家换粮食。”黄三元高声道。
<br/>　　整个青牛山，也就杏花岭种的辣椒比较多，双方合作也算比较和谐。辣椒都炒过一遍，完全符合标准。
<br/>　　高飞豹脸上的瀑布汗又来了，他只感觉自己身周围杀气弥漫。
<br/>　　“高飞豹，怎么，杏花岭也有面子，她们种的辣椒也少！”
<br/>　　
<br/>(本章完)
