        一个人闷在家里，童筱颖并未等到傅夜沉回来，而是等来了从澳洲坐专机过来的“小爱”。
<br/>
<br/>        小爱被托运员送上了家门，童筱颖站在院门口签收时，被关在宠物笼里的小爱，看到她后兴奋不已地“汪汪”直叫。
<br/>
<br/>        童筱颖随后将小爱从宠物笼里放了出来，轻轻地抱在了怀中。
<br/>
<br/>        托运员将随博美犬小爱一起托运过来的资料和狗粮一并递给了童筱颖后，拿了签收单就离开了。
<br/>
<br/>        自从她和昼景离婚后，小爱就像一个被亲妈亲爹抛弃的孩子，被寄养在了澳洲的宠物幼儿园里，起初，她以为，昼景会管它，却没想到昼景也不想要小爱。
<br/>
<br/>        既然养了它，就不要轻易遗弃它。
<br/>
<br/>        这是澳洲那家宠物幼儿园的园长，给她发的电子邮件里说的。
<br/>
<br/>        虽然，看到小爱会触物伤情，但是，童筱颖决定坦然面对。
<br/>
<br/>        她都已经跟昼景结束了，小爱是无辜的。
<br/>
<br/>        “小爱，以后，就由妈咪来守护你！”童筱颖亲了亲小爱的小脑袋，发现这小家伙身上有些发臭后，决定先带它去宠物店里做个洗护，再做一个检查，然后买一些生活用品回来。
<br/>
<br/>        出发前，童筱颖用手机APP挑了一家滨江城里最好的宠物店，而且还给小爱订了一套八百八十八元的VIP独立护理套餐。
<br/>
<br/>        或许是因为，这段时间被大哥给养叼了，大哥给她的吃喝穿住全是最好的，所以，童筱颖也想给她家小爱享受这最好的待遇。
<br/>
<br/>        到了宠物店里后，童筱颖怀抱着小爱，跟在女服务员身后，上了楼上的VIP套间。
<br/>
<br/>        她们才到楼梯平台上，就只见那边的VIP套间门口，有两个女人正和另一个女服员说些什么，其中一个年轻的女人怀里还抱着一只黑白色长毛小型犬。
<br/>
<br/>        童筱颖虽然养狗，但是对狗狗的品种不大了解，但是也看得出，那个年轻女人怀里抱着的应该是一条名贵犬。
<br/>
<br/>        “既然我家安迪喜欢这个VIP套间，那你们就给我家安迪换这间好了。”年轻的女人一边抚摸着怀里的狗狗，一边傲慢地说道。
<br/>
<br/>        她怀里的狗狗，还十分认同地吠了一声。
<br/>
<br/>        童筱颖忽然间觉得那个抱着狗狗的女人很眼熟，但是一时间又想不起来在哪里见过。
<br/>
<br/>        旁边那个穿着粉色西装套裙的女人，立即用谄媚的声音，随声附和道：“是是是，我们马上给您和安迪公主换。”
<br/>
<br/>        “可是，店长，这个VIP套间已经预定出去了，对方也全额付款了。”女服员一脸憋屈地说道。
<br/>
<br/>        女店长正色道：“那位顾客若是来了，就带她去另一个VIP套间不就行了。”
<br/>
<br/>        “好的，店长。”女服员抿了抿唇，只能应声领命，然后微微颔首，转身离去。
<br/>
<br/>        这边，陪同童筱颖的女服员见状，倒是挺会随机应变，立即对童筱颖微笑着说道：“童小姐，您给您的爱犬订的套间在这边。”
<br/>
<br/>        童筱颖顿时一脸不悦地皱了下柳眉，不听身旁这女服员的话，而是直接朝那个女店长走了过去：“店长，未经过我家小爱的同意，就擅自给我家小爱换房间，似乎不妥吧？”
<br/>
<br/>        这个VIP套间是她事先在手机APP上下单时看中的，里面是装修风格是根据狗狗的视觉色彩辨识度来装修的，而且里面的设施齐全。
<br/>
<br/>        而这个VIP套间里大部分是黄、蓝、灰三种颜色，她觉得小爱会喜欢，因为小爱对这三种颜色情有独钟。
<br/>
<br/>        女店长愣了一下，瞥了童筱颖一眼，又看了看陪同在童筱颖身旁的女服员。
<br/>
<br/>        女服员与女店长的目光对视，会意了后，连忙疾步走到女店长的身边，轻声解释道：“她就是今天全额付款的那位童女士。”
<br/>
<br/>        女店长了解完情况后，立即面带微笑地看着童筱颖，并和颜悦色地说道：“童小姐，我们隔壁的那个VIP套间，也非常的不错，我带您过去看看。”
<br/>
<br/>        “这间本来就是我家小爱订的，我为什么要换？”童筱颖淡定地反问。
<br/>
<br/>        “这——”女店长顿时有些为难了。
<br/>
<br/>        按理来说，她确实不应该自作主张地给这位童女士换套间，毕竟她提前预定的，而且也已经全额付款了。
<br/>
<br/>        可是，她身旁的这位李小姐，也不好得罪，人家可是市长家的千金大小姐，而且她怀里的狗狗也是犬中的贵族罗秦犬。
<br/>
<br/>        而那位童女士怀里的狗狗，只不过是一只品种不纯，价格不过两三百块钱的博美犬罢了。
<br/>
<br/>        在女店长的眼里，就觉得是什么样的人，养什么样的狗狗。
