
<br/>　　<br/>　　    七月份的天气不是六月份的天孩子的脸说变就变，上午清河崔氏来人提亲的时候还晴空万里不见一片乌云，早间的天气预报也说今天天气不错，甚至李兰荣和茅清水特意查了下今天的黄历，今日易婚丧嫁娶。<br/>　　<br/>　　    除去以上条件，那这突然冒出来的雷是打哪劈下来的？<br/>　　<br/>　　    老天爷感冒，忽然打了个喷嚏？<br/>　　<br/>　　    五道惊雷落下之后，正厅里一片寂静，随即外面就有脚步声传来，李家的人慌慌张张的跑了过来，完全不顾屋内凝重的气氛，就嚷嚷道。<br/>　　<br/>　　    “族长，刚刚的雷，把咱家的祖宗祠堂上面给劈了一道窟窿，好几个祖宗牌位都碎了·····”<br/>　　<br/>　　    “唰”正厅内至少十几道目光突然间就汇集到了王惊蛰的身上，特别是李兰荣，茅清水还有李伯和李氏的几个老人全都一脸怒气，气的身子都哆嗦了。<br/>　　<br/>　　    毫无疑问，这雷劈的肯定跟王惊蛰有关系，谁都看见了他脸上挂的那道戏虐的笑容。<br/>　　<br/>　　    祖宗牌位对于一些延续了千年的大家族来讲乃是一族的荣耀，特别是李家的灵位还有好几个是曾经响彻历史的大人物，比如就有那位高祖的名号。<br/>　　<br/>　　    能坐在屋里的自然不是外面那些不信鬼神的普通人，谁都多多少少知道这世上除了人和牲口以外，还有很多常人所不知道的东西。<br/>　　<br/>　　    比如南昆仑掌风水一道，他们就清楚这九百六十万平方公里，不管在哪都脱离不开风水两字。<br/>　　<br/>　　    比如梵净山天道峰被誉为道门图书馆，道藏无数，自然也知晓妖魔鬼怪魑魅魍魉什么的并不是杜撰出来的。<br/>　　<br/>　　    再比如，崔里两家都有千年历史，家中典籍留存下来的也不少呢。<br/>　　<br/>　　    所以，谁都清楚刚刚那雷声可不是突然巧合间就落下来的，也不是老天爷抽疯了，而是肯定出自王惊蛰的身上。<br/>　　<br/>　　    只是，谁都奇怪了，这雷是怎么劈下来的呢？<br/>　　<br/>　　    崔玄策的目光深深的看了王惊蛰一眼，然后从他的身上挪开，又落到了李家人那边，缓缓地说道：“据我所知道门里有一种真言誓语，一旦立下了就绝无反悔的可能，除非是立誓之人亲自撤了，或者两方协商之后共同放弃，不然如果对方擅自立誓不守的话，必然会受责罚，我刚才想了想，这位先生之前说道李伯父你敢说一句李家长孙女和朝歌王氏一门婚约一事并不存在这样的话么，随后伯父你就依葫芦画瓢的重复了一遍，这雷就劈了下来，我想这就是道门中的真言誓语了吧？如此说来，这位来自朝歌的王氏一门中人看来是真和你们陇西李氏这一代的大小姐之间有一场婚约了，不然这雷劈的岂不是毫无道理了？”<br/>　　<br/>　　    赊刀一脉最重承诺，赊刀人立下的誓言从不可破，如若赊出去的刀取不回来，就会印证了王惊蛰之前所说过的那句话。<br/>　　<br/>　　    “后果自负·······”<br/>　　<br/>　　    只不过这次让陇西李氏后果自负的不是王惊蛰自己，而是老天代为效劳了。<br/>　　<br/>　　    崔玄策的这句话无疑相当于是给李氏的脸上“啪，啪”的扇了一顿小巴掌，巴掌扇的响亮无比，之前李兰荣和茅清水都信誓旦旦的说王惊蛰是胡搅蛮缠，一派胡言，紧接着李家祠堂就被劈了。<br/>　　<br/>　　    真真是耳光响亮啊！<br/>　　<br/>　　    “是又怎么样？这婚约立的不明不白的我们为什么要认？李家长孙女为什么要认·····”李家一个老人冷冷的笑道：“现在好了，婚约出来了，雷也劈完了，这订婚是不是还得继续往下订啊？”<br/>　　<br/>　　    王惊蛰愣了愣，鄙夷的从嘴里吐出三个字：“不要脸！”<br/>　　<br/>　　    李氏老人又很不要脸的说道：“那又怎样？”<br/>　　<br/>　　    “呵呵······”王惊蛰留下个神之一笑也不说话，扭头就走。<br/>　　<br/>　　    李兰荣随即冲着崔辰良说道：“辰良兄，这无非就是个小插曲罢了，插曲过了就结束了，我们还是来继续谈谈两家订婚之事吧？”<br/>　　<br/>　　    崔辰良皱了皱眉，这插曲来的不太美妙，甚至还有点恶心人，但就像李兰荣说的那样插曲已经过去了，婚是不是还可以接着订呢。<br/>　　<br/>　　    崔辰良扭头跟南昆仑的长老低声问了一句，对方点了点头，示意订婚照旧，虽然李家的长孙女有婚约在身，但现在似乎已经过去了，李家人想嫁女，他们崔家想娶，这确实有在继续的道理。<br/>　　<br/>　　    这时候王惊蛰已经走到了正厅门口，略微侧着脑袋轻轻的摇了摇头，正要接着迈步出去的时候，李兰荣手边的电话响了，他看了眼电话的屏幕。<br/>　　<br/>　　    打电话过来的，是李家看守陵园的老人，每年中只有李氏祭祖的时候陵园那边才会有电话进来。<br/>　　<br/>　　    陇西李氏有两处陵园，其一人尽皆知在唐昭陵，现在差不多已经成为了旅游胜地。<br/>　　<br/>　　    那处陵园说实话已经不归李家所有了，只是每年都过去祭拜。<br/>　　<br/>　　    还有一处陵园就在陇西渭河附近，是自大唐之后李家历代族长还有重要人物的陵寝。<br/>　　<br/>　　    李兰荣诧异的接起电话，里面就传来守陵老人低沉的动静：“兰荣，陵园这里出了变故······”<br/>　　<br/>　　    李兰荣当即大惊，连忙问道：“怎么了？”<br/>　　<br/>　　    “你父亲的陵墓开裂了，里面钻出了不少蛇虫蚁兽，还有你太爷爷和兄长的墓，也都出了问题，我刚才粗略的看了几眼，至少有四五座墓有异”<br/>　　<br/>　　    这边守陵老人的电话还没有结束，李家又有人来到正厅，慌张的说道：“后院里的那口老井，有水从井口里面溢了出来，已经把院子都快浸满了”<br/>　　<br/>　　    李家大院后面有一口老井，这井在李氏可有些年头了，一直可以追溯到李氏在陇西立足的那个年代了。<br/>　　<br/>　　    李家人都知道，陇西李氏大宅风水甚佳，但少有人知道的的是那口早在几百年前就干枯了的老井，是这处风水大宅的阵眼。<br/>　　<br/>　　    李兰荣惊慌不已，胸口起伏不定。<br/>　　<br/>　　    崔玄策幽幽的说道：“风水坏了······”
