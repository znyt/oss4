　　“先遣大队的事情就是他这个搅屎棍，不然的话，陆勇和他的三千部下早就死了。结果到头来，我们死了一堆人，还有十几个异能者！！”
<br/>　　几个长老恶狠狠道。
<br/>　　“莫非，阎王大人您的意思，这一次据点也是这小子背后做的？”
<br/>　　几个长老眯着眼道。
<br/>　　“没错，就是他。”
<br/>　　黑衣人点了点头。
<br/>　　“可恶。”
<br/>　　几个长老气的都是锤了下桌子。

<br/>　　“那又如何，这家伙杀了我战门这么多成员，我们肯定要杀了他。”
<br/>　　“没错，之前就是他在背后捣鼓，让我们和尸傀门和血尸堂误会彼此，甚至大动干戈，都是这小子在背后推波助澜做的。”
<br/>　　黑衣人忽然哈哈大笑，一群战门外门长老都有些恼怒的看着他。
<br/>　　“事到如今了，还在这里说这些没用的屁话，我要是你们，就不要墨迹了，直接放出你们的所有底牌，趁这个林峰趁曙光基地还没完成成为气候之前，把他给摧毁了，不然，你们战门纵然躲在这个好地方，也必定要被曙光基地给连根拔起。”
<br/>　　黑衣人冷笑道。
<br/>　　“你的意思是。”
<br/>　　几个外门长老目光微微眯起。
<br/>　　“发动兽群和尸群，进攻曙光基地？”
<br/>　　“怎么、舍不得了？放心，这一次，我第一邪教也会帮助你们，就是我，也会亲自出手，到那时，这林峰若是敢插手，那么我来亲自解决他。”
<br/>　　黑衣人背负双手道。
<br/>　　“这是个好主意！不过事关重大，我们几个还要仔细商榷一下！”
<br/>　　“你们慢慢商量吧，机不可失失不再来。”
<br/>　　黑衣人说完就消失不见变成了一团黑烟。
<br/>　　“怎么办。”
<br/>　　几个外门长老面面相觑。
<br/>　　“这是个好机会，既然第一邪教的人也愿意出手，那么我们还是把这件事情告诉上面，让上面的人来做定夺。”
<br/>　　转眼。
<br/>　　七天度过。
<br/>　　林峰一直呆在老龙口内。
<br/>　　矿山每天都在拉矿出来，不过效率不高。
<br/>　　而林峰每一天都捡空投。
<br/>　　不知不觉又存了十五个C级空投。
<br/>　　这十五个C级空投。
<br/>　　林峰又开除了许多强大火力。
<br/>　　三挺重机枪外加两门防空炮一门防空机枪。
<br/>　　这个防空炮来的非常是时候，就在昨天，他们听说，军部有个据点被一群变异鸟给突袭了，虽然据点守住了，但是死伤惨重，就是因为没有布置战防炮。
<br/>　　防空炮和防空机枪的出现，直接增加了老龙口的防空保卫力量。
<br/>　　除此之外，则是一辆军用悍马外加五辆军用运输车。
<br/>　　这些车出现的非常的及时，可以从基地内拉物资拉人力，都绰绰有余，不需要林峰每次都再去找陆勇开口借了，毕竟这可都是要花人情的。
<br/>　　至于军用悍马就成为了林峰的私人座驾。再然后就是一些弹药补给以及一桶汽油，短时间内几辆车都不再担心油耗问题。
<br/>　　另外，林峰还用几车煤矿，和陆勇换了五十把95步枪和若干子弹。

<br/>　　老龙口的安保力量上去了，林峰也算是暂且放下心了，他现在就可以专注一下子自己的问题了，他的天上地下唯我独尊八荒六合第三重到现在都没有开到。
<br/>　　眼看这十五个C级空投又已经只剩下最后两个了。
<br/>　　“尼玛的，这么难开？我就不信邪了。”
<br/>　　林峰立即开启。
<br/>　　咔擦。
<br/>　　开启倒数第二个空投，里面赫然是一本书。
<br/>　　“握草！爆了！终于爆了吗？”
<br/>　　林峰顿时兴奋了。
<br/>　　不过再一看，根本不是天上地下唯我独尊八荒六合。
<br/>　　因为上面的字太短了。不过才五个字罢了。
<br/>　　《百鸟朝凤枪》

<br/>　　林峰倒也颇为熟悉，好像有这么一回事。赵云那可是人尽皆知的，帅气潇洒，而且还特厉害，子龙一身是胆。
