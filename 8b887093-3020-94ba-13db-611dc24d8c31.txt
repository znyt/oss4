　　火龙长老是谁？
<br/>　　恐怕整个云丹城，乃至整个云丹星域，都没有多少人不知道。
<br/>　　那可是云丹宗十大长老之一！
<br/>　　地位仅在云丹宗宗主，以及两大太上长老之下。
<br/>　　不知道有多少人，都梦寐以求，想要成为他的门下弟子。
<br/>　　可是从数十年前，他便已经不再收徒，即便有人想要拜入他的门下，也都是让徒弟代收。
<br/>　　原因很简单，达到他这种级别和地位，除非遇到那种天资绝顶的好苗子，不然的话，根本就不会考虑。
<br/>　　而如今，已经几十年不曾收徒的火龙长老，居然当面主动提出收徒，这让众人无比震惊的同时，也对叶尘感到分外的羡慕、嫉妒。
<br/>　　要知道，一旦成为十大长老的亲传弟子，不但可以直接进入，云丹宗核心弟子之列，而且辈分也远在那些同龄人之上，甚至可以和屠苏公子、蝉衣小姐等一众天才，去竞争下一任的云丹宗宗主之位！
<br/>　　这可是真正的一步登天呐！
<br/>　　就连那屠苏公子，听到火龙长老的这个决定，也顿时脸色一阵阴沉，
<br/>　　曾几何时，他一度是云丹宗最为耀眼的天才，甚至是下任云丹宗宗主的不二人选！
<br/>　　然而好景不长，半路里突然杀出一个冷蝉衣，将他的风头彻底抢走，取代了他，成为了宗门第一天才。
<br/>　　如今，冷蝉衣还没有搞定，又突然冒出了一个叶尘。
<br/>　　尽管在他看来，眼前这个少年，还远远不足以威胁到他，不过一想到核心弟子中，又多了一个竞争对手，屠苏公子心里，就十分的不爽。
<br/>　　然而就在这时，
<br/>　　叶尘开口了，
<br/>　　“不好意思，我拒绝！”
<br/>　　轰！
<br/>　　叶尘这话一出口，整个考核大殿的气氛，一下子变得无比诡异起来，甚至所有人都怀疑自己，是不是出现了幻觉？
<br/>　　竟然有人拒绝成为火龙长老的弟子？
<br/>　　这……这世界是怎么了？
<br/>　　难道疯了不成？
<br/>　　先不说，成为火龙长老的亲传弟子，足可以一步登天，单单是‘拒绝’两个字，恐怕就没人能够承受的起！
<br/>　　得罪了十大长老，你还想不想在云丹城混了？
<br/>　　“什么？”
<br/>　　原本心情已经不爽到极点的屠苏公子，也忽然为之一愣。
<br/>　　因为他从来想过，叶尘会拒绝，即便是他和对方易地而处，也肯定会毫不犹豫的答应啊！
<br/>　　不过在惊愕、不解、诧异之后，屠苏公子彻底反应过来，心里顿时感觉说不出的畅快，
<br/>　　“哈哈哈！这个傻子，他居然拒绝了！哈哈，真是老天都在帮我啊！”
<br/>　　一旁的冷蝉衣，同样秀眉紧蹙，美眸不由得在叶尘身上，深深的打量了一眼，脸上露出一抹疑惑之色。
<br/>　　至于台下的众人，在反应过来之后，也顿时一下子炸开了锅，
<br/>　　“火龙长老主动提出要收他为弟子，这家伙居然拒绝了？”
<br/>　　“天哪！难道这家伙不知道，眼前的这位火龙长老，是何等人物吗？”
<br/>　　“不知道多少天才人物，做梦都想成为十大长老的门下弟子，这小子莫非是得了失心疯，居然敢拒绝火龙长老！”
<br/>　　“疯了！疯了！简直就是疯了！！”
<br/>　　……
<br/>　　一时间，众人望向叶尘的目光，像是在看傻子一般。
<br/>　　很多时候，一个人的命运，往往都在一念之间！
<br/>　　原本，一个可以一步登天的机会摆在面前，这家伙竟然不知道珍惜，反而自寻死路。
<br/>　　火龙长老此时，同样是脸色铁青，甚至眼眸深处，隐隐透着一抹杀意！
<br/>　　他位列云丹宗的十大长老，这一生，不知道有多少人哭着求着，想要拜他为师，他连看都不看那些人一眼。
<br/>　　而如今，他主动收徒，居然被这小子给拒绝了！
<br/>　　如果不是当着这么多人的面，要顾忌自己的形象问题，他此刻肯定直接一巴掌，将这个不知好歹的小子，活生生拍死了。
<br/>　　当然了，在他眼中，对方不过就是一个金丹境修为的小子，根本挡不住他随手一击。
<br/>　　倘若让他知道，光是死在叶尘手里的化神天君，已经有三位之多，恐怕也就不敢说出之前那样的话了。
<br/>　　“你确定要拒绝？老夫念在你年轻不懂事，再给你一次机会，顺便提醒你一句，老夫的道号为火龙真人，在云丹宗十大长老中，排名第五！”
<br/>　　“现在，你还要拒绝吗？”
<br/>　　轰！
<br/>　　火龙长老说完这话的同时，身上陡然爆发出一股强悍的气息。
<br/>　　蹬蹬蹬！
<br/>　　台下一些离得近的，顿时感觉到一股，强大到令他们心悸的气息，不由得脸色煞白，连连倒退。
<br/>　　然而，
<br/>　　首当其冲的叶尘，却昂然站在那里，纹丝未动，甚至脸色的表情，依旧淡然无比，仿佛丝毫没有察觉到，火龙真人的气势压迫，
<br/>　　“我说了，我拒绝！”
<br/>　　叶尘本想说，就凭你，还想做我的师父，你配吗？
<br/>　　不过他也清楚，如果他真的那样说了，必然会彻底激怒火龙真人，甚至会与整个云丹宗为敌，最终将会不好收场。
<br/>　　对于云丹宗，因为前世的关系，叶尘还是有些感情的，所以不到万不得已，他不想走到那一步。
<br/>　　轰！
<br/>　　台下众人听到叶尘这话，顿时都快要晕过去了，
<br/>　　这家伙，居然再次拒绝了！
<br/>　　他们见过胆子大的，却没有见过胆子如此大的！
<br/>　　一而再，再而三的拒绝，你真以为云丹宗十大长老，是这么好得罪的吗？
<br/>　　“好好好！”
<br/>　　火龙真人死死盯着叶尘半响之后，忽然怒极反笑，不过眼眸深处，却透着一抹惊疑之色，
<br/>　　在他刚刚的气势压迫之下，对方竟然还能纹丝不动，单是这份定力，就不是一般人能够做到的，
<br/>　　‘难道这小子，是笃定了我不会杀他？所以才如此有恃无恐？’
<br/>　　一念及此，火龙真人缓缓收起身上的气势，脸上重新露出一抹温和的笑容，道：
<br/>　　“好小子！能够在老夫的威压下面不改色，你也算是头一个，既然你不愿意拜老夫为师，那老夫也不强求！你以后若是遇到什么问题，还是可以来火龙居找我！”
<br/>　　说完这话，火龙真人直接转身返回自己的座位之上。
<br/>　　不过，
<br/>　　就在火龙真人转身的那一刹那，眼眸中闪过一抹浓浓的杀意。
