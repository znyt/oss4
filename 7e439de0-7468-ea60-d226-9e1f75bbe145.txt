        &lt;h3&gt;第三卷第一百六十七章  喜忧参半的陈子乔&lt;/h3&gt;
<br/>
<br/>        冯睿在年前给陈子乔打电话，约陈子乔一起去探望陈子澜，虽然和陈子建离婚了，但是他们之间又没有多大的仇恨。
<br/>
<br/>        陈子乔这几年真的是忙得一塌糊涂，调到西城，那次陈子澜提出忠告之后，陈子乔收敛了很多，想一想，其实她真的是不缺什么，再加上这几年的各种周旋让她多少生出了一些厌恶。
<br/>
<br/>        陆总将她和马总调过来之后就走了。
<br/>
<br/>        如今她也没有必要周旋在他们中间，对于马总的那些事情她心知肚明，慢慢的她也就淡出来了。
<br/>
<br/>        马总的圈子比以前更大了，什么样的女人都会有。
<br/>
<br/>        走到现在这个地步，她明白了一个道理，虽然说她掌握着很多的资金，但是并不是她说了算，是别人说了算，她只是一个工具而已。
<br/>
<br/>        常啸林的产业还在洛城，他并没有因为妻子到了西城而将他的大本营也迁到西城，他在洛城的项目都是蒸蒸日上，尤其是多了一项物流，别小看这个事情，他是拿下了整个洛城的一级代理，一年下来也赚不少钱，而且这个钱赚得放心，又不求爹爹告奶奶的，只要将货物安全的送达到客户的手中。
<br/>
<br/>        他是西城和洛城两头跑，孩子也随着老婆去了西城，西城的教育比洛城要好很多，再加上老婆成天的唠叨俩个老人将孩子带得不成样子了，要不早点掰过来怕是以后会养歪。
<br/>
<br/>        在常啸林的意识里，老婆说的话就是圣，旨。他在西城买了三百多平米的房子，雇了保姆和家庭教师严格按照老婆的要求来教育俩个孩子，好的是，孩子本质不坏也还聪明着，努力努力学习成绩也上去了，教育环境不一样，孩子懂礼貌了很多。
<br/>
<br/>        今天常啸林开着车到了西城，恰好陈子乔没有应酬，一家人难得的坐在一起吃顿饭，俩个孩子也显得乖巧懂事。
<br/>
<br/>        “老婆，真是辛苦你了，又要上班又要带孩子。”常啸林知道老婆工作繁忙，心疼的说。
<br/>
<br/>        “你不是也很辛苦吗？我们一家大小的吃喝用度都是你的。”陈子乔很温柔，她和常啸林孩子都有了，看着三姐夫对三姐那么好，她的心里突然就想明白了一个道理，外面的世界再怎么热闹，都抵不过家里的一份安宁。
<br/>
<br/>        常啸林也不是一无是处，人除了憨厚一点外，倒是能做生意，虽然是仗着他表哥的关系，但是他对人却很不错。
<br/>
<br/>        陈子乔对现在的要求就是一家人平平安安的生活，虽然平淡但是也很满足。
<br/>
<br/>        上一次回家，看着一家人在一起真的很好，除了老六不在，其余的人都过得不错，而且二哥做的事情无形之中给她上了一课，也给她了一个警醒，再加上三姐的一席话，让她突然就醒悟了，一家人开开心心的在一起比什么都重要。
<br/>
<br/>        最近三姐躺在床上，她也经常去看望三姐，三姐就经常教育她，不是自己的亲姐妹谁管你？姊妹之间什么话都能说，她也将这几年心里不能说的话都说给三姐听了。
<br/>
<br/>        三姐对她说：“我们老陈家的孩子都是清清白白，干干净净的，父亲也教育过我们，你之前的事情就不要提了，以后一定要好好做人，向二哥学习。”
<br/>
<br/>        “啸林，你工作上有没有请客送礼？”陈子乔想起三姐的叮咛。
<br/>
<br/>        “你傻啊，请客肯定是有的，送礼就算了，仗着表哥谁还敢收我的礼？”常啸林哈哈的笑着。
<br/>
<br/>        “那你有没有给表哥送过？”陈子乔还是很担心。
<br/>
<br/>        “表哥好意思拿我们家的东西？不过这么几年，表哥就是要了一套凤凰苑的房子。”
<br/>
<br/>        陈子乔听说凤凰苑的房子心里一惊。
<br/>
<br/>        “没有给别的就好。三姐最近经常唠叨我过得太奢侈了。”
<br/>
<br/>        “我们自己挣的，既没有偷也没有抢，你怕什么？”常啸林不以为然。
<br/>
<br/>        “可是我之前也有拿过人家的包包啊什么的。”陈子乔在那几年确实拿过别人送的东西，花天酒地的日子也过的不少，要不是最近三姐经常唠叨她，她还没有认识到错误。
<br/>
<br/>        “你要是觉得不安心了退给他们就是，咱现在也不缺那么点钱。我做生意虽然有表哥的面子，但是我每一件事情都做得好好的，不吭不骗。”常啸林这才明白妻子的担忧。
<br/>
<br/>        他赚了多少钱按说陈子乔都知道，只是陈子乔只忙着工作上的事情，对他生意上的事情都不闻不问，他给了陈子乔一张卡，陈子乔想买什么就买什么，也不知道卡里有多少钱。
<br/>
<br/>        第一次去见陈子乔，陈子乔还是个小姑娘，工作认真，长得好看，他一看就喜欢上了，于是就缠着陈子乔要贷款，看着陈子乔穿的衣服也不怎么好，他就偷偷的塞了一点钱在母亲给陈子乔的鸡蛋里头，陈子乔要退给他，他装作不知道，后来也经常给陈子乔一些钱，一来二去就熟悉了，刚开始他的生意也不好，资金周转不开，经常找陈子乔贷款，也帮朋友贷款，每次都会给陈子乔拿点东西，拿的都是他自己的。
<br/>
<br/>        他求着表哥带着陈子乔一起出来吃饭，对陈子乔展开追求，再加上表哥的撮合，他终于将陈子乔娶回来了。
<br/>
<br/>        陈子乔一直是冷冷淡淡的性子，到西城之后反而温柔了很多，对家里也多了份关心，让常啸林开心不已，基本两天就开车跑一趟西城。想一想他这么努力其实都是为了陈子乔，看着陈子乔打扮的漂漂亮亮的开着他买的豪车他心中就升起一种自豪感。
<br/>
<br/>        “我也拿得不多，不都是经过你的手吗？你想一想都是谁给的，我都记不住了。”陈子乔忧虑的说，在洛城的那几年，基本都是马总和常啸林的朋友，只要有人找她办事，常啸林就送给她东西，说是别人感谢的。
<br/>
<br/>        “你傻呀，我经什么手？我送给你的礼物都是我们自己的钱买的。你从来都不关心我，也不知道我有多少钱。”常啸林幽怨的说。
<br/>
<br/>        “真的？”陈子乔欣喜的问。
<br/>
<br/>        “当然是真的。”常啸林真是很难过，为什么他老婆不看看他的能力？
<br/>
<br/>        “哎呀，那我就放心了，你不知道三姐说得我这心里啊，简直就是七上八下，日夜不眠。”陈子乔舒了口气，但是陈子乔还是不关注常啸林到底生意有多大，能赚多少钱。
<br/>
<br/>        “子乔.....”当常啸林正准备要给陈子乔讲他生意的时候电话响起来了。
<br/>
<br/>        陈子乔看了一眼电话，犹豫了一下，接起电话：
<br/>
<br/>        “二嫂，你好。”客气而疏离，冯睿和陈子建离婚了，但是冯睿毕竟跟了陈子建那么多年。
<br/>
<br/>        “子乔啊，最近都好吧？”冯睿热情的打着招呼。
<br/>
<br/>        “都好呢，二嫂有什么事情吗？”冯睿基本不给他们打电话，永远都是一副高高在上充满优越的感觉。
<br/>
<br/>        “我前两天听说子澜出车祸了，我想去看看，你能不能陪着我去啊。”
<br/>
<br/>        陈子乔觉得冯睿一定是有意的，三姐出车祸这么久了，她才想起来去看？想要去看，自己就去了，为什么要拖上她？
<br/>
<br/>        “也好，我这两天也没有见三姐了，那我们一起去看看。”陈子乔这会心情还不错，就同意和冯睿一起去看看陈子澜。
