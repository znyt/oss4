　　叶少阳看着他，“我只问你一句，如果被他侮辱的是你妹妹，或者……你身边这位姑娘，你也由他这么做？”
<br/>　　“你放肆！”曹雨兴气得跳起来，妙心也是冷冷地瞪着叶少阳，“不要拿我做比喻！”
<br/>　　叶少阳耸了耸肩，道：“本来就是一个道理，被伤害的不是自己或身边人，总觉得没啥。”
<br/>　　曹雨兴冷笑起来，讽刺道：“你可真是侠义丹心，路见不平啊。”
<br/>　　叶少阳道：“不敢当啊，法师而已，遇到这种事就忍不住想管。”
<br/>　　妙心道：“你可知道，没有了三足金蟾，整个大阵就不完整了，如果古墓里的东西出来，要荼毒多少人？岂是侮辱几个姑娘的事？”
<br/>　　“那不知道，兵来将挡，我们法师干的就是那个。眼看着有人被邪物欺负我都保护不了，说什么保护更多人，我还是那句话，除非你自己愿意牺牲让三足金蟾玩三年，不然别跟我说什么大道理，如果你自己都不愿意，你凭什么让别人牺牲？”
<br/>　　毛小方拍掌说道：“讲得好！”
<br/>　　妙心和曹雨兴气的脸色铁青，但又不好反驳。
<br/>　　叶少阳心里其实也是挺憋闷的，之前，在自己那个时代，自己不管是在人间还是阴司，都有相当的地位，不管干什么都是一路绿灯，当时也没觉得什么，现在遇到眼前的处境，叶少阳真的有点怀念自己之前的待遇了，至少做什么随心所欲，不用跟别人交代，也没人会把他怎么样，哪像现在，突然蹦出来两个人都可以嘲讽自己……
<br/>　　再加上自己现在只有真人的实力……这次真的是变成了法术界的小透明。
<br/>　　对叶少阳来说，他从未体会过这种感觉。
<br/>　　在法术界，茅山的地位就相当于大学里的清华北大，想象一下，如果北大只有一个学生，这个学生得是什么样的地位。
<br/>　　叶少阳就是这样一根独苗，尤其是在道风被逐出师门之后，就算那时候实力不太行，在法术界那也是高高在上的地位，就算凌宇轩那样的海归看不上他，但也指名道姓要把他踩下去，就是因为他的地位。
<br/>　　人间第一天师，茅山掌教，阴阳司之主，捉鬼联盟的首领……
<br/>　　这些闪耀的名头，一个都没有了，自己现在没名没分，连实力都比不上人家。之前来到这个时代，没怎么接触法术界这些人，也没被人怼过，现在遇到这个两个人，这些感觉一下子就涌了上来。
<br/>　　好在叶少阳本不是容易患得患失的人，自我调整了一下，对妙心说道：“再说，古墓里出现情况，也不是我们的责任，三足金蟾自己也是那是之前的事，当时他已经通知你了，不然你也不会来，关于我们发落三足金蟾的事，毕竟过去了，现在我们还是联手处理古墓的事，我在镇口留下法结，也就是这个意思。”
<br/>　　“联手……呵呵。”曹雨兴轻蔑地笑了两声。
<br/>　　妙心没有嘲讽，不过也是轻轻摇头，说道：“你们以为古墓里那个东西，这么好对付？”
<br/>　　“炼尸缸嘛，不好对付我知道，所以才想着找你联手。”
<br/>　　“你怎么知道是炼尸缸？”妙心问完，立刻想到可能是三足金蟾跟他们说的，于是说道：“关于这座古墓，你们知道多少？”
<br/>　　“基本上什么都不知道，只知道那蛤蟆谷有个洞，好像能下去，别的什么都不知道，这不等你来告诉我们呢。”
<br/>　　妙心分别看了他和毛小方一眼，道：“古墓下面的东西，并非你们能够对付，可能会死人，这本是我家事，跟你们无关，你们可以走了。”
<br/>　　叶少阳无奈一笑，道：“说白了，你还是瞧不上我们。”
<br/>　　妙心道：“我是为你们安全着想，不要以为天师很了不起，在炼尸缸面前，天师根本没用。”
<br/>　　叶少阳笑道：“那你是什么牌位？”
<br/>　　曹雨兴挑着眉毛说道：“妙心姑娘是紫狂真人嫡传，地师传人，当年封印炼尸缸就是紫狂真人所为，就算出了问题，她也知道怎么禳补，要你们何用？”
<br/>　　叶少阳道：“那要你何用？”
<br/>　　“我……”曹雨兴挑了挑眉毛，“我是昆仑弟子，八年前就晋升天师，如今只差一个契机就能晋升地仙，岂能跟你们一样？”
<br/>　　叶少阳道：“哦，也就是说，你都当了八年的天师，还没晋升地仙，这个很光荣？”
<br/>　　叶少阳说这一席话本来是想嘲讽曹雨兴，结果曹雨兴哑然失笑，“除了张晓寒和几个逆天的家伙，你见过有人八年时间从天师晋升地仙的吗？”
<br/>　　叶少阳道：“当然见过。”
<br/>　　“哦？那你说来听听？”
<br/>　　“张晓寒用了几年？”
<br/>　　曹雨兴笑了一下，有点懒得回答，不过为了刺激他，还是说道：“最多五年，天下绝顶。”
<br/>　　“五年就天下绝顶了？”叶少阳哑然失笑。
<br/>　　“难不成还有更短的？”
<br/>　　叶少阳没有再说下去，因为说了他肯定也不信，五年时间，从天师到地仙，绝对算是非常短的时间，甚至可以说是奇迹，不过……跟自己比还是弱爆了，自己用了三年，说起来还是坚持寻找道之所在，不愿意晋升的缘故。另外一个变态道风……当初只用了一年。
<br/>　　妙心对这话题不感兴趣，道：“你们不要扯题外话，我让你们走，你们就快走，不要再掺和这件事，到时候小命丢了，后悔莫及。”
<br/>　　“我不怕，你如果不想合作，我们就自己去，总之这件事让我遇到了，我肯定不会不管。”叶少阳说完，冲她点了一下头，做出一副毅然的表情。
<br/>　　妙心盯着他看了一会，从他的眼神中，看到了一种她理解不透的东西。信念？
<br/>　　“姑且一起吧，不过，你们一切要听我安排，不能擅自行动，不然死了我可不管。”
<br/>　　“OKOK，一切跟组织，听你指挥。”叶少阳道。
<br/>　　“你会英语？”妙心倒是一惊。
<br/>　　英语？一句“OK”就是英语了？
<br/>　　叶少阳愣了一下明白了，这个时代还很闭塞，英文没普及，就算是“OK”这个最普通的词，在这个时代也没什么人知道。
