　　“杀！”
<br/>　　准提圣人尽管心中愤怒异常，但他并没有被愤怒冲昏了头，反而是更加冷静，他明显能够感受到，万劫魔尊的境界好像比他高一层。
<br/>　　这怎么可能？要知道，他如今可是圣人一重天，而若是万劫魔尊真的比他高一层，那不就是混元二重天吗？
<br/>　　这怎么可能？要知道，他准提已经突破圣人数十万年了，但是如今还只是被困在圣人一重天，而万劫魔尊刚刚突破，却就直接跨过一重天，成为二重天，这就十分惊人了，让人难以置信。
<br/>　　准提圣人内心沉重，不敢大意，眼中闪过奇异的光芒，他一步踏出，穿过层层虚空，来到万劫魔尊眼前。
<br/>　　压下心中的沉重，准提圣人双眼中那股奇异的波动猛然增加数万倍，像是两个黑洞，要将所有的东西都吸了进去。
<br/>　　轰！
<br/>　　万劫魔尊只感觉自己的灵魂一阵颤动，像是要脱离自己而去，要被那黑洞吞食进去。他瞬间便想到，准提圣人的梦中证道之法，眼神一阵闪烁，心中便有了思量。他放弃抵抗，只感觉一个恍惚，自己就已经出现在一个奇异的空间内。
<br/>　　在这个空间里，无限大，金碧辉煌，各种殿堂，鳞次栉比，随处可见，各种庙宇，里面有着许多和尚，他们全部身穿袈裟，默念经文，准提圣人化为一尊大佛，高坐云台，在他身上，佛光万丈，梵音缭绕，像是在歌颂，又像是在赞美……
<br/>　　这就是一个伟大的佛国。
<br/>　　而像这样的佛国，在这片空间之中，不知道还有多少。
<br/>　　万劫魔尊出现在一座宏大的殿堂之中，他的神识触碰上去，也只是若有若无，大殿上方最高处，准提圣人高坐于上，在下方，站着各种菩萨、罗汉、护法等和尚，他们一瞬间便将目光投向万劫魔尊。
<br/>　　没有恐惧，万劫魔尊一边感受着自己的实力的存在，一边打量着这奇异的佛国空间。
<br/>　　这应该就是梦中证道之法了，能够将人的意识拉入一片奇异的空间，还能带着本身的法力，进行精神与法力的对决。
<br/>　　在这里，他虽然能够发挥出自己的实力，但是却要受到很大的限制，谁来都会如此，因为此处不是自己的场域，此处空间，只受准提主宰。
<br/>　　这样一来，在这里，几乎没有人会是他的对手。
<br/>　　“大胆魔头，见到我佛，为何不跪？跪下！”
<br/>　　一道空洞却激荡灵魂的声音在万劫魔尊脑海响起，同时，一股看似虚无却又缥缈的神秘力量，渗透过来，要强行使万劫魔尊跪下。
<br/>　　“放下屠刀，立地成佛，证我道者，入我佛门，皈依我佛，跪下……皈依……”
<br/>　　“放下屠刀，立地成佛，证我道者，如我佛门，皈依我佛，跪下……皈依……”
<br/>　　……
<br/>　　声音连绵不绝，带着浓浓的佛门梵音，在大殿之中回响不断，空洞，却又每一声都击在听着的灵魂深处。
<br/>　　一股来自灵魂深处的呼唤传到万劫魔尊意识之中，一股虚无缥缈的神秘力量在侵蚀着万劫魔尊的法力，在侵蚀着他的意识，在不断呼唤这万劫魔尊匍匐下来，祈祷祭拜，甚至要跪下，献出生命，只有这样，才能证得大道，成佛做祖。
<br/>　　万劫魔尊心中升起一股强烈的虔诚之感，甚至就要跪地膜拜。他表面平静，内心深处却如同煮沸了的油一般。这种恐怖的力量轰入，想要度化他的内心，换做是修为和心志稍微弱点的人，也许早就匍匐在地，皈依佛门了。
<br/>　　万劫魔尊的心志何等坚定，身为魔尊，修炼的又是劫气，天天与劫气熬炼，早已经将他的意识磨练的坚硬如铁，不可撼动。再加上他修炼的劫气，又正好与这神秘的力量有些相克，一时之间，就算那股神秘的度化之力，也无法奈何得了他。
<br/>　　“我为魔尊，天生就是你们这些秃子的对头，皈依佛门，真是笑话！我看，尔等还不如都入我魔门好了！哈哈哈……”
<br/>　　万劫魔尊面无表情，一声冷笑，接着哈哈大笑，声音嚣张，传遍大殿，久久不散。
<br/>　　“放肆，大胆魔头，竟敢冲撞我佛，给我拿下！”
<br/>　　见声音蛊惑不了万劫魔尊，大殿中的众多高僧便纷纷出手，想要用武力征服，使他屈服。
<br/>　　“哼！以多欺少，谁怕谁啊？”
<br/>　　万劫魔尊冷哼一声，紧接着，大手一挥，身边滔天魔气汩汩而出，纵横大殿之中，将金碧辉煌的大殿瞬间变得乌烟瘴气，魔气冲天。随后，这股滔天魔气瞬间便化为种种劫魔，飞天纵地，与大殿中的僧人大战在一起。
<br/>　　兵对兵，将对将！
<br/>　　万劫魔尊却将目光投向高台之上的准提圣人，冷笑道：
<br/>　　“圣人还不出手吗？”
<br/>　　“善哉，既然魔尊已经准备好了，那么贫道也不客气了。”
<br/>　　高台之上，传来准提圣人的声音，声音落下，那一直紧闭着双眼的准提圣人也在此刻缓缓睁开眼睛，莫名的威压降临，他那双眼中射出无比璀璨的光芒，整个空间忽然颤动了起来。
<br/>　　这一刻，准提圣人就仿佛是天地的主宰，能够轻易操控整片空间。
<br/>　　那漫天佛光光芒越来越盛，到最后化为一道巨大的“卍”字佛印，旋转之中，一股浩瀚广博的天威，顿时向着万劫魔尊封印镇压下来。
<br/>　　“南无准提佛母，佛法无边！”
<br/>　　与此同时，在这诡异的空间之中，四面八方，居然传来无数颂佛之音，竟然有无数佛子佛孙，供奉佛祖。
<br/>　　“轰隆隆！”
<br/>　　无数信仰之力汇聚而来，融入到那巨大的“卍”符文之中，使得威力大涨，佛光万丈。
<br/>　　“哼！准提圣人，不愧为圣人，竟然创出如此梦中大法，不过，你的法力也不能无限制的增长下去，若是超过了肉体的承受能力，你的肉体便会随之爆炸，到时你的念头便成了无根之浮萍，不能长存。而且，你还真是异想天开，居然在梦中幻化出亿万佛国，国中子民，皆为佛徒，为你提供无数信仰。”
<br/>　　“不过，这些信仰都是虚幻的，只有在梦中，才有用，呵，你认为仅凭这些，就能镇压本尊，真是可笑！”
<br/>　　万劫魔尊冷笑一声，满脸不屑。
<br/>　　“虚虚实实，真真假假！又有何区别呢，这里，对于你我来说，是虚幻的，但对于这里的无数佛子、佛孙们来说，他们是真实的，而你却是虚假的。他们在这梦幻空间中，繁衍生息，认为自己是真实的。至少，在这里我能够用这些信仰之力镇压你。”
<br/>　　“笑话！若是你真的达到化虚为实的地步，也许真的能够镇压本尊，不是本尊看不起你，就凭你这圣人一重天的修为，还远远达不到那种境界？想要镇压本尊，你就连梦中都休想成功！”
<br/>　　万劫魔尊一声冷笑，手中魔刀斜劈，撕裂空间，扭曲时空，斩破那道“卍”封印。同时，他自己亦继续向着准提杀去。
<br/>　　……
