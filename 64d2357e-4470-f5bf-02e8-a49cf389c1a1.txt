　　楚风眠感觉到黑玄剑阵的力量，又进一步，心中不由的感叹一声。
<br/>　　这黑玄剑阵，每一把灵剑，都是其中一个阵眼。
<br/>　　阵眼越强，组成剑阵的威力也就会越发强大。
<br/>　　最好十万灵剑，都拥有器灵，倒时候一旦激发出来，也就蕴含有着十万妖兽的力量。
<br/>　　不过想要凑齐十万器灵，简直是千难万难的事。
<br/>　　“走吧！”
<br/>　　楚风眠身躯一动，再次向着山谷深处走去，如今这山谷，楚风眠已经是通过的大半。
<br/>　　隐约之间，楚风眠甚至都已经感觉到，快要离开这山谷了。
<br/>　　不过等到楚风眠走到一半的时候，他便是注意到了一个问题，那就是在他的身后，似乎已经是没有了其他武者的存在。
<br/>　　隐约之间，所有武者，似乎都是偏离了不同的路线。
<br/>　　虽然每个人，都是向着山谷深处走去，但是隐约之间都已经是前去了不同的路线。
<br/>　　不止是这山谷，这一处空间，都是极为诡异，不知道为何会变化成这样。
<br/>　　这山谷，虽然只有一座，但是蕴含不同空间，现在的楚风眠，便是在其中一处空间之中。
<br/>　　也许每个人在其中，所遭遇的，都不一样。
<br/>　　楚风眠一边观察这四周，一边继续向着前方走去。
<br/>　　而就在刚刚，楚风眠斩杀那头猛虎的地方，突然一道大门开启，一名中年男子，从中缓缓的走了出来，看到那猛虎的尸体，顿时脸色大怒。
<br/>　　“谁？是谁敢杀我虎儿！”
<br/>　　这中年男子愤怒的大吼道。
<br/>　　“我要将你，碎尸万段！”
<br/>　　轰！
<br/>　　突然之间，这中年男子一身灵力爆发出出来，在他的面前，突然出现了一道水镜。
<br/>　　在其中映照的，正是楚风眠的身影。
<br/>　　“死！”
<br/>　　只见这中年男子指尖一点，直接向着楚风眠一股灵力轰击出去。
<br/>　　而在另一边，楚风眠瞬息之间，也是感觉到了危险。
<br/>　　一股灵力，突然从天而降，正是向着他的头顶，轰击过去。
<br/>　　“什么？在这山谷中，还有其他人的存在？这猛虎难道还有主人？”
<br/>　　楚风眠眼神露出几分忌惮，一身灵力，全部爆发出来，黑玄剑阵出现在他的头顶上，跟着一股灵力，直接交锋起来。
<br/>　　黑玄剑阵，跟着一股灵力厮杀一阵，这灵力便是不支，直接退散。
<br/>　　这一股灵力，毕竟是破空而来，虽然凶猛，但是并不浑厚，只是这样，楚风眠的心中也是震惊无比。
<br/>　　因为他感觉到，这突然出手的人，绝不会是归墟天中的人。
<br/>　　此人能够在山谷中，破空出手，一身实力只怕是比起什么剑苍穹，小魔主等人，都要强大的多。
<br/>　　在这一届进入归墟天的武者中，楚风眠还从来没有听说过这样一个人的存在。
<br/>　　“看来这山谷中，果然还有隐秘，不止是有着妖兽，如今就连武者都出现了。”
<br/>　　楚风眠心中震惊一下，但是很快便是平复了他的心情。
<br/>　　这出手的人，实力虽然强大无比，但是也无法杀的了楚风眠，起码对楚风眠，没有太大的威胁。
<br/>　　楚风眠所忌惮的，是这山谷中，既然会有一个武者，还会不会有第二个？第三个？
<br/>　　起码这些人的实力，比起这一次进入归墟天的武者，都要强大的多。
<br/>　　对楚风眠都是有着极大的威胁。
<br/>　　“给我破！”
<br/>　　楚风眠想到这里，当机立断，剑锋突然向着空中斩去。
<br/>　　只见在空中的隧道，被直接搅碎，随后楚风眠脚下一动，直接离开。
<br/>　　“哦？”
<br/>　　一击不成，那中年男子的脸上似乎都有些惊讶。
<br/>　　“这一届进入归墟天的武者中，居然还有如此人物，不过他敢杀我虎儿，让我在见到他，我一定要杀了他！”
<br/>　　中年男子默念一声，捡起那猛虎的尸体，直接再度走入到了大门之中去，这空间大门，逐渐关闭。
<br/>　　“归墟天中，果然还有着其他武者的存在，看来历届之中的武者，并非是所有没有离开归墟天的武者，都是死在了其中，应该也有一些人，正是从其中活了下来。”
<br/>　　楚风眠感觉到那中年男子放弃，也是松了一口气。
<br/>　　毕竟他在明，敌在暗，对着山谷，楚风眠都还是极为陌生。
<br/>　　这个时候贸然交手，对楚风眠不利，不过那中年男子的身份，楚风眠大约也已经是猜了出来。
<br/>　　这中年男子，很有可能便是曾经上一届，甚至是更早之前，进入到归墟天的武者。
<br/>　　每一届归墟天开启，能够从其中离开的武者，都不足三成。
<br/>　　至于其中的那七成，按照说法，都是死在了归墟天中，毕竟归墟天中，危险重重。
<br/>　　陨落在其中，也并非是不可能。
<br/>　　不过也有许多天才，在进入到归墟天中，便是没有出去，这些人，意外陨落的可能性很低。
<br/>　　所以也有传闻，他们是留在了归墟天中修炼。
<br/>　　毕竟论起资源，整个九域，又有哪里可以跟归墟天媲美，虽然危险，但是归墟天中，正是修炼最好的地方。
<br/>　　很有可能，一些武者都在归墟天中留下，在其中修炼。
<br/>　　这中年男子，便有可能是其中之一，只是不知道他到底是在归墟天中，待了多久时间。
<br/>　　“这些人才是真正的威胁。”
<br/>　　楚风眠心中暗自想到。
<br/>　　这些人在归墟天中修炼，不知道是得到过多少宝藏，一个个的实力肯定都是极为惊人。
<br/>　　就像是这中年男子，他的境界还是生死境巅峰，没有踏上生死台阶。
<br/>　　但是他的一身实力，可是要比起什么剑苍穹等人，要强大的多。
<br/>　　要是回到九域中，也是横扫生死境的人物，楚风眠跟他交手，都没有必胜的自信。
<br/>　　既然是有着这样一个中年男子，那在这归墟天中，很有可能还隐藏这更多。
<br/>　　这些人一个个都是极为危险的人物。
<br/>　　甚至是在这样的情况下，这归墟天会不会有着超越生死的人物？
<br/>　　生死境巅峰的武者，楚风眠不怕，但是那些真正超越生死，踏上生死台阶的人物。
<br/>　　就算是第一重，碎命之境的武者，楚风眠现在想要对抗起来，都不一定能胜，甚至能不能全身而退，都是一个未知数。
<br/>　　这才是最让楚风眠忌惮的。
<br/>　　“看来这山谷中，肯定是隐藏这什么，云羽天宫？看起来并没有这么简单。”
<br/>　　楚风眠心中暗自想到。
