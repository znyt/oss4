　　第51章
<br/>　　纱帘后有一道倩影，缓缓吟罢，略停顿片刻道：“这诗是欧公子做的？”
<br/>　　“对我直呼其名就可，我这人随和。”
<br/>　　“欧巴……”
<br/>　　“……”程大雷。
<br/>　　这软糯糯，甜丝丝的声音沁入程大雷心底，那颗粗糙的心也为之颤了颤。
<br/>　　屋内点着红烛，有花芳香，纱帘后有清影缓动。程大雷翘起脚尖，身子向前探，眼睛睁得老大，想要看穿纱帘后的倩影。
<br/>　　这幅样子自然有失体面，好在程大雷向来也不注重体面。他真的很想仔细看看柳芷，心里想弄清楚，一个十六岁的小姑娘，身体还没长开，究竟能倾国倾城，国色天香到什么地步。
<br/>　　该不是虚假宣传吧。
<br/>　　然后房间内沉默下来，柳芷静坐不动，程大雷是不知道该说什么。他心里其实明白，应该扯些风花雪月，诗词歌赋，可如何启齿，却是有些为难。
<br/>　　对于一个第一次逛青楼的雏儿来说，程大雷目前所做已经相当不错了。
<br/>　　“欧公子，天色不早……”
<br/>　　天色不早后面跟着的，肯定不是咱们快点上床歇息。这就是下逐客令的节奏。如此被赶出去，一贯钱花得岂不是冤枉。
<br/>　　程大雷忽然心中一动，恍然大悟，自己并非什么才子，自己是个山贼呐，扯什么诗词歌赋，都忘了自己的本职工作，唉，入戏太深了。
<br/>　　想到这里，程大雷忽然大踏步向前，烛光将他的影子投在纱帘上，狰狞如怪物。
<br/>　　“公子，你要做什么……”柳芷心里就是一慌，旁人进得屋来，都是温文尔雅，使劲卖弄温柔，像程大雷这样生往里闯的，可是头一个。
<br/>　　“公子，请自重！”
<br/>　　“公子！”
<br/>　　沙……
<br/>　　程大雷大手掀开纱帘，眼前也就出现了柳芷的身影。她吓得跌坐在床上，双手抱紧胸脯，脸色煞白。
<br/>　　程大雷满意的点点头，宣传虽有夸张成分，但没有到诈骗的地步。再者说，一个十六岁的小萝·莉，只要不是太歪瓜裂枣，又能丑到哪里去。
<br/>　　“小盆宇，不要怕，我不会伤害你的。”
<br/>　　程大雷说着，大手向前伸，他是不想柳芷叫出声来，问问她家里的情况，面对一个十六岁的少女，他还没到鬼迷心窍的地步。
<br/>　　“不要，你不要过来……”
<br/>　　可落在柳芷眼中，这幅情形就显得恐怖。一男人伸出大手抓向自己，脸上露出狰狞的笑：桀桀，我不会伤害你的。
<br/>　　呸，我信了你才有鬼！
<br/>　　“不要啊，不要碰我！”
<br/>　　“小盆宇，别叫得这么大声，我这个人很温柔的。”
<br/>　　程大雷向前走，已快能捂住柳芷的嘴，不可否认，他有些恶趣味的心理在。
<br/>　　正这个时候，程大雷身后突然窜出一人，抬脚踹在程大雷屁股上，程大雷身体趴在床上，紧跟着，一柄刀压在他脖子上，冷冷的声音在背后响起。
<br/>　　“小姐，杀不杀！”
<br/>　　气氛有些诡异，床上坐着柳芷，程大雷趴在床上，两人的脸距离很近，而且两个人都很害怕。在程大雷身后立着一人，英气逼人，杀机赫赫。
<br/>　　“云哥哥，别，别杀他！”
<br/>　　刀如此才稍稍抬起，却也不离程大雷肩膀，压着程大雷缓缓站起来。
<br/>　　程大雷慢吞吞回头，看到身后站着一英武壮士，脸像石头一样肃穆。
<br/>　　姓名：孟子云（小有名气的优秀武将）
<br/>　　年龄：25
<br/>　　技能：柳家刀
<br/>　　隐藏属性：无
<br/>　　程大雷怔了怔，忽然恍然大悟：“喔，你屋里原来藏着男人！”
<br/>　　孟子云用刀背砸在程大雷肩膀上，一下将他砸倒在地，程大雷疼地龇牙咧嘴，心中直骂娘。可自己和对方差着等级呢，真要动手却也是找虐。程大雷只有在心底骂娘，想着此仇不报非君子之类的话。
<br/>　　“小姐，这混蛋知道我躲在这里，不能让他把消息走漏出去，干脆让我一刀杀了！”
<br/>　　程大雷抬头看到柳芷目光闪烁，似乎正在思考这件事。妈的，你还真想杀我，还真是最毒妇人心，古人诚不我欺。
<br/>　　孟子云亮起铁刀，刀锋寒人，照着程大雷的脑袋就要砍下。
<br/>　　“呔，我能救你一家老小！”程大雷还喊了一句。
<br/>　　“死到临头，胡说八道有用么。”
<br/>　　“慢，云哥哥，且听他怎么说！”
<br/>　　柳芷恰到好处的道了一声，孟子云的刀顿住，如此程大雷才捡回一条命。
<br/>　　程大雷趴着站起来，揉着发疼的肩膀。
<br/>　　柳芷问：“你是什么人，怎么救我家人？”
<br/>　　柳芷也是名门之后，其父曾镇守海卫，不过现在被砍头抄家，男的贬为奴隶，女的卖为官奴。
<br/>　　“他是干嘛的？”程大雷瞪了孟子云一眼：“信得过嘛？”
<br/>　　“你……”
<br/>　　“云哥哥是我父亲义子，待我如亲妹妹一般，这几天亏得他保护我，他现在是我最信任的人。”
<br/>　　程大雷恍惚想起来，白日看到自己的通缉令旁，似乎还悬赏着一人，正是这孟子云。
<br/>　　孟子云是个孤儿，被柳父收留，传授武艺，后来在军中任职。这次柳家出事，孟子云也被贬为奴隶，他杀了几个人逃监牢，第一时间就来寻柳芷。
<br/>　　柳芷一口一口云哥哥，让程大雷心底酸溜溜的，还什么亲妹妹，恐怕孟子云是想把这亲妹妹变成情妹妹。这种事，看对方的眼神就明白了，大家都是男人，有什么不懂的。
<br/>　　看着孟子云一张英俊的脸，程大雷暗中憎恶，你都二十五了，竟打人家小姑娘主意，呸，真不要脸。
<br/>　　人可还是个孩子！
<br/>　　“你究竟是什么人？快说！”孟子云催促。
<br/>　　程大雷整整衣冠，施施然道：“某家不才，正是那青牛山蛤蟆岭蛤蟆寨蛤蟆大王是也。”
<br/>　　“山贼……”孟子云吐出两个字，眉头下意识皱起。
<br/>　　我擦，我是山贼我挖你家祖坟了，用得着这么厌恶么。你一个通缉犯，有什么资格鄙视我。
<br/>　　“你就是程大雷？”柳芷问。
<br/>　　“不才正是在下。”
<br/>　　“修得胡言，那蛤蟆大王一招就能击败陆哼，怎会如你这般不中用，你再敢撒谎，我一刀砍掉你的脑袋！”
<br/>　　呃……
<br/>　　“我就知道骗不过你们。”程大雷叹口气：“实话实说，在下姓欧，名巴，字伐若，只是蛤蟆岭上一个小喽啰。”
<br/>　　“伐若……”孟子云下意识重复这两个字。
<br/>　　“然。”程大雷满意的点点头，心底补充一句：“我的乖儿子。”
<br/>
