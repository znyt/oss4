　　“果然，寒冰之后便是黑暗！”
<br/>　　元始越过寒冰，踏入又一层天幕，在这一层天之中，满目尽皆黑暗，无穷无尽的黑暗，除了黑暗还是黑暗，没有其他任何一丝光彩。
<br/>　　元始走在其中，就像是双眼被挖，成为了一个瞎子，看不见任何事物。
<br/>　　此处的黑暗，简直黑到了极致，甚至就连元始的神识都被黑暗所蒙蔽，不能看见任何东西。
<br/>　　在此处，神识的作用被削弱到了极致，神识变得不再无所不能。
<br/>　　要是平常时刻，神识的作用就是一双比眼睛还要眼睛的眼睛，他能看到许多眼睛所能看到的东西，也能看到更多眼睛所不能看到的东西。
<br/>　　但，此时此刻，元始那无所不能的神识却变得有所不能了。因为，他现在就不能看破这片黑暗，此处除了黑暗就是黑暗，就算神识能用，扫到的同样也是黑暗。
<br/>　　元始在原地观察了好一阵子后，这才选一个方向小心移动了几步。随即，他就发现，这一方天幕不仅仅全是黑暗，甚至就连虚空也变得粘稠起来，他每一步迈出，都好像是从糖稀中抬脚一般，很是艰难。
<br/>　　甚至，元始还感受到一种压抑渗人的黑暗气息渗透到灵魂之中，似乎要将他的灵魂染成黑色。
<br/>　　“哗啦！”
<br/>　　元始在虚空点燃一束光亮，但很快，这束光芒就被黑暗所吞噬，不再留下任何一丝光芒。
<br/>　　“嗯？麻烦！”元始眉头一皱，厌声道。
<br/>　　当然，这对于元始来说，仅仅是麻烦，他并没有太过担忧，对于黑暗，最有效的就是光明，没有光明，难道就没有其他方法了吗？
<br/>　　不，有！而且还有其他不少方案。比如，雷、电，再如，心！黑暗能遮蔽眼睛，能挡住神识，能侵蚀灵魂，但，人的内心，你不想让它进来，它就永远进不来。
<br/>　　但是，元始并不想用这种方法，只因为此时他想参悟此中的黑暗法则。
<br/>　　黑暗太常见了，可是黑暗规则却是很难遇到。他平常所见的一些黑暗，仅仅只是天地间五行规则或者一些异属性规则延伸出来的而已。这种黑暗，是可以用神识感受到的，和真正的黑暗规则完全不对等。
<br/>　　而此地，却有真正地黑暗规则，元始若不想办法趁机参悟，就对不起上天给的机会了。
<br/>　　元始的神识缓慢的向周围渗透出去，周围的空间一片黑暗。那种粘稠，让元始感觉到是他没有接触过的黑暗规则。
<br/>　　尽管元始知道领悟法则很难，之前意外领悟了寂灭规则就已经很幸运了，现在却又想领悟黑暗规则，简直是天人说梦。但，元始不愿放弃，他心中存有一丝侥幸，若是能领悟这种黑暗规则神通，以后对敌，也多了一种底牌。
<br/>　　元始静下心来，神识已是渗透到眼前的黑暗碎片之中，他对自己的推衍能力很有信心，只要他能获取这种黑暗属性规则的信息，他就有可能推演出黑暗神通来。
<br/>　　再说，就算短时间推衍不出来，那就用更多时间去推衍，反正他什么都缺，就是不缺少时间。
<br/>　　就在元始静静地推衍着黑暗属性规则之时，一道撕裂识海的黑潮涌了过来。陡然间，元始的识海一阵阵剧烈晃动，一种无穷无尽的黑暗袭来，让元始差点惊叫出来声。
<br/>　　这是黑暗规则的攻击，元始第一时间就明白过来。只要他意识稍弱一些，他将在这种黑暗规则的攻击下陷入永久的黑暗世界。
<br/>　　元始惊怒不已，这种黑暗属性规则对识海的攻击，绝对是有意识的，而不是因为此处的空间造成的。
<br/>　　既然是有意识的，那就是有人在这里控制这种攻击。元始的识海中第一时间形成界域，同时，鸿蒙元始印护住识海深处，灵魂之火瞬间笼罩整个元神。
<br/>　　与此同时，元始手捏元心印，虽身处黑暗，但心中光明，心痕道眼，看破虚妄，亦能看透这无尽黑暗，直达光明。
<br/>　　心宫之中，赤帝火皇挥手一道火焰闪现，瞬间就驱散识海之中所有黑潮。也许是元始的火焰太吓人了，那股黑潮并没有继续侵袭过来。
<br/>　　同时，元始也看到了是什么在控制黑潮攻击自己了，那是一个长有八只长长的黑暗触角，脑袋圆秃，眼睛漆黑，眉心闪烁着一道黑色火焰的怪物。怪物的下半身已经残破不堪了，胸前一道凶狠的裂缝，里面黑色血迹斑斑。显然，这怪物之前受到极为致命的创伤。
<br/>　　黑暗之气正是从那道裂缝之中传出来的，并且，元始在那裂缝创伤之中，感受到更为精纯的黑暗本源。怪物身上散发着一种隐晦的混沌气息，好像是被他隐藏起来，不想被人发现。但这种气息，元始却是十分熟悉。
<br/>　　“滚出此处，否则，死！”这时，一道低沉而沙哑的声音从那怪物身上传出。
<br/>　　“混沌魔神？！而且还是黑暗魔神？！”元始惊讶道。他倒是真的被惊到了，那股混沌气息元始再熟悉不过了，每一个混沌魔神都有。他倒是没想到混沌劫中黑暗魔神能够逃过一劫，并且躲过天道的发现，藏到了此。要知道，这里可是不周山啊，盘古脊骨所化的不周山，他当初是怎么躲过盘古的搜索呢？
<br/>　　“轰隆隆！”
<br/>　　一股极为恐怖的气息在瞬息之间，紧接着席卷了整片空间，空间震荡，黑潮涌滚，黑暗侵袭，大片威压涌向元始。
<br/>　　元始不慌不忙，手捏不周印，阻挡威压，那庞大的黑暗之潮，瞬间就被定在了不周印之外，不得寸进。
<br/>　　“你是谁？”那声音再次响起，低沉而嘶哑，声音中隐藏着一丝慌乱和惊怒。
<br/>　　“呵呵，还真是黑暗魔神呐！堂堂混沌魔神，如今却混得如此之惨！”元始幸灾乐祸道。
<br/>　　“你——！”显然，那黑暗魔神也十分愤怒，要不是此处盘古意志最重，好躲藏天道，修复创伤，他也不至于躲在此处。现如今竟然被一个不知名的小辈撞见，认出身份，他能不愤怒吗！
<br/>　　“不说？！不说你就去死吧，小辈！”黑暗魔神惊怒，一股滔天黑气向元始侵袭过来，同时，三道触角以迅雷不及掩耳之势向元始飞来，要先将他洞穿。
<br/>　　“呵呵，要是全盛时期，以我如今的能力，倒还忌惮你几分，但现如今，就你这样，能发挥出几分实力？”元始嘴上轻笑道，但心中，却将警惕提升到十二分，能逃过混沌劫的魔神，会是简单货色吗？显然不会！
<br/>　　同时，元始赤帝火皇之身，用太阳之火，瞬间凝聚成一个小型太阳，放出炽热光辉，朝黑暗魔神砸去。一个不够，元始便再多扔了几个，太阳之火，太阴之火，三昧真火，只要是火，都扔了过去。
<br/>　　太阳之火对黑暗法则，最好不过了。
<br/>　　黑暗是光明的克星，但光明又何尝不是黑暗的克星呢？只是两者孰强孰弱罢了。
<br/>　　如今，黑暗魔神遭受重创，尚未恢复，十分实力发挥不出二三。而元始，则是妥妥的大罗金仙圆满境界，而且还不是一般的大罗金仙。
<br/>　　此时用与光明相近的太阳之火对于黑暗，亦是轻而易举。
