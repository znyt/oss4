　　瞥了眼黝黑的通道，天鬼皇倒也讲义气，率先迈入其中主动探路。
<br/>　　走进漆黑的通道中，里面没有布置阵法，却内有乾坤，道路弯弯绕绕，错综复杂，包含数不清的分岔口，哪怕大型地宫也不过如此。
<br/>　　天鬼皇站在分岔口前，望着数十个通道，顿时有些举棋不定。
<br/>　　慕清霄微微摇头，跃过天鬼皇率先走进第二个分岔口，后者脸色有些尴尬，脚步却不慢，紧随其后。
<br/>　　“阁下……”
<br/>　　“其余的通道，或多或少都有些风痕，唯有这条通道中有些潮湿，密封的通道是不可能出现这种情况的。”
<br/>　　慕清霄选择这条通道，最主要的原因，还是鬼厄破境气息是从这里面传过来的。
<br/>　　约莫三分钟后，慕清霄与天鬼皇行走的通道中，便出现了微弱的绿色光芒，正是墙壁上点着的幽魂灯。
<br/>　　天鬼皇下意识瞥了眼慕清霄，这里的阴气明显比前面要浓郁数倍不止，看来确实没有走错。
<br/>　　不多时，走出通道后，便抵达地宫的尽头，映入眼帘的是一座土黄色墙壁，周围近百平方空荡荡一片。
<br/>　　天鬼皇阴沉的目光在空间中扫视一圈，却没有发现任何的端倪，慕清霄屈指一弹，一道锐利的剑气疾射而出。
<br/>　　“咻…”
<br/>　　就在剑气与墙壁碰撞的瞬间，原本平整的墙壁却像是水面一样，荡起一阵阵涟漪。
<br/>　　“摄魂镜，空间结界！”
<br/>　　天鬼皇脸色有些难看的望着对面的土黄色墙壁，面前的结界布置的非常精妙，哪怕鬼王级别，都很难看出哪里不对劲。
<br/>　　摄魂镜，乃是鬼族强者善用的一种幻术，顾名思义，在不知不觉的情况下，便能摄魂，轻则陷入幻境，重则灵魂受创。
<br/>　　不过，摄魂镜在被发现后，冥冥中的暗示就已经失效，更何况慕清霄与天鬼皇实力都不弱。
<br/>　　现在，挡在两人面前的，就只有一面空间结界而已。
<br/>　　此时，慕清霄缓步走到墙壁前，轻轻一掌印在墙壁上，黑色火焰乍然而现，瞬间蔓延至整座墙壁。
<br/>　　眨眼间，空间结界如同镜面一样破碎开来，随后露出墙壁后面的一道身影。
<br/>　　只见一名相貌俊朗，身穿白衣的身影正盘坐在一朵黑色莲花上，五心向天，白净的脸蛋上各种颜色互相替换。
<br/>　　“鬼厄！”
<br/>　　看到青年的瞬间，天鬼皇眼中闪过一抹杀意。
<br/>　　下一秒，青年骤然睁开双眼，似乎明亮的空间都为之一暗，深幽阴冷的声音自嘴中吐露而出，令人头皮发麻。
<br/>　　“天鬼皇，倒是小觑你了，胆子如此大，居然带人族进入鬼界。”
<br/>　　慕清霄并没有隐匿气息，虽然前面服用了阴魂丹，但也只是将他的气息变得接近鬼族而已。
<br/>　　但是直接站在鬼王面前，后者一看就知道，慕清霄并非鬼族的成员。
<br/>　　“只要你死了，谁又能知晓？”
<br/>　　天鬼皇也是破罐子破摔，眼中透露着冰冷的杀意，朝着黑莲上的鬼厄开口说道。
<br/>　　“你杀了鬼冥，你们天鬼一族会为他陪葬。”
<br/>　　阴冷的声音中充满了自信，鬼厄目光盯着慕清霄，道：“天鬼皇，难道这人族修士就是你的底牌吗？”
<br/>　　慕清霄与鬼厄的目光在空中对视，骤然一股阴冷的感觉席卷全身，似乎有一股吸力，要将他的灵魂都抽离身躯。
<br/>　　“哼！”
<br/>　　慕清霄冷哼一声，星眸替换，眨眼间就化作永恒万花筒，妖异突生。
<br/>　　下一秒，鬼厄双眼中闪过一抹迷惘，但眨眼间就恢复了清明，语气凝重，道：“猩红的妖异的双眼，你是魔族？”
<br/>　　慕清霄眼底闪过一抹不屑，眨眼间封锁空间，随后一股恐怖空间力量向鬼厄压迫而去。
<br/>　　“法则！你是大乘期修士！”
<br/>　　鬼厄的惊叫声响起，原本淡然的脸色，瞬间变得惨白无比，吼道：“住手，天鬼皇能给你的，本王也能给你！”
<br/>　　话语刚落，一抹令人余悸的黑色自鬼厄身上燃起，以迅雷不及掩耳的速度席卷而上。
<br/>　　“啊啊……”
<br/>　　惨绝人寰的叫声响起，在黑色很火的焚烧下，鬼厄凝聚出的鬼身根本无法抵抗，转眼就被焚烧成一片透明色。
<br/>　　“住手，求你放过我……”
<br/>　　伴随着鬼厄最后一抹凄厉的惨叫声消逝，慕清霄淡漠道：“我不喜欢给自己留下后患，你还是消失的好。”
<br/>　　不杀鬼厄，他暴露的几率就越高，到时候引来鬼族鬼圣或者鬼仙的追杀，这可不是他想要的结果。
<br/>　　最直接的办法，便是永绝后患，正如天鬼皇所说，只要鬼厄死了，谁又能知道呢？
<br/>　　站在一旁，天鬼皇目瞪口呆，魁梧的身躯都有些僵硬，喉咙滚动一下，暗自咽了口唾沫。
<br/>　　原本，他以为慕清霄击杀鬼厄最起码也需要点时间，却不曾想到，鬼厄在他面前，连一个呼吸的时间都抵不住。
<br/>　　那令人余悸的黑色火焰，瞬间就将鬼厄焚烧的魂飞魄散，一丝意念都没留下。
<br/>　　待鬼厄死亡，永恒万花筒化作深邃的星眸，慕清霄嘴角牵起一抹弧度。
<br/>　　撤销封锁的空间的结界后，不理会天鬼皇呆愣的神色，缓步来到黑莲前，黑莲上有一枚自鬼厄身上掉落的黑色戒指，应该是空间戒指。
<br/>　　拿起戒指，将其上的灵魂印记抹除，神识窥探其中，慕清霄眼中闪过一抹喜色。
<br/>　　空间戒指中拥有不少给人族使用的高阶灵药，最多的还是给鬼族修炼的东西，鬼丹堆积成山，数都数不过来。
<br/>　　鬼王就是鬼王，底蕴就是深厚。
<br/>　　将空间戒指收起，慕清霄看了眼底下的黑色莲花，目光看向天鬼皇，道：“这是什么？”
<br/>　　闻言，天鬼皇自惊愕中回过神来，看向黑莲闪过一抹惊喜。
<br/>　　“这应该是罗刹鬼莲，鬼族利用它修炼，事半功倍。”
<br/>　　话语刚落，在天鬼皇惊喜的目光中，慕清霄手一挥，罗刹鬼莲就消失不见。
<br/>　　想到倩女幽魂中的聂小倩，慕清霄无奈的摇了摇头。
<br/>　　在那种低级仙侠位面中，哪怕有《玄阴素女经》这种最顶级的心法，有天道枷锁，前途终究有限。
<br/>　　现在有了鬼王的一身家当，聂小倩哪怕资质在愚钝，修成鬼王也只是时间问题。
